# 基土拉

## 定義

亞伯拉罕在撒拉死後所娶的妻子（創25:1）。名字意為「香料、金香」（希伯來文 קְטוּרָה）。部分經文稱她為「妾」（代上1:32），本書則稱她為「妻」。

## 主題發展

- **經文爭議**：創25:1 稱她為「妻子」，代上1:32 稱她為「妾」
- `ishshah` 可泛指「女人」；創25:6 和代上1:32 均含「妾」的含意
- 基土拉雖是亞伯拉罕暮年唯一合法妻子，但與撒拉相比地位次要
- 神已揀選撒拉為以撒之母，以撒是應許之約的唯一嗣子
- 有學者認為亞伯拉罕在撒拉在生時已娶基土拉；丁良才則認為在撒拉死後
