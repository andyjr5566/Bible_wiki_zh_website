# 不可使異類混雜（牲畜種子衣料）

## 定義

要守神的律例，不可叫牲畜與異類配合，不可用兩樣攙雜的種種地，也不可用兩樣攙雜的料做衣服穿在身上；目的是維持神創造中「各從其類」的原則，並防止以色列人與外族同化，效法外邦宗教和生活習俗。

- 與本章關聯：屬於向著財物的聖潔生活條例，教導在日常生活細節中維持聖別。
