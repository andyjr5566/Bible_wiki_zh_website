# 亞伯拉罕之約

## 定義

神與亞伯蘭所立的約（創十二1-3），包含七層應許。此約是整本舊約的轉捩點——在此之前是全人類的記錄，在此之後以以色列民族為焦點。

#### [[01 創世記/第32章|第32章]]
- 本章重點：創32章：雅各禱告中三度引用亞伯拉罕之約應許：（1）v.9「回你本地本族去、我要厚待你」（創31:3；28:13-15）；（2）v.12「我必定厚待你、使你後裔如海邊沙、多不可勝數」（創28:14）；（3）v.10「我先前只拿著杖過約旦河、如今成了兩隊」見證約中「賜福、增多」應許成就。雅各以約為憑據向神禱告（丁良才：禱告六要素含「依約、憑命」），神在雅博渡口摔跤、改名以色列、賜福，確立雅各為約的繼承人（華爾頓：改名代表立約繼承人身分確立，同創17:5亞伯蘭→亞伯拉罕）。
- 與本章關聯：亞伯拉罕之約（創12:1-3；15；17）在雅各身上逐步應驗：創28:13-15伯特利重申、創31:3命令回鄉、創31:13「我是伯特利的神」提醒許願、創32:9-12禱告持定、創32:28改名以色列確立繼承人身分、創35:9-13伯特利再立約。本章為約應驗關鍵節點：從「奪取者」雅各轉為「與神較力得勝者」以色列。

## 主題發展

### 七層應許

1. **必叫你成為大國**（2節）— 肉體後裔如塵沙多，屬靈後裔如眾星
2. **必賜福給你**（2節）— 屬世和屬靈的福
3. **叫你的名為大**（2節）— 「多國的父」「先知」「尊大的王子」「神的僕人」「神的朋友」
4. **你也要叫別人得福**（2節）
5. **為你祝福的，我必賜福與他**（3節）
6. **那咒詛你的，我必咒詛他**（3節）
7. **地上的萬族都要因你得福**（3節）— 應驗在基督身上

### 亞伯蘭放棄的三樣 ↔ 神回應的三樣

- 本地 → **土地**（迦南地）
- 本族 → **後裔**（大國）
- 父家 → **祝福**（叫別人得福）

### 約的重申

此約至少相隔24年四次重申：創十二4；十三14-17；十五1-21；十七1-8

### 與[[挪亞之約]]的對比

- 挪亞之約：神對全人類和一切活物的應許
- 亞伯拉罕之約：神藉一個人使萬族得福
