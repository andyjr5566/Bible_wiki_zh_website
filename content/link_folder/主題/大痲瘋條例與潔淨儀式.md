# 大痲瘋條例與潔淨儀式

## 定義

利未記第13章詳盡記載了各類大痲瘋災病的察看條例與潔淨判定標準。條例涵蓋人體皮膚的癬、瘡、火毒、頭疥，以及衣物和皮子上的災病。祭司需透過觀察毛髮變白、現象深於皮、發散程度等特徵，並執行關鎖七天的隔離觀察機制，來判定患者或物件是否潔淨。此條例旨在維護以色列營區的禮儀聖潔，並防範不潔淨的傳染。
