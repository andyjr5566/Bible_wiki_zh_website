# 「野驢」的隱喻

## 定義

耶和華使者豫言以實瑪利「為人必像野驢」。野驢象徵自由、兇猛、不服管轄、在曠野漂泊的性情。豫表宗教徒的特點：不受管束，彼此嫉妒分爭。
