# 苦水試驗

## 定義

祭司將聖水盛在瓦器中，取會幕地上的塵放在水中，寫咒詛的話抹在苦水裡，叫婦人喝。這水進入腸裡變苦，若婦人有罪，肚腹發脹、大腿消瘦，成為咒詛；若無辜，免受災且懷孕（民5:17-24, 27-28）。此為神聖審判儀式，將隱藏之罪交託神揭示。
