# 肢體有餘缺少

## 定義

利未記 22:23 指出祭牲若「肢體有餘的，或是缺少的」（原文 yeter 多餘、chaser 缺少），只可作甘心祭獻上，用以還願卻不蒙悅納。這屬於「有殘疾」的一類（v.20-21），身體結構不完整（如多指、缺肢）即屬此列。經文區分「甘心祭」與「還願祭」的接納標準：前者可寬容此類缺陷，後者嚴格要求「純全無殘疾」。

- 本章重點：第 23 節規定肢體有餘或缺少的牲畜只能作甘心祭，不可用於還願，因不屬「純全無殘疾」而不蒙悅納。
- 與本章關聯：本章獻祭牲無殘疾條例的細目，與「陽物損傷」同屬身體缺陷分類，共同界定「無殘疾」的標準。
