# 寧錄與敵基督

## 定義

創十8-12的寧錄被視為敵基督（大罪人）的豫表，其六項特徵與帖後二及啟十三的敵基督描寫對應。

## 主題發展

### 寧錄 vs 敵基督的對照

| 寧錄特徵 | 敵基督對應 |
|----------|-----------|
| 「背叛者」（名字含義） | 不法的人（帖後二8） |
| 世上英雄之首 | 大有能力（帖後二9） |
| 獵戶（兇暴） | 獸（啟十三15） |
| 「在耶和華面前」英勇 | 目中無神、狂傲（帖後二4） |
| 國起頭是巴別 | 對抗並混亂神的國度 |
| 征服、建造黑暗權勢 | 聯合成大勢力範圍 |

### 獵戶 vs 牧者

- 寧錄從獵殺動物開始，進展到踐踏人來建立帝國
- 建造神國度的人都是**牧者**，為羊捨命（約十11、14）
- 例子：[[亞伯]]、雅各、摩西、大衛——最大的例子是主耶穌
