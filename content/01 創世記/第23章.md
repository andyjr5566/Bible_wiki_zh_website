# 創世記 第23章

1. [[撒拉]]享壽一百二十七歲，這是撒拉一生的歲數。
2. [[撒拉]]死在迦南地的[[基列亞巴]]，就是[[希伯崙]]。[[亞伯拉罕]]為他哀慟哭號。
3. 後來[[亞伯拉罕]]從死人面前起來，對[[赫人]]說：
4. 我在你們中間是外人，是[[寄居者身份|寄居的]]。求你們在這裡給我一塊地，我好埋葬我的[[死人]]，使他不在我眼前。
5. [[赫人]]回答[[亞伯拉罕]]說：
6. 我主請聽。你在我們中間是一位尊大的王子，只管在我們最好的[[墳地]]裡埋葬你的[[死人]]；我們沒有一人不容你在他的墳地裡埋葬你的死人。
7. [[亞伯拉罕]]就起來，向那地的[[赫人]]下拜，
8. 對他們說：你們若有意叫我埋葬我的[[死人]]，使他不在我眼前，就請聽我的話，為我求瑣轄的兒子[[以弗崙]]，
9. 把田頭上那[[麥比拉洞]]給我；他可以按著足價賣給我，作我在你們中間的[[墳地]]。
10. 當時[[以弗崙]]正坐在[[赫人]]中間。於是，赫人以弗崙在城門出入的赫人面前對[[亞伯拉罕]]說：
11. 不然，我主請聽。我送給你這塊田，連田間的洞也送給你，在我同族的人面前都給你，可以埋葬你的死人。
12. [[亞伯拉罕]]就在那地的人民面前下拜，
13. 在他們面前對[[以弗崙]]說：你若應允，請聽我的話。我要把田價給你，求你收下，我就在那裡埋葬我的[[死人]]。
14. [[以弗崙]]回答[[亞伯拉罕]]說：
15. 我主請聽。值[[四百舍客勒]]銀子的一塊田，在你我中間還算什麼呢？只管埋葬你的死人吧！
16. [[亞伯拉罕]]聽從了[[以弗崙]]，照著他在[[赫人]]面前所說的話，把買賣通用的[[銀子]]平了[[四百舍客勒]]給以弗崙。
17. 於是，麥比拉、幔利前、[[以弗崙]]的那塊田和其中的洞，並田間四圍的[[樹木]]，
18. 都定準歸與[[亞伯拉罕]]，乃是他在[[赫人]]面前並城門出入的人面前買妥的。
19. 此後，[[亞伯拉罕]]把他妻子[[撒拉]]埋葬在迦南地幔利前的麥比拉田間的洞裡。（幔利就是[[希伯崙]]）。
20. 從此，那塊田和田間的洞就藉著[[赫人]]定準歸與[[亞伯拉罕]]作[[墳地]]。

## 本章整理

### 撒拉之死（1-2節）

撒拉享壽一百二十七歲——聖經中唯一記載壽數的女人，表示她一生的年日蒙神記念；此時以撒三十七歲，距亞伯拉罕離開吾珥六十二年（GT）。自亞伯拉罕受割禮（創17）至撒拉死，其間約三十八年，令人聯想以色列人在曠野繞行三十八年（申2:14；CT 靈意）。

她死在迦南地的[[基列亞巴]]，就是[[希伯崙]]（字義「同盟、友誼」）。亞伯拉罕為她「哀慟哭號」——「哀慟」指裏面的情緒，「哭號」原文是大聲哀號（CT）；這是聖經中亞伯拉罕第一次流淚的記錄（KC）。撒拉存著信心而死（來11:11、13）；信徒為親人哀傷是正常的，但不像沒有指望的人過度憂傷（帖前4:13-14）——自然的眼淚與信心兼容。

### 求購墳地與寄居者身份（3-9節）

亞伯拉罕從死人面前「起來」——哀哭有時，但信心使人不被憂傷吞噬（KC）。他對[[赫人]]承認自己是「外人，是[[寄居者身份|寄居的]]」（4節）——憑現實承認地位，非憑應許自居；雖然神應許迦南全地歸他，此時卻連一塊葬身之地都沒有（來11:13；彼前2:11）。

赫人稱他為「尊大的王子」（原文「有神威的王子」）——他在外邦人中蒙神眷顧、受人敬重，與被藐視的羅得成對比（創19:9；KC）。但他們表面慷慨的「只管用我們最好的墳地」實為外交辭令，意在避免外人購置永久性土地（CT）。

亞伯拉罕兩次下拜（7、12節）——古代近東禮儀上的謙恭，不是敬拜（BH）。他不繞遠路，直接指名向瑣轄的兒子[[以弗崙]]購買田頭上的[[麥比拉洞]]（字義「雙倍」），並堅持「按著足價」——不要人情壓力，避免日後糾葛；恩典不叫我們可以不顧公義。

### 以弗崙的談判（10-16節）

以弗崙在城門出入的赫人面前說「我送給你這塊田，連田間的洞也送給你」（11節）——中東以退為進的買賣手段（CT）：按赫人法規，僅出售部分土地，原主仍須負擔整塊田的稅賦與義務（GT），故他要亞伯拉罕連田帶洞整塊買下。

以弗崙開價[[四百舍客勒]]銀子（15節），與其他聖經交易對照：

| 交易 | 價格 |
|---|---|
| 麥比拉田與洞（本章） | 400舍客勒銀子 |
| 大衛買禾場和牛（撒下24:24） | 50舍客勒銀子 |
| 耶利米購地（耶32:9） | 17舍客勒銀子 |
| 奴隸身價（創37:28） | 約20舍客勒 |
| 僱工年資（背景資料） | 約10舍客勒 |

估價顯然過高，但亞伯拉罕沒有討價還價（16節）：全額付款可確保永久產權，避免給以弗崙的後人留下追討口實（舊約背景註釋）；在承受應許之日來到以前，他不肯佔別人便宜（CT）。他用「買賣通用的[[銀子]]」——商用標準砝碼，公正無欺（GT/BH）。

### 交易完成與埋葬（17-20節）

麥比拉田、洞、四圍的[[樹木]]都「定準」歸亞伯拉罕（17-18節）——界線清楚劃定，在赫人及[[城門口交易|城門出入的人]]面前買妥，有公開的法律保障。此後亞伯拉罕把「他妻子撒拉」葬在麥比拉田間的洞裡——他埋葬的是「我的死人」、「他妻子撒拉」，不是「遺體殘留」；身體就是那個人（KC；約19:40-42）。

希伯崙後來成為族長家族的祖墳：亞伯拉罕（創25:9）、以撒、利百加、利亞（創49:29-31）、雅各（創50:13）皆葬於此。古代近東以洞穴墳為常見，多代共用（舊約背景註釋）；麥比拉洞今日上方為希伯崙的易卜拉欣清真寺建築。

關鍵事實：神應許賜迦南全地為業，但亞伯拉罕此時所得唯一的永久性產業，竟是一塊[[墳地]]——應許地的初熟果子。神所應許的產業不止於地上（來11:8-16）；購地安葬正是他對神應許不動搖的信心：先有死與安葬，才有將來的復活與得地。

### 信徒與世人交往的五要（CT）

1. 表明自己只是寄居的（4節）——不堅持權利
2. 對人謙恭有禮（7節）——兩次下拜
3. 不佔別人的便宜（8-9節）——堅持按足價付錢
4. 要有智慧（10-16節）——聽得出話中的話
5. 一切手續辦得清清楚楚（17-18節）——有憑有據

### 埋葬與復活的神學（KC）

> [!quote] KC
> 埋葬就是撒種，撒種是為著收成——「埋葬」與「復活」相連，而不是與「死亡」相連（林前十五3-4）。

| 要義 | 說明 |
|------|------|
| 埋葬＝種下 | 種子埋入土裏等候收割，死亡不是終結 |
| 復活次序 | 林前15:20-21——在基督裏的人都要復活 |
| 身體就是人 | 「我的妻子撒拉」而非「遺留物」；約翰福音記「把耶穌安放」而非「遺體」 |
| 神知道祂子民在哪裏 | 以色列如今看似枯骨（結37），但神知道他們的位置 |
| 全額買贖 | 亞伯拉罕付足價；神也用獨生子的血付足了買贖世界的代價 |
| 埋葬與焚燒 | 埋葬連於復活盼望；焚燒在舊約多連於審判（書7:25；利21:9） |

### 豫表與神學意義

| 項目 | 豫表／意義 |

| 撒拉之死 | KC：撒拉是以色列的圖畫——先被擱置，然後以撒（基督）得著新婦利百加（教會） |
