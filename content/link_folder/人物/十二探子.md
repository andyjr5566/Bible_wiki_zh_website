# 十二探子

## 定義

民數記13章記載耶和華吩咐摩西從各支派選派一位首領共十二人去窺探迦南地，名單見第4-15節，包括流便的沙母亞、西緬的沙法、猶大的迦勒、以薩迦的以迦、以法蓮的何西阿（後改名約書亞）、便雅憫的帕提、西布倫的迦疊、瑪拿西的迦底、但的亞米利、亞設的西帖、拿弗他利的拿比、迦得的臼利。他們窺探四十天後回報，十人報惡信，唯迦勒與約書亞持信心。
