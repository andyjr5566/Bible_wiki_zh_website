# 創世記 第23章

1. [[撒拉]]享壽一百二十七歲，這是撒拉一生的歲數。
2. 撒拉死在迦南地的[[基列亞巴]]，就是[[希伯崙]]。[[亞伯蘭]]為她哀慟哭號。
3. 後來亞伯蘭從死人面前起來，對[[赫人]]說：
4. 「我在你們中間是外人，是[[寄居者身份|寄居的]]。求你們在這裏給我一塊地，我好埋葬我的[[死人]]，使她不在我眼前。」
5. 赫人回答亞伯蘭說：
6. 「我主請聽。你在我們中間是一位尊大的王子，只管在我們最好的墳地裏埋葬你的[[死人]]；我們沒有一人不容你在他的墳地裏埋葬你的[[死人]]。」
7. 亞伯蘭就起來，向那地的赫人下拜，
8. 對他們說：「你們若有意叫我埋葬我的[[死人]]，使她不在我眼前，就請聽我的話，為我求瑣轄的兒子[[以弗崙]]，
9. 把田頭上那[[麥比拉洞]]給我；他可以按著足價賣给我，作我在你們中間的墳地。」
10. 當時以弗崙正坐在赫人中間。於是，赫人以弗崙在城門出入的赫人面前對亞伯蘭說：
11. 「不然，我主請聽。我送給你這塊田，連田間的洞也送給你，在我同族的人面前都給你，可以埋葬你的死人。」
12. 亞伯蘭就在那地的人民面前下拜，
13. 在他們面前對以弗崙說：「你若應允，請聽我的話。我要把田價給你，求你收下，我就在那裏埋葬我的[[死人]]。」
14. 以弗崙回答亞伯蘭說：
15. 「我主請聽。值[[四百舍客勒]]銀子的一塊田，在你我中間還算甚麼呢？只管埋葬你的死人吧！」
16. 亞伯蘭聽從了以弗崙，照著他在赫人面前所說的話，把買賣通用的[[銀子]]平了[[四百舍客勒]]給以弗崙。
17. 於是，麥比拉、幔利前、以弗崙的那塊田和其中的洞，並田間四圍的[[樹木]]，
18. 都定準歸與亞伯蘭，乃是他在赫人面前並城出入的人面前買妥的。
19. 此後，亞伯蘭把他妻子撒拉埋葬在迦南地幔利前的麥比拉田間的洞裏。〈幔利就是希伯崙〉。
20. 從此，那塊田和田間的洞就藉著赫人定準歸與亞伯蘭作墳地。

## 本章整理

### 一、撒拉之死（1~2節）

- 撒拉享壽**127歲**——聖經中**唯一記載壽數的女人**，表示她一生蒙神記念
- 以撒此時**37歲**（GT）
- 亞伯拉罕離開吾珥後**62年**（GT）
- 亞伯拉罕受割禮（創十七）至撒拉死，其間**38年**——呼應以色列人在曠野繞行38年（申二14）
- 死在迦南地基列亞巴，就是希伯崙（「同盟、友誼」之意）
- 亞伯拉罕為她**哀慟哭號**——「哀慟」指裏面情緒，「哭號」原文為大聲哀號至頓足捶胸（CT）
- 這是聖經中亞伯拉罕**第一次哭泣**的記錄（KC）
- 撒拉是存著信心而死（來十一11），死在與神的交通中（帖前四14）
- 信徒為親人哀傷是正常的，但不致像沒有指望的人過度憂傷（帖前四13~14）
- 自然的眼淚與信心兼容；信徒可以哀傷，但信心使他能再起來（KC）

### 二、亞伯拉罕求購墳地（3~9節）

1. **承認寄居者身份**（4節）——「我在你們中間是外人，是寄居的」
   - **憑現實承認自己地位，非憑神的應許**（CT）
   - 雖神應許迦南全地歸他，此時卻連一塊葬身之地都沒有
   - 赫人稱呼亞伯拉罕為「尊大的王子」「有神威的王子」——表明他在外邦人中蒙神眷鄰，十二2-3的應許局部應驗（BH）
   - 預表：我們是天上的國民（來十一13；彼前二11）

2. **赫人的恩惠與手段**（5~6節）
   - 他們稱願意借出最好的墳地——表面慷慨，**實際是外交辭令，目的在阻止亞伯拉罕購置永久性土地**（CT/KC）
   - 亞伯拉罕**兩次下拜**（7、12節）——古代近東禮儀上的謙恭（BH），但不是崇拜
   - 這表明亞伯拉罕在外的影響力，但真正的產業問題未解決

3. **亞伯拉罕的策略：指定交易對象**（7~9節）
   - 亞伯拉罕**不浪費時間試探 INDIRECT——直接向以弗崙指名購買**（CT）
   - 要求「按著足價」——不要任何人情壓力，避免未來糾葛
   - 恩典不叫我們可以不顧公義；雖有應許，仍不可奪取別人產業

### 三、以弗崙的談判（10~16節）

1. **以弗崙表面慷慨**（11節）——「我送給你這塊田，連田間的洞也送給你，在我同族的人面前都給你」
   - 中東人以退為進的買賣手段（CT）
   - 按赫人法規：僅出售部分，原主仍須負擔整塊稅賦（GT）
   - 表面慷慨、實際高價——**古時中東人以退為進的買賣手段**（CT）

2. **開價四百舍客勒銀子**（15節）——比對：
   - 大衛買禾場：50舍客勒
   - 耶利米購地：17舍客勒
   - 奴隸身價：約20舍客勒
   - 年工資：約10舍客勒
   → **估價過高**，但亞伯拉罕沒有討價還價

3. **亞伯拉罕為何不還價**（16節）：
   - 全額付款可確保永久產權，折扣可能給後人追討口實
   - 在承受之日來到前，不肯佔別人便宜
   - 亞伯拉罕**不與以弗崙爭論**——他知道那地是他的產業，後裔必要承受；但在承受之日來到前，不肯從別人身上佔便宜（CT）
   - 「四百」= 四十乘十，代表**完全的試煉**（CT）
   - 用「買賣通用的銀子」——標準砝碼，公正無欺（GT/BH）

### 四、交易完成與埋葬（17~20節）

1. **清楚劃定界線**（17~18節）——麥比拉田、洞、四圍樹木都**定準**歸亞伯拉罕
   - 在赫人及城門出入的人面前**買妥**——公開的法律保障（CT）
   - 靈意：「田」是生長生命的地方；「樹木」代表強韌的生命；信徒肉身雖死，卻滿了復活盼望（帖前四16）
   - 「田與樹」——復活生命的畫面：栽種生長，耐力不衰（CT）

2. **撒拉埋葬在麥比拉洞**（19~20節）——幔利就是希伯崙
   - 亞伯拉罕埋葬的是「**我的妻子撒拉**」——身體就是人本身，不是「遺留物」（KC）
   - 希伯崙後來成為希伯來人**祖墳**：六代族長——亞伯拉罕（創廿五9）、以撒（創四十九29-31）、利百加（同前）、利亞（同前）、雅各（創五十13）、以實瑪利（創廿五17）皆葬於此

3. **古代近東埋葬習俗**（GT 補充）
   - 洞穴墳為常見，多代共享，骨頭后来被移至後方內室（ossuary）
   - 麥比拉洞今日仍可辨識，上面有**亞伯拉罕清真寺（Ibrahimi Mosque）**建築

4. **關鍵事實**：神應許賜迦南地為業，但此時亞伯拉罕所得**唯一永久性產業，竟是一塊墳地**
   - 神所應許的產業非地上物質產業，乃是天上更美的家鄉（來十一8~16）
   - 墳地是應許地的**初熟果子**——先有死與安葬，才有將來的復活與得地
   - 埋葬連於復活：舊約信徒存信心而死，等候與新約信徒同得更美復活（來十一39~40）
   - 亞伯拉罕在靈裏認定死亡是「被埋葬」，不是「被遺棄」——這是信心的得勝（CT）

### 五、信徒與世人交往的五要（CT 特別強調）

亞伯拉罕在本章與赫人交往，呈現信徒與外邦人相處的榜樣：

1. **表明自己只是寄居的**（4節）——不堅持权利
2. **對人謙恭有禮**（7節）——兩次下拜
3. **不佔別人的便宜**（8~9節）——堅持按足價付錢
4. **要有智慧**（10~16節）——以弗崙的話中話他都聽得懂
5. **一切手續要辦得清清楚楚**（17~18節）——有憑有據

### 六、埋葬與復活的完整神學（KC 補充）

| 要義 | 說明 |
|------|------|
| 埋葬 = 種下 | 種子埋入土裏，等候收割——死亡不是終結 |
| 復活次序 | 林前15:20-21——在基督裏的人都將復活 |
| 身體就是人 | 「我的妻子撒拉」而非「遺留物」；耶穌是「耶穌」而非「遺留物」 |
| 神知道祂子民在哪裏 | 以色列如今看似「死了」，但神埋葬祂的子民，知道他們的位置 |
| 神的買贖 | 神用獨生子的血買贖世界（包括墳地/全地） |
| 埋葬 vs 焚燒 | 埋葬=復活盼望；焚燒=審判（書七25；利廿一9） |
