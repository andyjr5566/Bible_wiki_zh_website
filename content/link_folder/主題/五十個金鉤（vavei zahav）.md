# 五十個金鉤（vavei zahav）

## 定義

用金子製成的五十個鉤子，用以連結內層十幅幔子的兩組（每組五幅）。每組幔子末幅邊有五十個藍色鈕扣，金鉤穿過鈕扣使幔子相連，合成一個完整的帳幕（出26:6）。材質為金，區別於罩棚用的銅鉤（出26:11）。

## 主題發展

金鉤數目五十，與鈕扣數目對應，確保幔子連接穩固完整。金材質呼應聖所內部一切器具包金的原則，象徵神聖、榮耀、不朽壞的連結。
