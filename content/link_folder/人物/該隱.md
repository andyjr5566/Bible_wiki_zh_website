# 該隱

## 定義

希伯來文 *Qayin*（קַיִן），意思是「得著」。亞當和夏娃的長子，是人類歷史上第一個殺人者。以種地為業，因獻祭不被神看中而嫉妒弟弟亞伯，將他殺死。

### 與目前整理書卷的關聯

該隱是人類墮落後的第二代。他代表「不要神」的路線：人意宗教、靠行為稱義、嫉妒、殺人、說謊、最終與神完全隔絕。

## 主題發展

### 神學意義／研讀意義

- 該隱墮落的六步：不順從啟示 → 發怒 → 嫉妒殺人 → 說謊 → 自憐 → 與神隔絕
- 該隱的祭代表人意宗教（靠行為）；亞伯的祭代表啟示宗教（靠血）
- 屬血氣的逼迫屬靈的（加四29）
- 該隱路線是舊造的族類；塞特路線是新造的族類
- 該隱的後代發展無神文化（經濟→藝術→戰爭→殺戮）
