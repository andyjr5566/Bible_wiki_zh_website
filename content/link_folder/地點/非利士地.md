# 非利士地

## 定義

非利士地（希伯來文 Pelesheth，希臘文用近音 Palaistina，英文 Palestine，譯「巴勒斯坦地」原文出處）是古代位於迦南地西南沿海的地區，相當於今日以色列南方加薩走廊一帶。在出埃及記 13:17 時，非利士地指從埃及通往迦南的一條主要交通路線所經之地，這條路線後來稱為「沿海大道」（Via Maris，又譯「非利士地的道路」），串聯埃及東北邊至迦南美地。

- CT指出從疏割往迦南走西乃半島北部的非利士道路（即沿海大道）本是最近路線。
- BH指出非利士地的沿海大道雖是從埃及到應許地最直接的路，卻是當時埃及駐軍與非利士人重兵要塞區域。

## 主題發展

非利士地作為地理路線與族群，連結了出埃及敘事（出 13:17 沿海大道）與士師時代的敵對關係（士 13-16 非利士人與參孫）。新約時期的「巴勒斯坦」地名（希臘文 Palaistinē）從非利士（《七十士譯本》Palaistinoi）演變而來，其語源延續到今日的地理與政治名稱之中。
