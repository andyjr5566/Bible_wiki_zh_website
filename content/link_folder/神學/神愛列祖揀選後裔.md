# 神愛列祖揀選後裔

## 定義

申命記 4:37-38 宣告神因愛列祖（亞伯拉罕、以撒、雅各），揀選他們的後裔（以色列），用大能親自領他們出埃及（脫離鐵爐），趕出比他們強大的國民，賜地為業。此揀選非基於以色列的公義或人數（參申 7:7-8，9:4-6），純屬神主權之愛與列祖之約的信實，作為「悔改蒙恩」（v.31）不撇下、不忘記約的根基，也是「神的獨一無二」（v.35, 39）在歷史中的具體彰顯。

## 主題發展

此主題在 v.37-38 以「因他愛你的列祖、所以揀選他們的後裔」為邏輯起點，串聯「用大能親自領你出埃及」（出埃及記 12-14）、「趕出強大國民」（民數記 21、約書亞記 1-12）、「賜地為業」（v.38, 46-49）。此揀選恩典向上連結「列祖之約」（v.31），向下支撐「遵守律例得福長久」（v.40），向內奠基「神的獨一無二」（v.35, 39）的救贖歷史見證。埃及「鐵爐」意象（v.20, 37）強調救贖的徹底性與代價。
