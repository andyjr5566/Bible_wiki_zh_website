# 誰可吃聖物的資格條例

## 定義

利未記第22章1至16節所定規，論到祭司與其家屬在何種條件下可以吃以色列人分別為聖歸給耶和華的聖物，以及何種人不可吃。凡身上有污穢（如長大痲瘋、漏症、觸摸死屍或不潔之物等）的祭司，必須潔淨後方可吃；外人、寄居者、雇工、嫁給非祭司的女兒均不可吃，但祭司用錢買的奴僕和家中生的兒女可以吃；寡婦或被休無子而歸回父家的祭司女兒也可吃。誤吃聖物者須償還原數另加五分之一。祭司有責任不讓聖物被褻瀆，免得百姓自取罪孽。
