# 遵耶和華吩咐起行安營

## 定義

民數記 9:18 「以色列人遵耶和華的吩咐起行，也遵耶和華的吩咐安營」。強調百姓的行動完全依照神透過雲彩所發的命令，體現順服與信靠的神學主題。
