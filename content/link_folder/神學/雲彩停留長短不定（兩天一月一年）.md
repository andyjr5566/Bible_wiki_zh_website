# 雲彩停留長短不定（兩天一月一年）

## 定義

民數記 9:22 「雲彩停留在帳幕上，無論是兩天，是一月，是一年，以色列人就住營不起行」。說明神引導的時間長短完全由主決定，百姓需耐心等候，彰顯神主權與百姓信靠的關係。
