# 瑪3（Malaki 3）

## 定義

瑪拉基書第三章（希伯來文：Malaki），預言主必忽然進入殿，如鍊金之火、漂布之鹼，潔淨利未人，使他們獻上公義的供物；責備人偷竊神的當納十分之一和獻祭，應當將當納的帶入倉庫，試驗神是否敞開天窗傾福。

## 按書卷累積

### 民數記

<!-- accumulation:民數記:7:start -->
#### [[04 民數記/第7章|第7章]]
- 本章重點：民數記7章詳細記載十二支派首領為壇獻供物，供物細目嚴謹（按聖所平）、牲畜完整（燔祭、贖罪祭、平安祭），利未人按職分服事（革順、米拉利、哥轄），神在會幕與摩西說話。瑪3章責備祭司藐視神名、獻殘缺牲畜，對比民數記7章「按聖所平」、「公牛犢、公綿羊、一歲公羊羔」無殘缺、「神悅納奉獻」、「利未人無分神為產業」的標準，形成「潔淨利未人、獻公義供物」的先知性回顧與警戒。
- 與本章關聯：供物標準與利未人職分的先知性對比
