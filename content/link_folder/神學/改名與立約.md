# 改名與立約

## 定義

改名是創世記十七章的一個重要主題：亞伯蘭改名「亞伯拉罕」（多國之父），撒萊改名「撒拉」（多國之母）。改名代表生命的轉變——神的成分加增，自己的成分減少。

#### 第17章
- 「亞伯蘭」→「亞伯拉罕」：從「高舉之父」到「多國之父」
- 「撒萊」→「撒拉」：從「我的公主」到「多國之母」
- 神將自己名中的一個字母加進去，表明人惟有靠神才能成事
- 改名表明這人已經更改——從自己有辦法，變成沒有辦法
- 來源：CT, GT, KC
