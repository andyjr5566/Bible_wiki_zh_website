# 火從耶和華面前出來燒盡祭物

## 定義

「有火從耶和華面前出來，在壇上燒盡燔祭和脂油」（利9:24上）。這火不是人所點的，而是從耶和華面前出來的；它燒盡壇上的燔祭與脂油，公開表明神親自悅納亞倫所獻的祭與新設立的祭司事奉。眾民一見，就都歡呼、俯伏在地（v24下）。
