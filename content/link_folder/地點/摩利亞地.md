# 摩利亞地

## 定義

摩利亞地（Moriah），原文意為「神預備、神看顧」。位於耶路撒冷附近山區。所羅門王在此建聖殿（代下三1），臨近耶穌釘十字架的各各他山。亞伯拉罕在此獻以撒，是信心最嚴厲的考驗。
