# 古代近東身體排泄物與禮儀潔淨觀

## 定義

在古代近東文化中，身體的排泄物（如漏症、精液、經血）常被視為生命力的流失或與死亡相關，因此在禮儀上被歸類為不潔淨。利未記15章的條例反映了這一文化背景，但將其獨特地整合於耶和華信仰的聖潔體系中。與鄰近文化純粹驅邪或魔法的觀念不同，以色列的潔淨條例強調的是「不潔淨會玷污神的帳幕」這一神學原則，並透過活水洗身、計算七天與獻祭等程序，表明神是生命與聖潔的來源，必須與死亡的記號隔絕。

#### [[03 利未記/第15章|第15章]]
- 本章重點：本章針對漏症、夢遺、交合、月經與血漏等身體排泄現象，制定了詳細的禮儀潔淨與隔離規範。
- 與本章關聯：本章條例建立在古代近東對身體排泄物視為不潔的文化基礎上，但賦予其保護帳幕聖潔的獨特神學意義。
