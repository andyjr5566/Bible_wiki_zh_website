# 催逼

## 定義

出埃及記11:1「他容你們去的時候，總要催逼你們都從這地出去」——CT原文字義指出「催逼」原文希伯來文 khērēṣ / khōrēs（本處為動詞 ḥērēṣ / rsh, 「驅趕、強迫驅趕」），意為「無法容忍而以暴力驅趕」——不只是被動容讓，而是法老主動且迫不及待地驅趕。GT「啟導本」指出「催逼」也可以譯作「他要像打發不要了的新婦一樣，給她厚禮送她走」——驅趕的動作伴隨禮物（指向要金器銀器的吩咐）。

## 按書卷累積

### 出埃及記
<!-- accumulation:出埃及記:11:start -->
#### [[02 出埃及記/第11章|第11章]]
- 本章重點：v1「催逼」原文意為「無法容忍而以暴力驅趕」—— CT指出都從這地出去是一人一蹄都不留，全部男女老少和牛群羊群，埃及人都要把他們驅趕離開
- 與本章關聯：「催逼」與 v8 摩西預言的「你這一切臣僕都要俯伏來見我」配合——預告末災後法老與其臣僕反過來求以色列離開；與 v10「法老心剛硬不容」對應；與第12章出12:31 應驗「起來，連你們帶以色列人都從我民中出去」對應

## 主題發展

- 法老態度反轉：第5章「不容」（出5:2）→第8章條件性「容你們去」（出8:8）→第10章摩西要求「連一蹄也不留下」（出10:26）→第11章預告「催逼」（出11:1）→第12章應驗（出12:31-33）
- 催逼不只是放人，而是主動驅趕——CT 屬靈意義：世人「不見棺材不流淚」由未經歷神懲罰的不容到經歷末災後的催逼驅趕
- 「催逼」同時搭配「要金器銀器」—— 驅趕伴隨厚禮，是神智慧的雙重應驗
