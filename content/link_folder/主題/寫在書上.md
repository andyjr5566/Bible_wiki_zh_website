# 寫在書上

## 定義

祭司將咒詛的話（誓約條款）寫在書卷/捲軸上，作為法律文書的固定化步驟（民5:23）。書寫行為將口頭咒詛轉化為有形文本，準備進行下一步的塗抹儀式。
