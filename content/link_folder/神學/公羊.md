# 公羊

## 定義

亞伯拉罕在摩利亞山舉目觀看，見一隻公羊兩角扣在稠密的小樹中，就取了那隻公羊「代替他的兒子」獻為燔祭（創廿二13）。這是聖經中替代祭物的敘事本源：以撒被捆綁躺在壇上，最終流血的卻是神所預備的公羊——神的預備使替代成為可能，豫表「神的羔羊，除去世人罪孽的」（約一29）。註解進一步讀出：公羊的兩角被扣在小樹中不能脫身，正如基督甘心被拘限在人性裡、不從十字架上下來；而角本身表徵基督剛強的神性與能力（CT／GT）。
