# 「冒煙的爐與火把」的象徵

## 定義

「不料有冒煙的爐並燒著火把從那些肉塊中經過」（創十五17）。冒煙的爐象徵神公義與十字架苦難的試煉，燒著的火把象徵聖靈光照與神憐憫的救拔。
