# 挪亞與亞當的平行

## 定義

創九1-7神對挪亞的賜福與創一28神對亞當的賜福形成平行，挪亞被視為「第二亞當」。

#### 第9章
- 本章重點：挪亞像亞當一樣領受生養眾多、遍滿地面及管理活物的使命，卻也在所栽種的果實前失去節制並顯露赤身。
- 與本章關聯：平行顯明洪水後是新開始卻不是全新無罪創造；祝福與使命延續，罪與羞恥也再次進入家庭。

## 主題發展

### 平行對照

| 亞當（創一28-29） | 挪亞（創九1-7） |
|-------------------|-----------------|
| 要生養眾多 | 要生養眾多 |
| 遍滿地面 | 遍滿了地 |
| 治理這地 | 管理一切活物 |
| 只吃菜蔬 | 可以吃肉（但不可吃血） |
| 動物和人和諧共存 | 動物驚恐懼怕人 |

### 關鍵差異

- 洪水後動物不再與人和諧共存，要等到彌賽亞再來才恢復（賽十一6-9）
- 人在無罪時用「愛」統治；墮落的人用威嚴手段統治
- 洪水後新增了[[血的尊重]]的禁令和[[人的尊嚴與神的形像|保護生命的死刑制度]]

### 挪亞如同亞當

- 挪亞是新人類的始祖，如亞當是舊人類的始祖
- 但挪亞的失敗（醉酒）也與亞當的失敗（吃禁果）平行——新人類仍有舊罪性
