# 摩西代禱熄滅神怒（詩106：23）

## 定義

詩篇 106:23 記載摩西站在破口前代禱，使耶和華的怒氣轉消，與民數記 11:2 百姓向摩西哀求、摩西禱告火熄滅相呼應，顯示摩西作中保的角色。
