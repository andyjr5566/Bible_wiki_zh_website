# 血

## 定義

在利未記第1章中，血代表祭牲的生命，是贖罪與獻祭程序中不可或缺的核心元素。經文規定，公牛、綿羊或山羊被宰殺後，亞倫子孫作祭司的要奉上血，並把血灑在會幕門口、壇的周圍；若獻鳥為燔祭，鳥的血要流在壇的旁邊。血的正確處理與灑放確立了祭壇的潔淨與獻祭的有效性。
