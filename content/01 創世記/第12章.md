# 創世記 第12章

1. [[亞伯拉罕之約|耶和華對亞伯蘭說]]：你要離開本地、本族、父家，往我所要指示你的地去。
2. [[亞伯拉罕之約|我必叫你成為大國]]。我必賜福給你，叫你的名為大；你也要叫別人得福。
3. 為你祝福的，我必賜福與他；那咒詛你的，我必咒詛他。[[亞伯拉罕之約|地上的萬族都要因你得福]]。
4. [[信心的本質|亞伯蘭就照著耶和華的吩咐去了]]；[[羅得]]也和他同去。亞伯蘭出[[哈蘭]]的時候年七十五歲。
5. [[亞伯蘭]]將他妻子[[撒萊]]和姪兒[[羅得]]，連他們在[[哈蘭]]所積蓄的財物、所得的人口，都帶往[[迦南地]]去。他們就到了迦南地。
6. 亞伯蘭經過那地，到了[[示劍（城）|示劍]]地方、摩利橡樹那裡。那時迦南人住在那地。
7. 耶和華向亞伯蘭顯現，說：我要把這地賜給你的後裔。亞伯蘭就在那裡為向他顯現的耶和華[[帳棚與祭壇|築了一座壇]]。
8. 從那裡他又遷到[[伯特利]]東邊的山，支搭[[帳棚與祭壇|帳棚]]；西邊是伯特利，東邊是艾。他在那裡又為耶和華[[帳棚與祭壇|築了一座壇]]，求告耶和華的名。
9. [[漸漸的墮落|後來亞伯蘭又漸漸遷往南地去]]。
10. [[亞伯蘭下埃及是否出於不信|那地遭遇饑荒]]。因饑荒甚大，亞伯蘭就下埃及去，要在那裡暫居。
11. 將近埃及，就對他妻子[[撒萊]]說：我知道你是容貌俊美的婦人。
12. 埃及人看見你必說：這是他的妻子，他們就要殺我，卻叫你存活。
13. [[亞伯蘭稱撒萊為妹子的性質|求你說，你是我的妹子]]，使我因你得平安，我的命也因你存活。
14. 及至亞伯蘭到了埃及，埃及人看見那婦人極其美貌。
15. 法老的臣宰看見了他，就在法老面前誇獎他。那婦人就被帶進法老的宮去。
16. 法老因這婦人就厚待亞伯蘭，亞伯蘭得了許多牛、羊、駱駝、公驢、母驢、僕婢。
17. 耶和華因亞伯蘭妻子[[撒萊]]的緣故，降大災與法老和他的全家。
18. 法老就召了亞伯蘭來，說：你這向我做的是什麼事呢？為什麼沒有告訴我他是你的妻子？
19. 為什麼說他是你的妹子，以致我把他取來要作我的妻子？現在你的妻子在這裡，可以帶他走吧。
20. 於是法老吩咐人將亞伯蘭和他妻子，並他所有的都送走了。

## 本章整理

### 一、[[亞伯拉罕之約]]的七層應許（1-3節）

1. **必叫你成為大國**（2節）— 後文由土地、後裔與國度逐步展開
2. **必賜福給你**（2節）
3. **叫你的名為大**（2節）— 巴別塔人要傳揚自己的名，神卻使亞伯蘭的名為大
4. **你也要叫別人得福**（2節）
5. **為你祝福的，我必賜福與他**（3節）
6. **那咒詛你的，我必咒詛他**（3節）
7. **地上的萬族都要因你得福**（3節）— 加3:8、16把這應許連於因信稱義與基督

第3節可譯作萬族「因你」或「藉你」得福，也有資料討論反身理解「以你為得福的榜樣」。無論採哪一語法重點，本段都把亞伯蘭所領受的福延伸到地上萬族。

### 二、[[信心的本質]]（1、4節）

- 神先發出命令與應許，亞伯蘭照著吩咐前行
- 來11:8把他的出發解作因信順服，並強調他出去時還不知道要往哪裡去
- 創12:1只說「往我所要指示你的地去」，第7節到達迦南後才明說要把這地賜給後裔

### 三、亞伯蘭放棄的三樣 ↔ 神回應的三樣

| 放棄 | 得著 |
|------|------|
| 本地 | 神所指示的土地 |
| 本族 | 成為大國的後裔應許 |
| 父家 | 領受祝福並使萬族得福 |

把本地解作世界、本族解作肉體、父家解作邪情私慾，是《創世記第十二章註解》的靈意應用，不是三個詞的字面意思。此約在創13:14-17、15:1-21及17:1-8繼續重申和展開。

> [!note] 放下的代價（GT 背景註釋）
> 在古代世界，土地、家族、祖業是最重要的保障：土地是生計與地位，子孫是將來與安葬送終的憑靠。亞伯蘭放棄父家的地位，等於把生死、地位、將來、保障全數放在耶和華手中。

### 四、[[帳棚與祭壇]]（7-8節）

| 象徵 | 意義 |
|------|------|
| 帳棚 | 族長移動生活的居所；來11:9-10由此說明寄居與盼望 |
| 祭壇 | 回應神顯現、求告耶和華之名的敬拜場所 |

- 亞伯蘭在示劍、伯特利附近及幔利築壇（十二7、8；十三18）；二十一33則記他在別是巴栽樹並求告耶和華的名。下埃及時沒有築壇或求告神的記載
- 中文傳統註釋把「先築壇、後搭棚」應用為先恢復與神關係，再建立寄居生活；又以祭壇對照前章巴別塔。這些是敘事次序上的屬靈應用

### 五、伯特利與艾（8節）

- [[伯特利]]意為「神的家」，艾的名稱常解作「廢墟」
- 亞伯蘭把帳棚搭在兩地之間，經文用西、東標示位置，沒有說他面向哪一城
- 《創世記第十二章註解》據兩地名稱發展「離開天然生命、進入神家」的靈意；這不是第8節直接陳述的行進方向

### 六、[[漸漸的墮落]]（9-10節）

《創世記第十二章註解》把以下次序整理為「漸漸的墮落」：

1. 在伯特利附近築壇、求告神（8節）
2. 漸漸遷往南地（9節）
3. 因嚴重饑荒下埃及暫居（10節）
4. 以妹子之說保全自己（11-13節）

南地是迦南南部較乾旱的區域，也在通往埃及的路線上。遷往南地本身是地理陳述；把它解作屬靈邊界或失敗起點，是該註釋從後續埃及事件回讀出的應用。KC 同樣提醒：住在邊界地帶對信徒向來危險——在伯特利，亞伯蘭沒有饑荒。

### 七、亞伯蘭下埃及的失敗（10-20節）

#### 來源對亞伯蘭行動的評價

| 來源整理 | 內容 |
|------|----------|
| 歷史背景 | 埃及靠尼羅河供應糧食，是迦南饑荒時常見的避難地；「暫居」也表示亞伯蘭沒有放棄迦南 |
| 多數本章註釋 | 經文沒有記亞伯蘭求問神，後續恐懼與欺瞞顯出他依靠自保方法 |
| 經文明文 | 沒有直接宣判下埃及本身為罪，卻詳細記下妹子之說造成撒萊被帶入宮及法老受災 |

詳見[[亞伯蘭下埃及是否出於不信]]。

#### 屬世祝福成為屬靈難處

- 法老賜亞伯蘭許多牛羊、駱駝、僕婢（16節）
- 部分傳統註釋推測所得婢女可能包括後來稱為埃及使女的夏甲（創16:1），再連於以實瑪利事件；經文沒有說夏甲在本次進入亞伯蘭家
- 埃及段落沒有記亞伯蘭築壇或求告神；註釋據此與他在迦南的生活作屬靈對照

#### 神的保護和信實

- 神降大災與法老全家（17節）— **耶和華比法老大得多**
- 亞伯蘭的失敗**並未廢除神的信實**
- 法老三次質問亞伯蘭；敘事沒有記亞伯蘭答辯

#### [[亞伯蘭下埃及與出埃及|與出埃及的預表]]

| 亞伯蘭下埃及 | 以色列出埃及 |
|--------------|--------------|
| 因饑荒下埃及 | 因饑荒下埃及 |
| 在埃及的艱辛 | 在埃及為奴 |
| 神降大災與法老家 | 神降十災與埃及 |
| 攜帶財物出埃及 | 攜帶埃及財物出埃及 |

### 八、迦南地的戰略意義

- 迦南位於埃及與敘利亞、美索不達米亞之間，是古代近東陸路往來的重要走廊
- 本章卻先強調「那時迦南人住在那地」：土地已應許給亞伯蘭後裔，當下仍由別族居住
- 亞伯蘭沒有立即取得土地，而以帳棚、祭壇和遷徙生活回應應許
