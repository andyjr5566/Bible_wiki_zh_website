# 試探的本質

## 定義

創世記 3 章中，蛇對女人的引誘展示了試探的本質：懷疑神的話、扭曲神的命令、否認犯罪的後果、引誘人驕傲。

## 主題發展

### 神學意義／研讀意義

- 魔鬼的引誘策略（至今不變）：
  1. **懷疑神的話**：「神豈是真說」
  2. **誇大限制**：把「只有一棵樹不可吃」變成「所有樹都不許吃」
  3. **否認後果**：「你們不一定死」
  4. **引誘驕傲**：「你們便如神」
- 夏娃的六處更改神的話：
  - 省去「隨意」、省去「任何樹」、改「樹」為「果子」、把生命樹改為分別善惡樹在園中、增添「不可摸」、把「必定死」改為「免得死」
- 不認識神的人濫用神的話，給魔鬼有機可乘
- 我們必須在靈裡警覺，不隨意更改神的話（啟二二18-19）
