# 雅各騙取祝福

## 定義

雅各在利百加的設計下，穿上以掃的衣服、用山羊羔皮包裹手和頸項，冒充以掃騙取父親以撒的祝福。這是創世記的核心事件之一，標誌著亞伯拉罕之約的祝福正式歸於雅各而非以掃。

- 以撒年老（137歲）、眼睛昏花，打算臨終前祝福長子以掃（1-4節）
- 利百加聽見後設計讓雅各冒充以掃（5-17節）
- 雅各三次說謊：自称是以掃、說自己打獵回來、妄稱耶和華之名（19-20節）
- 以撒三次懷疑但仍被騙：速度太快（20節）、聲音是雅各的（22節）、觸覺被羊皮矇騙（22-23節）
- 雅各與父親親嘴，以撒聞衣服香氣後祝福（26-29節）
- 祝福一經出口不能更改，以掃放聲痛哭也得不著（34-38節；來十二17）

## 按書卷累積

### 創世記

<!-- accumulation:創世記:27:start -->
#### 第27章
- 本章重點：利百加設計→雅各三次說謊→以撒三次懷疑→祝福給了雅各→以掃痛哭與怨恨
- 與本章關聯：
  - 利百加偏愛雅各，用計謀成全神的旨意
  - 雅各虽有屬靈渴望（長子名分），但手段不正當
  - 以撒偏愛以掃，不理會神「大的要服事小的」的諭言
  - 神的旨意通過人的錯誤得以成就
- 四大來源討論：

**CT**：
- 以撒的錯：偏愛以掃、貪吃美味、不理會神的旨意
- 利百加的錯：未等候神，教唆兒子欺騙丈夫
- 雅各的錯：用詭計騙取祝福、妄稱神名
- 以掃的錯：企圖奪回自己所賣的福分
- 靈意註解：「聲音是雅各的聲音，手卻是以掃的手」代表言行不符
- 山羊羔皮豫表基督的救贖

**GT**：
- 以撒137歲推算方法（雅各77歲離家+以撒60歲生子）
- 祝福（berakah）與長子名分（bekorah）原文發音相近
- 親嘴被用作欺騙手段（平行猶大出賣耶穌）
- 以撒對以掃的祝福是變相的咒詛（原文「遠離肥土」而非「必為你所住」）

**KC**：
- 雅各名字意為「詭詐、篡奪」，名如其人
- 以掃說「我的長子名分他取了去；如今他又把我應得了祝福取了去」
- 神容許這件事發生，顯明神主權的揀選

**BH**：
- 以撒的瞎眼是物理和屬靈的雙重心盲
- 雅各說「耶和華你的神」而非「我的神」——對神缺乏主觀認識
- 親嘴的氣味引發以撒的祝福，衣服上的香氣代表在神面前的地位
- 祝福是「prophetic and binding pronouncement」

## 主題發展

- 神使用人的錯誤成就祂的計畫（羅九16；徒四28）
- 祝福的不可更改性是古代近東文化的共同理解
- 雅各日後也被拉班欺騙（創三十一），經歷「你怎樣待人，人也必怎樣待你」
- 雅各臨終祝福時已學會順服神的旨意（創四十八14-20）
- 希伯來書以雅各的信心為正面例子（來十一21）
