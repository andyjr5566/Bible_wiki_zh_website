# 第六年三倍出產的應許（信心試驗）

## 定義

神預知百姓會擔心安息年和禧年不耕種時的食物來源，因此應許「我必在第六年將我所命的福賜給你們，地便生三年的土產」。第八年耕種時仍要吃陳糧，直到第九年出產收來。這是對以色列人信心的試驗，要求他們遵行神的律例並信靠神的供應。表徵若遵行神的命令就必蒙神賜福，神的恩典綽綽有餘。
