# 哥轄子孫搬運至聖之物

## 定義

指哥轄子孫在會幕起營時，受亞倫和他兒子指派，負責抬運「至聖之物」（約櫃、陳設餅桌子、燈臺、金香壇、銅壇及聖所器具）的職任。但必須由亞倫和他兒子先將聖物遮蓋妥當，哥轄人才可進來抬運，不可親手摸聖物，免得死亡（民4:4-15, 19-20）。
