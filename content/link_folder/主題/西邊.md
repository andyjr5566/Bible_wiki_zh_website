# 西邊

## 定義

第18-23節描述西邊由以法蓮營為首，包括瑪拿西、便雅憫三支派，共十萬零八千一百名，作第三隊往前行。

- 本章重點：西邊營陣編組與人數，列為第三隊。
- 與本章關聯：西邊位置彰顯神對各支派的平衡安排。
