# 以色列繞行西珥山

## 定義

以色列人出埃及後，在曠野漂流期間，照耶和華吩咐從紅海的路轉向曠野，在西珥山（以掃子孫居住之地）繞行了許多日子（三十八年），直到那世代的兵丁從營中滅盡。這段期間神看顧以色列，使他們一無所缺。
