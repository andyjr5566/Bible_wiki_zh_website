# 神悅納奉獻

## 定義

民數記7章結尾（89節）記載：摩西進會幕要與耶和華說話時，「聽見法櫃的施恩座以上、二基路伯中間有與他說話的聲音，就是耶和華與他說話」。這發生在十二日獻壇禮完成之後（88節），表明神悅納了以色列首領們按吩咐獻上的供物，並以在施恩座上與摩西說話（同在、引導）作為悅納的印證。這呼應創4:4「耶和華看中了亞伯和他的供物」、瑪3:10「以此試試我、是否為你們敞開天上的窗戶」、來13:16「這樣的祭是神所喜悅的」。
