# 馨香之氣

## 定義

創八21「耶和華聞那馨香之氣」指神悅納挪亞所獻的燔祭。新約以弗所書五2指出基督愛我們，為我們捨了自己，當作馨香的供獻和犧牲獻給神。

- 與本章關聯：馨香表達祭物蒙神悅納；來源並將其連於基督完全奉獻所成就的蒙悅納根基。

## 主題發展

神悅納的不是挪亞本身，而是**基督的功績**。信徒蒙悅納非因自己良善，全然因著基督。

### 創八20-21

- 20節：挪亞築壇獻燔祭——聖經首次使用「壇」字
- 21節：耶和華聞那馨香之氣，就心裡說不再咒詛地

### 燔祭的意義

- 燔祭：完全焚燒，不准吃用；預表基督的贖罪之死
- 潔淨的牲畜：豫表基督為人在地上的完全
- 潔淨的飛鳥：豫表基督從天降下
