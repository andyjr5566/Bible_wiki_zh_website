# 以撒豫表基督

## 定義

創世記二十二章亞伯拉罕獻以撒的事件，是聖經中最完整的豫表之一。亞伯拉罕豫表父神，以撒豫表耶穌基督。以撒背柴、被捆綁、放在壇上、似死而復活，一一對應基督的救贖工作。

| 角色 | 豫表 | 經文依據 |
|------|------|----------|
| 亞伯拉罕 | 父神不愛惜獨生子 | 羅八32；約三16 |
| 以撒負柴 | 耶穌背十字架 | 約十九17 |
| 以撒被捆綁 | 耶穌甘心捨命 | 約十18 |
| 第三日得回 | 耶穌第三日復活 | 來十一19 |
| 公羊代贖 | 基督代死 | 約一29；林後五21 |

## 主題發展

以撒的豫表性生涯在摩利亞山事件中一次被呈獻出來。以撒是立約後的第一個果子，預表基督。
