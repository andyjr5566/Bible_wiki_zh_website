# 我以為這地方的人總不懼怕神

## 定義

創世記二十章11節，亞伯拉罕解釋稱妻為妹的原因：「我以為這地方的人總不懼怕神，必為我妻子的緣故殺我。」——憑眼見不憑信心，以狹窄心胸度量人。

亞伯拉罕的信心失足根源：「我以為」——我們的想法常靠不住。誤會別人、以狹窄心胸度量人，是人際難處的最大根源。平安時容易信靠神，面臨危險時便容易失去信心。
