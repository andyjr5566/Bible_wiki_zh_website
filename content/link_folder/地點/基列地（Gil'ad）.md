# 基列地（Gil'ad）

## 定義

基列地是約旦河東邊的地區，民數記第32章記載流便、迦得二支派請求在此得地為業，摩西應允並分給他們及瑪拿西半支派；希伯來文「גִּלְעָד」（Gil'ad）。
