# 示路蔑（西緬族長）

## 定義

蘇利沙代的兒子，西緬支派族長（民1:6）。在十二支派族長名單中排第二（民1:6, 16），協助數點西緬男丁59,300名（民1:22-23）。名字意為「神是我的平安」或「神是酬報」。
