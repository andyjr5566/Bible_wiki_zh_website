# 出埃及記 第36章

1. [[比撒列和亞何利亞伯的配搭|比撒列和亞何利亞伯]]，並一切心裡有智慧的，就是[[蒙靈充滿的智慧恩賜（比撒列、亞何利亞伯）|蒙耶和華賜智慧聰明]]、叫他知道做聖所各樣使用之工的，都要[[山上的樣式|照耶和華所吩咐的]]做工。
2. 凡[[蒙靈充滿的智慧恩賜（比撒列、亞何利亞伯）|耶和華賜他心裡有智慧]]、而且[[蒙靈充滿的智慧恩賜（比撒列、亞何利亞伯）|受感前來做這工的]]，摩西把他們和[[比撒列（Bezalel）|比撒列]]並[[亞何利亞伯（Oholiab）|亞何利亞伯]]一同召來。
3. 這些人就從摩西收了以色列人為做聖所並聖所使用之工所拿來的禮物。百姓每早晨還把[[甘心樂意的奉獻|甘心獻的禮物]]拿來。
4. 凡做聖所一切工的智慧人各都離開他所做的工，
5. 來對摩西說：百姓為耶和華吩咐使用之工所拿來的，富富有餘。
6. 摩西傳命，他們就在全營中宣告說：無論男女，[[奉獻過剩與節制|不必再為聖所拿什麼禮物來]]。這樣才攔住百姓不再拿禮物來。
7. 因為他們所有的[[奉獻過剩與節制|材料夠做一切當做的物，而且有餘]]。
8. 他們中間，凡心裡有智慧做工的，用[[十幅幔子（內層幕幔）|十幅幔子]]做[[會幕（帳幕整體）|帳幕]]。這幔子是[[比撒列（Bezalel）|比撒列]]用[[撚的細麻]]和藍色、紫色、朱紅色線製造的，並用巧匠的手工繡上基路伯。
9. 每幅幔子長二十八肘，寬四肘，都是一樣的尺寸。
10. 他使這五幅幔子幅幅相連，又使那五幅幔子幅幅相連；
11. 在這相連的幔子末幅邊上做藍色的鈕扣，在那相連的幔子末幅邊上也照樣做；
12. 在這相連的幔子上做五十個鈕扣，在那相連的幔子上也做五十個鈕扣，都是兩兩相對；
13. 又做[[五十個金鉤（vavei zahav）|五十個金鉤]]，使幔子相連。這才成了一個[[會幕（帳幕整體）|帳幕]]。
14. 他用[[十一幅山羊毛幔子（罩棚）|山羊毛織十一幅幔子]]，作為[[會幕（帳幕整體）|帳幕]]以上的[[十一幅山羊毛幔子（罩棚）|罩棚]]。
15. 每幅幔子長三十肘，寬四肘；十一幅幔子都是一樣的尺寸。
16. 他把五幅幔子連成一幅，又把六幅幔子連成一幅；
17. 在這相連的幔子末幅邊上做五十個鈕扣，在那相連的幔子末幅邊上也做五十個鈕扣；
18. 又做[[五十個銅鉤]]，使[[十一幅山羊毛幔子（罩棚）|罩棚]]連成一個；
19. 並用[[染紅的公羊皮蓋子|染紅的公羊皮]]做[[染紅的公羊皮蓋子|罩棚的蓋]]，再用[[海狗皮頂蓋|海狗皮]]做一層[[海狗皮頂蓋|罩棚上的頂蓋]]。
20. 他用[[皂莢木豎板|皂莢木做帳幕的豎板]]。
21. 每塊長十肘，寬一肘半；
22. 每塊有兩榫相對。[[會幕（帳幕整體）|帳幕]]一切的[[皂莢木豎板|板]]都是這樣做。
23. [[會幕（帳幕整體）|帳幕]]的南面做[[皂莢木豎板|板]]二十塊。
24. 在這二十塊[[皂莢木豎板|板]]底下又做四十個帶卯的銀座：兩卯接這塊板上的兩榫，兩卯接那塊板上的兩榫。
25. [[會幕（帳幕整體）|帳幕]]的第二面，就是北面，也做[[皂莢木豎板|板]]二十塊
26. 和帶卯的銀座四十個：這[[皂莢木豎板|板]]底下有兩卯，那板底下也有兩卯。
27. [[會幕（帳幕整體）|帳幕]]的後面，就是西面，做[[皂莢木豎板|板]]六塊。
28. [[會幕（帳幕整體）|帳幕]]後面的[[拐角板（雙板）|拐角做板兩塊]]。
29. [[拐角板（雙板）|板的下半截是雙的]]，上半截是整的，直到第一個環子；在[[會幕（帳幕整體）|帳幕]]的兩個拐角上都是這樣做。
30. 有八塊[[皂莢木豎板|板]]和十六個帶卯的銀座，每塊板底下有兩卯。
31. 他用[[皂莢木（atzei shittim）|皂莢木]]做[[閂（橫閂）|閂]]：為[[會幕（帳幕整體）|帳幕]]這面的板做五閂，
32. 為[[會幕（帳幕整體）|帳幕]]那面的板做五[[閂（橫閂）|閂]]，又為帳幕後面的板做五閂，
33. 使板腰間的[[閂（橫閂）|中閂]]從這一頭通到那一頭。
34. [[金子（包金）|用金子將板包裹]]，又做板上的金環套[[閂（橫閂）|閂]]；閂也用金子包裹。
35. 他用藍色、紫色、朱紅色線，和[[撚的細麻]][[內幔（隔聖所至聖所的幔子）|織幔子]]，以巧匠的手工繡上基路伯。
36. 為幔子做[[四根包金柱子（內幔柱）|四根皂莢木柱子]]，[[金子（包金）|用金包裹]]，柱子上有[[五十個金鉤（vavei zahav）|金鉤]]，又為柱子鑄了四個帶卯的銀座。
37. 拿藍色、紫色、朱紅色線，和[[撚的細麻]]，用繡花的手工織[[帳幕門簾|帳幕的門簾]]；
38. 又為[[帳幕門簾|簾子]]做[[五根包金柱子（門簾柱）|五根柱子]]和柱子上的鉤子，[[金子（包金）|用金子把柱頂和柱子上的杆子包裹]]。柱子有五個[[帶卯的銅座（院子柱子座）|帶卯的座，是銅的]]。

## 本章整理

### 奉獻過剩與工匠配搭（v1-7）

出埃及記第三十六章標誌著會幕實體建造的正式展開。摩西首先召集了蒙神賜智慧的[[比撒列（Bezalel）|比撒列]]與[[亞何利亞伯（Oholiab）|亞何利亞伯]]，以及一切心裡受感前來做工的巧匠。這裡突顯了事奉神的兩個必要條件：一是神所賜的[[蒙靈充滿的智慧恩賜（比撒列、亞何利亞伯）|智慧與能力]]，二是人心中受感而自願的奉獻。正如新約所言，我們的承擔不是出於自己，而是出於神。在[[比撒列和亞何利亞伯的配搭]]中，我們看見了身體事奉的圖像——不同支派、不同恩賜的人在聖靈的引導下同心建造。

百姓的[[甘心樂意的奉獻]]在這段達到了高潮。他們每早晨持續把禮物拿來，以至於材料「富富有餘」。這種景象在教會歷史中極為罕見，工匠們甚至必須暫停手頭工作，向摩西反映材料已經過剩。摩西隨即在營中宣告，攔住百姓不再奉獻。這展現了[[奉獻過剩與節制]]的屬靈原則：神的工作絕不會因缺乏物質而受限，但當奉獻超過所需時，教會也當懂得「知足」並適時制止，避免浪費或出於人意的勉強。

### 幔子與罩棚的層層預表（v8-19）

隨著材料齊備，工匠們開始編織帳幕的各層覆蓋物。帳幕由內而外共有四層，每一層都精準對應基督的不同層面與工作：

| 層次 | 材料與規格 | 預表意義 |
| :--- | :--- | :--- |
| 第一層（內層） | [[撚的細麻]]與藍、紫、朱紅色線繡上基路伯的[[十幅幔子（內層幕幔）]] | 表徵基督人性的美麗與屬天榮耀（細麻表聖潔，藍色表屬天，紫色表君尊，朱紅色表犧牲） |
| 第二層 | [[十一幅山羊毛幔子（罩棚）]]，以五十個銅鉤相連 | 表徵基督站在人的地位上負起擔罪責任，成為贖罪祭 |
| 第三層 | [[染紅的公羊皮蓋子]] | 表徵基督為人流血捨命，完成救贖 |
| 第四層（外層） | [[海狗皮頂蓋]] | 表徵基督在地無佳形美容，卻堅固耐苦，保護信徒免受侵害 |

在連接的細節上，內層幕幔用[[五十個金鉤（vavei zahav）|五十個金鉤]]相連，表徵神性的能力促成完美的聯結；而罩棚則用[[五十個銅鉤]]相連，銅表審判，表徵基督擔當人的罪而受神審判。這些細節的重複記載並非冗餘，而是表明神對[[山上的樣式]]的絕對要求——每一個細節都必須照神所吩咐的做成。

### 豎板與閂的結構連結（v20-34）

帳幕的骨架由[[皂莢木（atzei shittim）|皂莢木]]製成的[[皂莢木豎板]]構成。皂莢木材質堅硬、外表粗糙，表徵人的天然本性；但這些木板都被[[金子（包金）|金子]]包裹，表徵信徒蒙救贖後披戴基督的神性。每塊板有兩榫相對，插入帶卯的銀座中。銀座表徵以救贖為根基，使信徒能在神面前站立得穩。

> [!note] 板的數目與屬靈含義
> 南北西三面共有48塊板，96個銀座。48乃6乘8，表徵舊造的人（6）在復活裡（8）成為建造教會的材料。其中西面的[[拐角板（雙板）]]下半截是雙的、上半截是整的，以環子箍住，表徵信徒在行動上彼此適應，心靈卻在聖靈裡合而為一。

板與板之間用[[閂（橫閂）|皂莢木閂]]連結，三面共十五根閂，表徵三一神使信徒有力量負起連結的責任。中閂從這頭通到那頭，而板與閂皆用金子包裹，並以金環套住，表徵聖靈所賜合而為一的心。

### 幔子、門簾與柱子（v35-38）

帳幕內部還有兩處重要的分隔。第一是[[內幔（隔聖所至聖所的幔子）|隔開聖所與至聖所的幔子]]，同樣用細麻與彩色線繡上基路伯，掛在[[四根包金柱子（內幔柱）|四根包金的皂莢木柱子]]上。這幔子表徵基督的身體，將有罪的人與聖潔的神隔開；唯有大祭司帶著血才能一年一次進入。但基督在十字架上斷氣時，這幔子從上到下裂為兩半，為我們開闢了一條又新又活的路。

第二是帳幕入口的[[帳幕門簾]]，掛在[[五根包金柱子（門簾柱）|五根包金柱子]]上。門簾表徵基督是門，凡從祂進去的必然得救。值得注意的是，內幔的柱子立於銀座上（表徵救贖），而門簾的柱子立於[[帶卯的銅座（院子柱子座）|銅座]]上。銅表審判，表徵信徒若要為基督作見證（五根柱子的見證意義），就必須先經歷神審判的煉淨。

### 跨章脈絡：從山上的樣式到實體建造

出埃及記第三十六章並非單純重複第二十六章的建造藍圖，而是將「山上的樣式」落實為「實體的建造」。神不厭其煩地記錄每一個細節，表明祂不忘記為祂所做的任何工。從百姓甘心樂意的奉獻，到工匠精準的執行，我們看見神的工作需要兩方面配合：聖靈賜下的恩賜與人甘心的順服。會幕作為神在地上的居所，其每一層幔子、每一塊木板、每一個鉤子，都指向那最終「道成肉身，住在我們中間」的耶穌基督，以及祂所建立的教會——神藉著聖靈居住的所在。
