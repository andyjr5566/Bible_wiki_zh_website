# 永恆國度

## 定義
但以理書2:44「在這些王的時候、天上的神必另立一國、永不敗壞、國權也不歸別國的人、卻要打碎滅盡那一切國、這國必存到永遠」。基督所建立、超越時空、永不廢棄的神國（路1:33「他的國沒有窮盡」）。已今不完全、終極完全（已今：教會時代靈性掌權；終極：基督再來千禧年、新天新地）。

## 主題發展
- 永恆國度三階段：①預表（以色列神權國）；②已今（基督首次降臨、教會時代屬靈掌權）；③終極（再來、千禧年、新天新地）
- 國度核心：神在基督裡作王、萬有歸一（弗1:10）、公義平安喜樂為標誌（羅14:17）
- 信徒當前屬天國子民（腓3:20）、未來與基督同作王（提後2:12；啟20:6）
