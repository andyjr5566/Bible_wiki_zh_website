# 獻壇禮

## 定義

民數記第7章記載會幕建成、膏抹成聖後，以色列十二支派首領連續十二日，每日一支派來行奉獻壇之禮（第10-11、84、88節）。各支派獻上相同的供物：銀盤、銀碗、金盂、燔祭、贖罪祭、平安祭（第13-83節）。此禮儀確立壇的功能，展現十二支派同心合意、樂獻奉獻，神悅納其供物。
