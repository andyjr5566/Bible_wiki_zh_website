# 以諾

## 定義

希伯來文 *Enoch*（חֲנוֹךְ），意思是「開始、學習、教師、奉獻」。雅列的兒子，瑪土撒拉的父親。與神同行三百年，被神取去，未經死亡。共活三百六十五歲。

以諾是本章最核心的人物——十代先祖中唯一「不死」的例外。「與神同行三百年」是聖經中極獨特的記載；神將他取去，他就不在世了。

## 主題發展

### 神學意義／研讀意義

- **「與神同行」的定義**：亦步亦趨，不敢落後也不敢超前；以新約的話說，就是以基督耶穌的心為心（腓二5），凡事蒙祂引領
- 與神同行的榜樣：
  1. 順服神是長久不斷的事，不是一時的
  2. 「生兒養女」並不攔阻與神同行——兒女眾多不能作為沒時間讀經禱告的藉口
  3. 屬靈的人不是「怪人」，越是屬靈越像正常的人
  4. 家庭生活中一切大小事，以敬畏事奉神的態度處理，正是與神同行的明證
- **唯一不死的例外**（與以利亞同，王下二11）
- 因著信被接去，不至於見死（來十一5）
- 曾預言基督降臨施行審判（猶14-15）——聖經最早的一位先知
- **豫表災前得勝被提的信徒**（帖前四14-17；啟三10）
- 屬靈歷程：「以諾」（教師、學習）——開始學習認識神的心意
