# 樹木

## 定義

麥比拉交易的標的之一：「於是，麥比拉、幔利前、以弗崙的那塊田和其中的洞，並田間四圍的樹木，都定準歸與亞伯拉罕」（創廿三17）。契約把田地、洞與四圍的樹木一併列明，顯示產業界線劃定得完整周全，日後無人能就任何一項提出爭議。註解在靈意上讀出：樹木代表強韌不衰的生命，栽種生長、耐力持久；信徒的肉身雖然葬在地裡，這塊墳地卻四圍長滿樹木，滿了復活的盼望——埋葬不是結束，而是等候收成（CT）。
