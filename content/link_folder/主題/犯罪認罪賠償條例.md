# 犯罪認罪賠償條例

## 定義

耶和華對摩西說：無論男女，若犯了人所常犯的罪，干犯耶和華，那人就有了罪。他要承認所犯的罪，將所虧負人的如數賠還，另外加上五分之一，歸與所虧負的人；若無親屬受賠償，則歸與服事耶和華的祭司，外加贖罪的公羊（民5:5-8）。此條例強調認罪、賠償、加五分之一、祭司代受三大原則。
