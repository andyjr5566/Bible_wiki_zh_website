# 嫉妒

## 定義

指丈夫「生了疑恨的心，疑恨他妻子」（民5:14, 30），原文 qin'ah 含「熱心、嫉妒、憤怒」之意。此嫉妒非單純情緒，乃盟約關係中對專一忠貞的維護，驅動丈夫依條例帶妻至耶和華面前接受苦水試驗。

## 主題發展

本章將嫉妒制度化為「嫉妒條例」（v29-30），使主觀懷疑轉為客觀神聖程序，避免私刑與誣告，彰顯神對婚姻聖潔與公義的看顧。
