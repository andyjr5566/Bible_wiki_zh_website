# 復活盼望

## 定義

創世記第23章記載撒拉之死與埋葬，其核心神學主題是**復活盼望**。亞伯拉罕在迦南所得的唯一永久性產業竟是一塊墳地，但這不是失敗的記號，而是信心的「種下」——等候將來的收割與復活。

## 主題發展

### 神學或研讀意義

- **埋葬 = 種下**：種子埋入土裏等候收割——死亡不是終結（KC）
- **復活次序**：林前15:20-21——在基督裏的人都將復活
- **身體就是人**：「我的妻子撒拉」而非「遺留物」；耶穌是「耶穌」而非「遺留物」（KC）
- **神知道祂子民在哪裏**：以色列如今看似「死了」，但神埋葬祂的子民，知道他們的位置（KC）
- **墳地為應許初熟果子**：亞伯拉罕在迦南唯一產業是墳地，證明神必成就全地應許
- **埋葬 vs 焚燒**：埋葬=復活盼望；焚燒=審判（書七25；利廿一9）（KC）
- **六代族長同葬**：亞伯拉罕、撒拉、以撒、利百加、雅各、利亞皆葬麥比拉洞——舊約信徒存信心而死，等候與新約信徒同得更美復活（來十一39-40）
- **亞伯拉罕的信心進程**：「神」（Elohim）→「全能神」（El Shaddai）→「耶和華」（YHWH）——亞伯拉罕的信心認識神是步步進階的（CT）
