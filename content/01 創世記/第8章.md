# 創世記 第8章

1. [[神的記念|神記念]][[挪亞]]和挪亞方舟裡的一切走獸牲畜。神叫[[風與靈同字|風吹地]]，水勢漸落。
2. 淵源和天上的窗戶都[[審判已過|閉塞了]]，天上的大雨也[[審判已過|止住了]]。
3. 水從地上漸退。過了[[四十天降雨與一百五十天水勢|一百五十天]]，水就漸消。
4. 七月十七日，方舟停在[[亞拉臘山]]上。
5. 水又漸消，到十月初一日，[[審判已過|山頂都現出來了]]。
6. [[信心與忍耐|過了四十天]]，[[挪亞]]開了方舟的窗戶，
7. 放出一隻[[烏鴉與鴿子的象徵|烏鴉]]去；那烏鴉飛來飛去，直到地上的水都乾了。
8. 他又放出一隻[[烏鴉與鴿子的象徵|鴿子]]去，要看看水從地上退了沒有。
9. 但遍地上都是水，鴿子找不著落腳之地，就回到方舟[[挪亞]]那裡，[[挪亞]]伸手把鴿子接進方舟來。
10. 他又等了七天，再把鴿子從方舟放出去。
11. 到了晚上，鴿子回到他那裡，嘴裡叼著一個新擰下來的[[橄欖葉子]]，[[挪亞]]就知道地上的水退了。
12. 他又等了七天，放出鴿子去，鴿子就不再回來了。
13. 到[[挪亞]]六百零一歲，正月初一日，地上的水都[[乾的兩個希伯來字|乾了]]。[[挪亞]]撤去方舟的蓋觀看，便見地面上[[乾的兩個希伯來字|乾了]]。
14. 到了二月二十七日，地就都[[乾的兩個希伯來字|乾了]]。
15. [[神的記念|神]]對[[挪亞]]說：
16. 你和你的妻子、兒子、兒婦都可以出方舟。
17. 在你那裡凡有血肉的活物，就是飛鳥、牲畜，和一切爬在地上的昆蟲，都要帶出來，叫他在地上[[新造與新生活|多多滋生，大大興旺]]。
18. 於是[[挪亞]]和他的妻子、兒子、兒婦都出來了。
19. 一切走獸、昆蟲、飛鳥，和地上所有的動物，各從其類，也都出了方舟。
20. [[挪亞]]為耶和華[[祭壇敬拜|築了一座壇]]，拿各類潔淨的牲畜、飛鳥獻在壇上為[[燔祭]]。
21. 耶和華聞那[[馨香之氣]]，就心裡說：我不再因人的緣故[[神的恩典之約|咒詛地]]（[[人的全面敗壞|人從小時心裡懷著惡念]]），也不再按著我才行的[[挪亞之約|滅各種的活物了]]。
22. [[洪水後自然秩序的保存|地還存留的時候，稼穡、寒暑、冬夏、晝夜就永不停息了]]。

## 本章整理

### 神記念，水勢漸退（1-5節）

「神記念挪亞」不是說全知的神一度忘了方舟。CT、GT、KC與BibleHub一致把「記念」解作關注、顧念，並開始為所記念者行動；神不只記念挪亞，也顧念方舟裡的走獸牲畜。洪水的水源閉塞、大雨止住、水勢逐步下降，顯明審判轉向恢復。

神沒有使水瞬間消失，而是讓水逐漸退去。KC以此勸勉信徒：神可使試煉停止，試煉留下的後果卻可能需要時間消退。這是牧養應用；經文直接描述的是受神掌管的漸進退水過程。

「神叫風吹地」中的希伯來文 *ruach* 也可指「靈」。在本句語境中，它直接描述使水退去的風；CT、GT與BibleHub則把它與創1:2神的靈運行在水面上相照，視為再創造。CT更以神的記念、方舟及風分別連於父、子、靈的救恩工作，稱為[[三一神同工]]。這是基督教神學式的豫表閱讀，不是說希伯來文一句話明列三個位格。

方舟於七月十七日停在「亞拉臘眾山」。GT所引資料多把這理解為今日土耳其東部、凡湖附近的古烏拉爾圖山區，而非精確指明大亞拉臘峰。GT也保存俄國探險隊、宗教人士見到冰中船體的傳聞；這些敘述欠缺可核查的原始紀錄、材料鑑定與可靠定年。BibleHub同樣指出尋找精確遺址尚無決定性證據。

### 洪水消退時間軸（1-14節）

| 時點 | 事件 | 資料性質 |
|---|---|---|
| 挪亞600歲，2月17日 | 洪水爆發（7:11） | 經文明載 |
| 7月17日 | 方舟停在亞拉臘山區（8:4） | 經文明載 |
| 10月1日 | 山頂顯露（8:5） | 經文明載 |
| 再過40天 | 挪亞開窗，放出烏鴉（8:6-7） | 經文明載間隔 |
| 其後三次放鴿 | 第二、三次前各等7天（8:8-12） | 首次與烏鴉相隔7天是來源推算 |
| 挪亞601歲，1月1日 | 地面沒有積水，挪亞撤去方舟蓋（8:13） | 經文明載 |
| 2月27日 | 地全然乾透，神命挪亞出方舟（8:14-16） | 經文明載 |

GT第7章拾穗時間表以三十日月份推算若干日期，並給出371日總數；GT第8章所收另一研讀本則說挪亞一家在方舟共377天，包含洪水前七天的等候。兩者的差別來自「只算洪水起訖」或「連進舟後七天也算」以及首尾日算法，不能把任一總數當成單節經文明言。可確定的是：洪水從六百歲二月十七日開始，到六百零一歲二月二十七日地全乾，來源稱為一年又十天；挪亞在洪水前已進舟等候七天。

第13、14節的「乾了」使用兩個不同希伯來字。GT指出，13節著重地面不再有積水，14節則描述土地乾透。挪亞在正月初一已親眼看見地面乾了，仍等到二月二十七日神說話才出方舟；CT與GT以這五十多天說明[[信心與忍耐]]及不憑眼見擅自行動。

### 烏鴉、鴿子與橄欖葉（6-12節）

| 鳥與次序 | 敘事中的作用 | 來源的靈意閱讀 |
|---|---|---|
| 烏鴉 | 飛來飛去直到地面乾；食腐、能在較差環境停留 | CT、GT、KC以不潔淨的烏鴉象徵舊造或肉體 |
| 第一次鴿子 | 找不到落腳地，返回方舟 | 以基督為安息，不以受審判之物為落腳處 |
| 第二次鴿子 | 叼回新擰的橄欖葉，顯示植物恢復 | 橄欖與油連於聖靈，嫩葉象徵新生命 |
| 第三次鴿子 | 不再返回，顯示外界已可棲息 | CT解作聖靈在潔淨生命中安家 |

鳥的行為首先是挪亞觀察水與土地狀態的方法。經文沒有直接稱烏鴉為舊人、鴿子為聖靈；這些是來源按利11:15、太3:16等經文所作的象徵應用。GT所引丁道爾註釋還特別說，橄欖葉在本段首先是「新造的應許」，不是經文明說的政治和平標誌；後世口銜橄欖枝的和平圖像確實由此發展。

GT背景資料將本章與古代近東洪水故事比較：

| 來源 | 放鳥次序 |
|---|---|
| 創世記8章 | 烏鴉一次，鴿子三次 |
| 吉加墨斯史詩 | 鴿子、燕子、烏鴉 |
| 阿特拉哈西斯傳統 | 也有以鳥探察外界的情節 |

古代航海者可用鳥判斷陸地方向或登陸條件，但創世記的方舟此時已停在山區，所以挪亞使用鳥不是導航，也不是觀察飛行作占卜，而是判斷水退與土地復原。

### 出方舟與新造生活（15-19節）

挪亞因神的命令進方舟（7:1），也等神命令才出方舟（8:15-16）。人與各類活物離舟後，神再次吩咐活物多多滋生、大大興旺，與創1章的創造使命呼應。

| 創世記1章 | 創世記8章 |
|---|---|
| 神的靈運行在水面上（1:2） | 神叫風 *ruach* 吹地（8:1） |
| 水聚在一處、旱地顯露（1:9） | 水退、山頂與地面顯露（8:5、13） |
| 活物各從其類（1:20-25） | 動物各從其類出方舟（8:19） |
| 活物滋生眾多（1:22） | 活物多多滋生、大大興旺（8:17） |

CT與KC把出方舟到更新之地解作與基督同死同復活、行在生命新樣中（羅6:4）；彼前3:20-21則明文以挪亞一家藉水得救連於洗禮與基督復活。

### 築壇、燔祭與馨香（20-21節）

挪亞出方舟後首先為耶和華築壇。GT與BibleHub指出，這是聖經首次使用「壇」一詞。挪亞取各類潔淨牲畜和飛鳥獻為燔祭，顯示第7章保存較多潔淨活物也為洪水後獻祭留下餘量。

CT、GT與KC把祭理解為感恩、敬拜與全然奉獻，並以全然焚燒的燔祭豫表基督在十字架上的完全獻上；KC再把牲畜與飛鳥分別應用為基督在地為人及從天而來。GT所引《舊約背景註釋》較謹慎：經文沒有明說這祭的用意，也沒有稱它為感謝祭或贖罪祭；燔祭在後來制度中用途廣泛，也可與懇求相關。因此，感恩與基督豫表應保留為來源解讀，不宜改寫成20節明說的祭名功能。

「耶和華聞那馨香之氣」是擬人化語言，表示神悅納祭物。GT比較美索不達米亞洪水故事：其中眾神因洪水期間缺少供物而飢餓，像蒼蠅聚到祭物旁；舊約雖沿用「馨香」這類古代獻祭語彙，卻不描寫耶和華依賴人的食物，反把焦點放在神悅納挪亞的敬拜。CT與KC按弗5:2進一步把蒙悅納的根基連於基督的獻上。

### 人心仍惡，神仍施憐憫（21-22節）

第6章因人心終日所想盡都是惡而宣告審判；第8章同樣承認「人從小時心裡懷著惡念」，卻宣告不再照洪水所行的滅絕。KC指出，洪水審判沒有改變人的內心；上下文的轉折在於祭物蒙悅納，以及神主動選擇以憐憫保存世界。這不是神忽然發現先前判斷錯誤，也不是祂從此不再審判罪。

第8章先有神在心裡的應許，第9章才正式向挪亞一家及一切活物宣告約並賜下虹的記號。GT把這世界保存應許列為「無條件之約」，意思是經文沒有把季節繼續運作繫於人類先達成某種道德條件。

「稼穡、寒暑、冬夏、晝夜永不停息」保證的是自然節律不完全中止，不是每地每季都有豐收；GT以創26:1與41:54的饑荒說明局部中斷仍會發生。CT、GT與BibleHub按「地還存留的時候」連於彼後3:10-13的新天新地盼望。創8:21直接應許的是不再「按著我才行的」滅各種活物；創9:11、15會更明確說不再有洪水毀壞全地。

### 復活與三一神的靈訓

CT將本章整理為在基督裡死而復活：

1. 洪水消退，表示審判已過。
2. 方舟停在亞拉臘山，豫表與基督一同復活。
3. 山頂顯露，豫表與基督一同坐在天上。
4. 出方舟到更新之地，豫表行在生命新樣中。

CT又把七月十七日與基督復活相連：後來出埃及的宗教曆把原七月視為正月，逾越節在十四日，十七日遂成為第三日。這是一項傳統日期豫表；經文沒有說挪亞時代已按出埃及後的宗教曆計日，也不能據此證明方舟停靠與耶穌復活是同一現代曆日。

CT對新造生命的另一組教訓是：烏鴉表舊造性情，鴿子與橄欖新葉表聖靈和新生命，挪亞等候神的話才出舟，出舟後築壇把自己獻給神。這些教訓忠實保留來源的豫表框架，同時與敘事層面的歷史、曆法及鳥類背景分開。
