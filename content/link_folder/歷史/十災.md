# 十災

## 定義

出埃及記七14至十二36記載的神降臨埃及的十場災禍——血的災、蛙災、蝨災、蠅災、畜疫災、瘡災、雹災、蝗災、黑暗災、殺長子之災。目的為使法老容以色列人出埃及、使埃及人知道耶和華是神、審判埃及諸神。

## 主題發展

KC的結構分析：前九災分三組每組三災，第十災獨立：
1. 第1-3災：從地而出，由亞倫的杖施行，影響以色列人和埃及人，術士能仿造
2. 第4-6災：未提杖，由耶和華直接施行，以色列人免災，術士不能仿造
3. 第7-9災：從天而來，由摩西的杖施行，以色列人免災
4. 第十災：殺長子，逾越節之夜

每組首災摩西清早去見法老（七15, 八20, 九13）——指新的開始。每組末災無預告突然來臨（蝨、瘡、黑暗）。

GT命題9詳解十災中「埃及遍地」與「以色列人免災」的關係——希伯來文「所有」不是絕對的，由上下文決定；八22神「分別歌珊地」使以色列人免災。

GT啟導本列出十災的對象——埃及的偶像：尼羅河神 Hapi（第一災）、蛙神 Heqt（第二災）、牛羊神（第五災）、太陽神 Ra（第九災）、法老本身（第十災）等——「十災是對埃及大小菩薩的一次大掃除」。

KC指出啟示錄十六章七碗之災重现許多埃及之災——啟示錄的災不僅關乎以色列，也關乎普天下與有名無實的基督教。
