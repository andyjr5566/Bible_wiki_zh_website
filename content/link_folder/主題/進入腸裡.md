# 進入腸裡

## 定義

描述苦水被婦人飲下後進入其內腸（腸胃）的過程（民5:22, 24, 27）。經文強調水「入你的腸中」（5:22）、「進入他裡面」（5:24, 27），成為咒詛生效的生理通道。

- 本章重點：咒詛的水入婦人的腸中（5:22）；水進入他裡面變苦了（5:24, 27）。
- 與本章關聯：身體內部成為神審判運行的場域，咒詛透過生理機制顯現真相。
