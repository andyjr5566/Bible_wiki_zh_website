# 稱妻說謊

## 定義

族長時期重複出現的弱點：因害怕而稱妻子為妹子。亞伯拉罕如此（創十二、創二十），以撒亦如此（創廿六）。他們都因妻子容貌俊美怕被殺而說謊。

- 以撒因利百加容貌俊美，怕被殺，稱她為妹子（重蹈亞伯拉罕覆轍）
- 亞比米勒從窗戶見以撒與利百加戲玩，揭穿謊言
- 以撒缺乏信心，環境看起來危急時否認了與利百加的婚姻關係
- 亞比米勒責備並下令保護：「凡沾著這個人，或是他妻子的，定要把他治死」

## 主題發展

- 懼怕和說謊都不是信心的表現
- 神在人生中允许苦難，為要建造信心的功課
- 神保護屬祂的人，即使在他們犯錯時
