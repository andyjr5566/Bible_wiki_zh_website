# 污穢營地

## 定義

指因不潔淨之人留在營中，玷污神同在的居所。本章第 3 節明言「免得污穢他們的營；這營是我所住的」，神學意義在於神的聖潔同在要求營地儀式純全。

- 與本章關聯：奠定舊約「神與人同住」要求聖潔分離的神學基礎。
