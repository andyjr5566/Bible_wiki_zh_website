# 災殃

## 定義

出埃及記11:1「我再使一樣的災殃臨到法老和埃及」——CT 原文字義指出「災殃」原文 nagaʿ（nāḡapʿ，「擊打、疫病、疾病、疤痕」），不同於出9:14 的「災殃」（makkâ「擊打、瘟疫」）。此字在本節指第十災——一場針對法老和埃及地的擊打，「災殃」既是「擊打」的動作也是「疫病」的結果。第十災作為前九災序的壓軸災殃—— nagaʿ 在此承擔「耶和華親自擊打」的意涵（v4「我必出去」）。

## 主題發展

- 同源動詞 nāḡap 在出埃及記12:13「滅命的擊殺」、12:23「耶和華巡行擊殺」——同一字根貫穿「逾越」與「擊殺」對比
- 與希伯來文 makkâ（出9:14）的對比：makkâ 是瘟疫擊打，nagaʿ 是直接擊打——前九災是「災/makkâ」性質，第十災是「擊打/nagaʿ」性質
- 在舊約其他章節中 nāḡap 也用於受神擊打（撒上4:8「耶和華擊打他們」）
