# 贖罪日（Yom Kippur）

## 定義

希伯來文 Yom Kippur，意為「贖罪日」或「遮蓋之日」。在利未記第16章中，神定於每年七月初十日為以色列人設立此日，大祭司亞倫需進入至聖所為聖所、會幕、壇及百姓贖罪，使以色列人潔淨並脫盡一切的罪愆。此日百姓需刻苦己心，不可做工，作為永遠的定例。
