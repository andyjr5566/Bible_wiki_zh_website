# 罪如漏症：滲透、隔離、需神醫治

## 定義

漏症的特徵是持續不斷地流出體液，且其不潔會滲透到床鋪、座椅及接觸者身上。這正預表了罪的性質：罪如同漏症，是從人敗壞的肉體生命中自然流露出來的，具有極強的傳染與滲透力，會污染個人的生活環境與人際關係。罪導致人與神隔離，無法靠自身行為止住，必須等候神的醫治，並藉著祭司的贖罪方能徹底痊癒與恢復潔淨。
