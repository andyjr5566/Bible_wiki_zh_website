# 何3：3-5

## 定義

何3:3-5與出埃及記第二十一章的關聯：何西阿書3:3-5記載何西阿贖回妻子，說「你當多日為我獨居，不可行淫」，預表以色列「多日必無君王、無祭司」，但終將「歸回，尋求耶和華他們的神」，回應出21:7-11的女僕律例。女僕律例保障女僕的贖身權利，何西阿書以贖回妻子預表以色列終將歸回神。

## 主題發展

出埃及記的女僕律例保障贖身權利，何西阿書以贖回妻子預表以色列歸回。從律法保障到先知預表，女僕的贖身權利在以色列與神的關係中得到屬靈應驗——神終將贖回祂的子民。
