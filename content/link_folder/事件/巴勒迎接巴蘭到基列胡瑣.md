# 巴勒迎接巴蘭到基列胡瑣

## 定義

巴勒聽見巴蘭來了，往摩押京城（在邊界上、亞嫩河旁）迎接他，責問為何不早來、豈不能使他得尊榮。巴蘭回應：「我已經到你這裡來了！現在我豈能擅自說什麼呢？神將什麼話傳給我，我就說什麼。」二人同行來到基列胡瑣（民22:36-39）。
