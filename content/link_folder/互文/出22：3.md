# 出22：3

## 定義

出22:3與出埃及記第二十一章的關聯：出埃及記22:3規定「若太陽已經出來，就為他有流血的罪。賊若被拿獲，總要賠還；若他一無所有，就要被賣，頂他所偷的物」，回應出21:2的奴僕制度。因偷盜無力賠償者被賣為奴以償還債務，為希伯來奴僕制度的經濟背景——貧窮與債務是賣身為奴的主要原因。

## 主題發展

出埃及記21章規定奴僕自由期限，22章補充因債務賣身為奴的背景。利未記25:39-43進一步規定因貧窮賣身的以色列人不可嚴待，新約在基督裡不分自主為奴（加3:28），為廢除奴隸制度提供神學基礎。
