# 星出於雅各（彌賽亞）

## 定義

希伯來文「כּוֹכָב מִיַּעֲקֹב」（kokhav mi-Ya'aqov），字面意為「從雅各出來的星」。見於民數記24:17，巴蘭第三次詩歌預言：「有星要出於雅各」。此預言指向彌賽亞的來臨，象徵光輝、引導與統治權。在古代近東文獻中，星常象徵君王或神性人物。新約馬太福音2:2博士們看見祂的星，應驗此預言。
