# 逾越節條例（外邦人與寄居者）

## 定義

出埃及記12:43-51 所記載逾越節參與資格條例——外邦人都不可吃這羊羔、各人用銀子買的奴僕既受了割禮就可吃、寄居的和雇工人都不可吃、一個房子裡吃且不可把肉帶出、骨頭一根不可折斷、外人寄居中間願守逾越節他所有男子務要受割禮、本地人與寄居同歸一例。KC指出割禮是參與逾越節的先決條件——救贖的恩典透過立約記號臨到；BH指出此條例排他性與包容性並存——除外邦人與未受割禮者，但歡迎受割禮的外人「像本地人一樣」。

- 本章重點：v43-51 逾越節條例補充——外邦人不可吃、奴僕受割禮可吃、寄居與雇工人不可吃、骨頭不折、一房內吃、外人受割禮可吃、本地人與寄居同歸一例

## 主題發展

- 救贖恩典與立約責任並存——割禮作參與逾越節條件
- 排他性vs包容性：外邦人排除但受割禮外人「像本地人一樣」
- 預表新約：基督救恩臨到外邦人——但必須因信稱義（加3:7-9、弗2:11-13「靠著基督的血已經得親近了」）
- 割禮預表基督的割禮（西2:11「基督的割禮」）
