# 方舟的豫表

## 定義

創世記六章方舟豫表基督——在神的審判中，惟有在方舟（基督）裡的人得保全生命。方舟的每個細節都指向基督的某方面。

### 創世記

<!-- accumulation:創世記:6:start -->
#### [[01 創世記/第6章|第6章]]
- 本章重點：神命挪亞以歌斐木造方舟，設隔間、裡外抹松香並按指定尺寸建造；來源把方舟視為在審判中提供遮蔽與拯救的預表。
- 與本章關聯：方舟把神的審判與救恩結合：凡在其中者得保存生命，來源因而將其連於基督的救贖、遮蓋與信徒在基督裡的安全。
<!-- accumulation:創世記:6:end -->

<!-- accumulation:創世記:7:start -->
#### [[01 創世記/第7章|第7章]]
- 本章重點：方舟在洪水增長、漫過高山並淹沒地上一切活物時浮起，保存挪亞一家及按類帶入的活物。
- 與本章關聯：第7章具體呈現方舟作審判中唯一避難所的功能；來源據此將進入方舟連於在基督裡蒙拯救。
<!-- accumulation:創世記:7:end -->

<!-- accumulation:創世記:8:start -->
#### [[01 創世記/第8章|第8章]]
- 本章重點：方舟經過洪水後停在亞拉臘山地，所保存的人與活物最終按神命令出來。
- 與本章關聯：第8章使方舟預表從審判中的遮蔽推進到經過審判、進入新生活的救恩圖像。

## 主題發展

### 方舟豫表基督八方面

| 項目 | 說明 | 靈意豫表 |
|---|---|---|
| 歌斐木 | 富樹脂柏木，耐水浸蝕 | 基督堅固完美的人性 |
| 分一間一間造 | 內部隔間 | 基督是各種人的安息 |
| 抹松香 | 柏油/瀝青防水 | 基督寶血的救贖功效 |
| 尺寸 | 300×50×30 肘 | 基督的完備豐滿 |
| 透光處 | 上邊高一肘 | 基督是世界的光 |
| 門 | 門開旁邊 | 基督可親可近，是通神之門路 |
| 三層 | 上、中、下 | 對基督經歷的不同程度 |
| 松香原文 | 與「贖罪」同字 | 基督的救贖 |

### 神學意義／研讀意義

- 在方舟裡＝在基督裡，得保全生命
- 方舟內外的分別：惟有進入方舟者得救——「除祂以外，別無拯救」（徒四12）
- 洪水豫表審判（太廿四39）和受浸的水（彼前三20-21）
- 方舟也豫表教會——在基督裡一同經過審判
