# 亞倫宰贖罪祭牛犢

## 定義

亞倫就近壇前，宰了為自己作贖罪祭的牛犢（利9:8）。這隻公牛犢是摩西吩咐他從牛群中取的、沒有殘疾的（利9:2）。宰殺由亞倫親手進行，是他首次以大祭司身分執行的獻祭動作，隨後由他兒子把血奉給他，進行抹血與倒血（v9）。
