# 以薩迦支派

## 定義

民數記第7章第18-23節記載以薩迦支派首領蘇押的兒子拿坦業，在獻壇禮第二天獻上供物。供物內容與各支派完全相同：銀盤、銀碗、金盂、燔祭、贖罪祭、平安祭。以薩迦作為雅各第五子利亞的兒子，其支派在曠野行軍中位於東邊猶大營旁（民數記2章）。
