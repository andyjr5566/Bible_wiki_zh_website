# 安密巴

## 定義

安密巴（En Mishpat），意為「判斷的泉」或「審判的泉」，就是[[加低斯]]（又稱加低斯巴尼亞）。位於南地，是以色列人出埃及後漂流曠野的重要駐紮地。
