# 押瑪

## 定義

押瑪（Admah），死海平原五城之一，與所多瑪、蛾摩拉、洗扁、瑣珥齊名。「押瑪」意為「紅土」或「地」。後與所多瑪、蛾摩拉同被神毀滅（申廿九23）。
