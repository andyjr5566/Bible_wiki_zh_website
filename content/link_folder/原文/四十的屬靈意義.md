# 四十的屬靈意義

## 定義

「四十」在聖經中表徵**試探、試煉和苦難**。本章洪水四十晝夜降大雨，是此原則的典型實例。

#### 第7章
- 本章重點：洪水降在地上四十晝夜，水勢不斷增長，使方舟浮起並漫過高山。
- 與本章關聯：本章使「四十」與一段完整的試煉和審判時期相連；後續經文亦常以四十標示受試驗、預備或轉折。

## 主題發展

### 聖經中「四十」的其他實例

| 實例 | 出處 |
|---|---|
| 摩西在西乃山四十晝夜 | 出廿四18；申九18 |
| 以色列曠野漂流四十年 | 民十四33-34 |
| 主耶穌曠野受試探四十天 | 太四1-2 |
| 以利亞走四十晝夜到何烈山 | 王上十九8 |

### 神學意義／研讀意義

- 洪水四十晝夜——神審判中的試煉期
- 方舟經過四十天洪水——在基督裡經歷試煉仍得保全
- 四十天後水開始漸消——試煉有盡頭，神的恩典夠用
