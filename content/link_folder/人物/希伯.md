# 希伯

## 定義

閃的後裔，亞法撒之孫，沙拉之子。原文意為「渡過」，指從世界分別出來。希伯來人的祖先，以色列人嫡系祖先。

#### 第10章
- 本章重點：閃被稱為希伯所有子孫之祖；希伯由沙拉所生，生法勒與約坍，兩支在列國表中各自展開。
- 與本章關聯：希伯把閃的家譜導向後來的希伯來人與亞伯拉罕；來源也以其名可能有「越過、彼岸」之意發展客旅與分別的靈訓。

#### 第11章
- 本章重點：第11章從閃經亞法撒、沙拉、希伯、法勒一路記到他拉與亞伯蘭；希伯在三十四歲生法勒後又活四百三十年。
- 與本章關聯：希伯位於通往亞伯蘭的應許家譜中；法勒名字與其在世時地上人口分開相連，來源並把希伯與希伯來名稱及過河客旅的靈訓聯繫。

## 主題發展

### 希伯的四項特徵

| 特徵 | 說明 |
|------|------|
| 名字 | 「希伯」意**渡過**，指從世界分別出來 |
| 源頭 | 亞法撒生**沙拉**（兵器、萌芽）→ 希伯，在生命中萌芽生長 |
| 工作 | 只從事生命的蕃衍與建造 |
| 順從 | 為兒子起名**法勒**（分散），表明全心順從神分散全地的旨意 |

### 「希伯來」的由來

- 「希伯來」意為「渡過來的人」
- 亞伯蘭是第一個被稱為「希伯來人」的（創十四13）
- 他從大河那邊渡過來，與偶像世界分別

### 與寧錄的對比

|------|------|
| 背叛者 | 渡過者（分別出來） |
| 建造帝國 | 蕃衍生命 |
| 英勇的獵戶 | 生命的傳承 |
| 目中無神 | 順從神的旨意 |
