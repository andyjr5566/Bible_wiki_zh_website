# 收在監裡等候神指示

## 定義

指褻瀆聖名者被帶到摩西面前後，眾人將他收在監裡（bammišmār，בַּמִּשְׁמָר），要得耶和華所指示的話（利24:12）。這顯示以色列法律制度中，對於未曾有過先例的重案（特別是涉及神名聖潔），必須等候神直接啟示判斷，而非人自行斷定。這也是摩西多次遇難案求問耶和華的模式（參民9:6-8, 15:32-36, 27:1-5）。

- 本章重點：記載褻瀆聖名者被收在監裡，等候耶和華指示如何處置。
- 與本章關聯：觸發神頒布褻瀆聖名條例的關鍵程序步驟，體現神權政治下司法程序的神聖性。

## 主題發展

此事件確立「案例法」頒布模式：具體案件 → 求問神 → 神頒布判例 → 成為永遠定例。新約教會雖無物理監獄，但對教會紀律案件（太18:15-17, 林前5）同樣需謹慎禱告、尋求聖靈引導、按神話語處理，不可輕率定罪或寬恕。
