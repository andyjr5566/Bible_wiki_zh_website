# 假合一

## 定義

人類以自己的意志和計畫達成的合一，而非在神的旨意中的合一。創十一1-9的巴別塔是假合一的典型例子。

### 創世記

#### 第11章
- 本章重點：巴別眾人有共同語言與集體能力，卻以人的商議、材料和榮耀建造自己的名，拒絕分散全地。
- 與本章關聯：《創世記第十一章註解》以此區分人工維繫的假合一與聖靈所成就的真合一；問題不是合一自身，而是共同目標與神旨意相抗。
<!-- accumulation:創世記:11:end -->

## 主題發展

真正的合一是在基督裏、藉著聖靈達成的（弗四3-6），不是人的努力或妥協的產物。

### 巴別塔的假合一特徵

| 特徵 | 說明 | 經文 |
|------|------|------|
| 人的商量 | 以人的計畫取代神的旨意 | 3節「來吧」 |
| 人的取代 | 以人的工作取代神的工作 | 3節磚代石、漆代灰泥 |
| 人的榮耀 | 尋求人的名而非神的名 | 4節「為自己傳揚名」 |
| 人的抗拒 | 拒絕神的命令（遍滿全地） | 4節「免得我們分散」 |

### 真合一 vs 假合一

| 假合一（巴別） | 真合一（教會） |
|----------------|----------------|
| 人的「來吧」（4節） | 神的「來吧」（7節） |
| 磚（人造物） | 活石（神造物，彼前二5） |
| 漆（天然美德） | 灰泥（聖靈變化的品格） |
| 為自己傳揚名 | 為神的名作見證 |
| 拒絕分散 | 順從神遍滿全地 |
| 結果：混亂和分散 | 結果：在基督裏合而為一 |
