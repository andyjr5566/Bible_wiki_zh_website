# 禧年土地贖回計算法（按距禧年年數定價）

## 定義

以色列人買賣土地的價格以禧年為標準設定。買賣時要按距離下次禧年的年數計算，因為到了禧年所有土地都要回到原主人手中。年歲若多，要照數加添價值；年歲若少，要照數減去價值，因為他照收成的數目賣給人。若要贖回，就要算出賣地的年數，把餘剩年數的價值還那買主。
