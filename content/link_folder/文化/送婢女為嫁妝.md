# 送婢女為嫁妝

## 定義

創世記第二十九章記載拉班將悉帕給利亞、將辟拉給拉結作使女。來源把這放在古代近東婚嫁背景中，指出父親把婢女給女兒作陪嫁或私用家僕，是當時常見作法。
