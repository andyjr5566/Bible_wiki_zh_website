# 以惡報惡的罪

## 定義
指因受人傷害而以同等或更惡劣手段回擊，而非交託神伸冤（羅12:19）。聖經教導：「不可以惡報惡」（羅12:17；帖前5:15；彼前3:9），主耶穌教導「愛你們的仇敵」（太5:44）。以惡報惡屬肉體情慾（加5:19-21），阻擋神公義彰顯。

  - 擄掠一切財物、婦女、孩子（27-29節）

## 主題發展
- 以暴制暴導致罪惡升級：強暴→詭計→屠城→擄掠→全家受威脅
- 「人的怒氣不成就神的義」（雅1:20）
- 神伸冤時間表非人可掌控（羅12:19；申32:35）
