# 雅各全家下埃及

## 類型

事件

## 觸發來源

## 目前資料

### 定義

指雅各（以色列）在得知約瑟尚存、在埃及掌權後，舉全家七十人遷居埃及歌珊地的歷史事件。這是亞伯拉罕之約「寄居四百年」預言（創 15:13-14）的開始應驗，也是以色列民族從家族變為民族的關鍵轉折點。

### 經文記載

| 經文 | 重點 |
|------|------|
| **創 45:16-28** | 法老邀請雅各全家來埃及，雅各心靈甦醒：「我兒約瑟還活著，我要去見他。」 |
| **創 46:1-30** | **雅各在別是巴獻祭 → 神異象安慰 → 七十人名單 → 雅各見約瑟** |
| **創 46:31-47:12** | 約瑟安排家族見法老，獲賜歌珊地，雅各為法老祝福 |
| **創 47:27-31** | 以色列人在埃及生養眾多，雅各囑咐約瑟埋葬迦南 |
| **出 1:1-7** | 「雅各的兒子各人帶著家眷來到埃及……以色列人生養眾多，極其強盛」 |
| **徒 7:14-15** | 司提反見證：「約瑟打發人請父親雅各和全族七十五人下來」 |

### 關鍵細節

### 1. 神在別是巴的應許（創 46:2-4）
> 「雅各啊，雅各啊……我是神，你父親的神。你下埃及去，不要害怕，**我必使你在那裡成為大族**。我必與你同下埃及去，也必定帶你上來。」

- **四重應許**：(1) 成為大族 (2) 神同在 (3) 必帶上來 (4) 約瑟給你送終

### 2. 七十人名單（創 46:8-27）
| 來源 | 人數 | 組成 |
|------|------|------|
| **創 46:27** | **70 人** | 雅各 1 + 兒子 12 + 孫子/曾孫 53 + 儿媳 2（猶大妻、西緬妻）+ 約瑟夫婦 2 = 70 |
| **徒 7:14** | **75 人** | 七十士譯本（LXX）計算：將約瑟的孫子（瑪拿西、以法蓮的子孫）計入 |

> **註**：70 象徵「萬國」（創 10 列國 70 個），以色列作祭司國度代表萬國。

### 3. 歌珊地（創 47:1-6、11）
- **位置**：尼羅河三角洲東部，沃土宜牧。
- **目的**：(1) 埃及人厭惡牧羊人（創 46:34），隔離保護以色列純正信仰 (2) 靠近約瑟掌權中心 (3) 為日後出埃及預備出發點。

### 4. 雅各為法老祝福（創 47:7-10）
> 雅各進去給法老祝福……「我平生的年日是一百三十歲，年日又少又苦。」

- **屬靈權柄超越政治權柄**：年老的牧羊人為統治強國的法老祝福（來 7:7「無可爭辯，小的被大的給祝福」）。

### 神學意義

### 1. 約的應驗進程
```
創 12:1-3  亞伯拉罕之約（應許地、後裔、福源）
    ↓
創 15:13-16  預言：寄居四百年、受苦、後出來、大富
    ↓
創 46:1-4  雅各下埃及 —— **四百年寄居正式開始**
    ↓
出 1:7-14  以色列人在埃及生養眾多、被苦待
    ↓
出 12-14  出埃及、紅海得救
```

### 2. 「死而復活」預表
- **雅各以為約瑟死** → 發現活著（創 45:26-28）
- **雅各全家「死」（離開應許地）** → 在埃及「復活」（成為大族）
- **預表基督**：死裡復活、成為多子長兄（羅 8:29；林前 15:20-23）。

### 3. 神主權在「惡」中成就善
- **兄弟嫉妒賣約瑟**（創 37）→ **約瑟在埃及備糧救全家**（創 45:5-8；50:20）
- **創 50:20**：「你們本意是害我，但神意圖是好的，要保全許多人的性命。」

### 

### 4. 族長時代結束、民族時代開始
- **族長時代**：亞伯拉罕、以撒、雅各——個人、家庭層面
- **民族時代**：出埃及記起——民族、律法、國家層面
- **過渡人物**：約瑟（族長式信心 + 行政智慧）與摩西（先知、立法者、中保）
