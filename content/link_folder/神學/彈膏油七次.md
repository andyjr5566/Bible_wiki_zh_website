# 彈膏油七次

## 定義

摩西用膏油在祭壇上彈了七次，並抹了壇和壇的一切器皿，使其成聖。在聖經中「七」代表完全，彈膏油七次象徵著完全的潔淨與成聖，表明祭壇及其器皿被徹底分別為聖供神使用。
