# 創一章與創八章的對照

## 定義

創八1-5與創一1-2形成創造與拯救的平行結構，表明洪水後的拯救過程在神學上與創造過程對應。

#### [[01 創世記/第8章|第8章]]
- 本章重點：神使風吹水退、旱地再現、活物出來繁衍，與創世記1章水中顯出旱地及生物繁殖相呼應。
- 與本章關聯：平行結構把洪水後世界描繪為新創造，同時保留人心仍然懷惡的張力。

## 主題發展

洪水是「去創造」，方舟出來是「重新創造」。審判之後是新的開始。

### 創造與拯救的對照

| 創一章 | 創八章 |
|--------|--------|
| 神的靈運行在水面上（一2） | 神叫風吹地（八1） |
| 神說「天下的水要聚在一處」（一9） | 山頂現出來了（八5） |
| 神看著是好的（一10） | 挪亞看見地乾了（八13） |
| 神賜福「滋生眾多」（一22） | 神說「多多滋生，大大興旺」（八17） |

### 「風＝靈」的關鍵連結

רוּחַ 在創一2譯為「靈」，創八1譯為「風」，是同一個希伯來字。這表明8-9章是洪水後人類新生活開始的次序，與天地創造次序平行。
