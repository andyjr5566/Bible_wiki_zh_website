# 萬國同源

## 定義

創十記載全人類都從挪亞三個兒子的後裔繁衍出來，萬國的人都出於一本（徒十七26）。

#### 第10章
- 本章重點：挪亞三子洪水後生出各宗族，按家族、方言、地土與邦國分布；全地的邦國都由這些支系分出。
- 與本章關聯：第10章的列國表同時陳述民族與語言的多樣分布，以及他們同出挪亞一家的共同來源；這也為第12章地上萬族因亞伯蘭得福預備背景。

## 主題發展

### 創十的族譜結構

- 雅弗後裔（2-5節）→ 北歐、黑海、地中海沿岸各民族
- 含的後裔（6-20節）→ 非洲北部、巴勒斯坦及阿拉伯曠野
- 閃的後裔（21-31節）→ 西亞，處於雅弗和含中間

記錄方式：範圍逐漸縮小，一直縮到以色列民先祖亞伯蘭（11章）的譜系上。

### 屬靈意義

- 全人類同出一源，不應有「非我族類」的敵視態度
- 在基督裏不再有種族、文化、語言的分別（西三11；林前十二13；加三28）
- 萬國的事都由主早定（申三十二8）
