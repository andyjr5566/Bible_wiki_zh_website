# 巴蘭見神喜悅賜福不求法術

## 定義

巴蘭見耶和華喜歡賜福與以色列，不再像前兩次去求法術，卻面向曠野，表明他不再試圖用占卜手段操控神意，而是順服神的旨意發預言（民24:1）。

## 主題發展

此事件標誌巴蘭從「求法術」的占卜者轉為「神的靈臨到」的先知口舌，顯示神主權掌控預言內容。
