# 法老心剛硬三字

## 定義

出埃及記中描繪法老心剛硬的三個希伯來文字：חָזַק (hazaq, 加強、堅定)、כָּבַד (kabed, 沉重)、קָשָׁה (qashah, 剛硬)——描繪法老由固執到完全剛硬的漸進過程。

## 主題發展

GT中文聖經註釋指出三字的漸進意義：
1. *kabed*（沉重）——六14「法老心裡固執」，原文是沉重的，不易被搬動和改變
2. *hazaq*（加強、堅定）——七13,22「法老心裡剛硬」，堅持己意不願改變
3. *qashah*（剛硬）——完全剛硬、頑固不化

CT指出「固執」原文意指「沉重」——凡存心和觀念太過沉重的人，不容易被別人搬動和改變。

這三字也對應三個主體：法老自己剛硬他的心（八15,32, 九34）、法老心自己剛硬（七13,22, 八19）、神使法老心剛硬（九12, 十1,20, 十一10, 十四4,8）——顯示剛硬過程中神主權與人責任的交織。頭幾個災前是法老自己剛硬（八15,32），後幾個災是神使他剛硬——任憑他堅持自己選擇的方向。
