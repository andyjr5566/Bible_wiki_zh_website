# 祭司的女兒行淫用火焚燒

## 定義

利未記第21章9節規定，祭司的女兒若行淫辱沒自己，就辱沒了父親，必用火將她焚燒。此條例表明祭司家族成員的聖潔直接影響祭司本人的事奉資格，對領袖家庭有更高的道德要求。
