# 亞設（有福）

## 定義
利亞使女悉帕所生的次子（雅各第八子），名字意為「有福/歡樂」。利亞說「我有福啊，眾女子都要稱我是有福的」，故以此命名。

## 按書卷累積

### 創世記
<!-- accumulation:創世記:30:start -->
#### [[01 創世記/第30章|第30章]]
- 本章重點：悉帕又給雅各生了第二個兒子，利亞說：「我有福啊，眾女子都要稱我是有福的。」於是給他起名叫亞設（就是有福的意思）。
- 與本章關聯：
  - 利亞在生子競爭中感到被肯定
  - 利亞認為眾女子都要承認她的福分
  - 名字反映利亞的喜樂與滿足
