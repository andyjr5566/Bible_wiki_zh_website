# 抹血在右耳垂、右手大拇指、右腳大拇指

## 定義

在承接聖職之禮中，摩西將第二隻公綿羊的血抹在亞倫和他兒子的右耳垂上、右手的大拇指上並右腳的大拇指上。此動作象徵祭司全人被潔淨並全然獻身事奉神：右耳垂代表聽從神的命令，右手大拇指代表行事為人按聖潔履行祭司職責，右腳大拇指代表行走聖潔的道路。這血同時也灑在壇的周圍，表明祭司與祭壇聯合，其事奉建立在贖罪之血的基礎上。
