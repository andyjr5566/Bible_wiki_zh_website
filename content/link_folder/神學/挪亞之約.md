# 挪亞之約

## 定義

神與挪亞及其後裔並一切活物所立的**無條件之約**，應許不再用洪水毀滅全地，以虹為記號。又稱「彩虹之約」。

## 主題發展

「約」是雙方互相同意締結、有約束力的關係；但在神與人的關係中，神是創造者、人是受造物，因此神人之約是**神對人的賞賜**，非對等條件交換。

### 約的五項內容

1. **尊重生命、保障生命** — 毀人生命者須以生命償還（九5-6）
2. **不再用全球性洪水刑罰人，不加重對地的咒詛**（八21；九9-16）
3. **自然界四季秩序常保**（八22）
4. **人可以吃肉**（九3-4），但不可吃帶血的肉
5. **預言經由閃的後代將有特別預備**（九26-27）

### 約的特性

1. **無所不包**：包括「一切活物」
2. **永遠的**：「永約」
3. **單方面施惠**：沒有要求人任何條件
4. **有記號**：「虹」
5. **給神看見就記念**：神降卑自己情願受約的拘束

### 立約的根據

- 不是根據人的情況（人從小時心裏懷著惡念，八21）
- 乃是根據神的祝福（神賜福給挪亞）
- 又是根據神的旨意（人照神形像造的）
