# 創世記 第4章 link candidates

## 人物

- 亞當 → 人物 — 經文25節；本章記亞當與妻子再生塞特，延續被殺的亞伯之後的後裔盼望。
- 夏娃 → 人物 — 經文1、25節；來源討論她為該隱、亞伯、塞特命名時呈現的盼望與理解。
- 該隱 → 人物 — 經文1-16節；CT/KC/BH均討論其名字、獻祭、殺亞伯、受咒詛與離開神面。
- 亞伯 → 人物 — 經文2-10節；來源討論其名字、牧羊、因信獻祭、被殺與血的見證。
- 塞特 → 人物 — 經文25節；來源指出其名有設立、代替之意，承接亞伯的位置。
- 以挪士 → 人物 — 經文26節；來源指出其名有脆弱必死之人，並與求告耶和華名相連。
- 以諾（該隱之子） → 人物 — 經文17節；來源討論該隱之子以諾與建城命名。
- 拉麥 → 人物 — 經文18-24節；來源討論其多妻、暴力自誇與該隱支線敗壞。
- 以拿 → 人物 — 經文18節；CT/BH列於該隱支線家譜並解釋名字。
- 米戶雅利 → 人物 — 經文18節；CT/BH列於該隱支線家譜並解釋名字。
- 瑪土撒利 → 人物 — 經文18節；CT/BH列於該隱支線家譜並解釋名字。
- 亞大（拉麥之妻） → 人物 — 經文19-20、23節；來源討論她為拉麥妻子、雅八與猶八之母，需與以掃妻子亞大區分。
- 洗拉 → 人物 — 經文19、22-23節；來源討論她為拉麥妻子、土八該隱與拿瑪之母。
- 雅八 → 人物 — 經文20節；來源指出他是住帳棚、牧養牲畜之人的祖師。
- 猶八 → 人物 — 經文21節；來源指出他是一切彈琴吹簫之人的祖師。
- 土八該隱 → 人物 — 經文22節；來源指出他打造銅鐵利器，代表冶金與利器工藝。
- 拿瑪 → 人物 — 經文22節；來源記她為土八該隱的妹子，名字意為悅心或美麗。

## 地點

- 伊甸 → 地點 — 經文16節；該隱住在伊甸東邊挪得之地，來源指出向東離開神同在的方向意義。
- 挪得 → 地點 — 經文16節；來源指出其名意為流浪、飄蕩或放逐。

## 神學與主題

- 血祭與素祭 → 主題 — 經文3-5節；來源討論該隱與亞伯供物差異、信心、啟示與獻祭者本人。
- 罪伏在門前 → 神學 — 經文7節；CT/KC/BH均討論罪被人格化為蹲伏門前的危險。
- 亞伯的血哀告 → 神學 — 經文10節；來源討論無辜之血向神呼求，並與來12:24互文。
- 該隱的記號 → 神學 — 經文15節；來源指出記號兼具審判中的保護與憐憫。
- 該隱的後代與無神文化 → 主題 — 經文17-24節；來源討論建城、畜牧、音樂、冶金與暴力文化。
- 我豈是看守我兄弟的麼 → 主題 — 經文9節；來源討論該隱說謊、頂撞神與弟兄責任。
- 求告耶和華的名 → 主題 — 經文26節；來源討論塞特支線在軟弱中轉向呼求神。
- 該隱的道路 → 神學 — CT/KC引用猶11，指人憑己意親近神、拒絕恩典而走向離開神。
- 拉麥之歌 → 主題 — 經文23-24節；來源指出這是古老詩歌，呈現暴力自誇與復仇升級。
- 拉麥的一夫多妻 → 神學 — 經文19節；來源指出拉麥首次破壞創2:24的一夫一妻婚姻模式。
- 該隱妻子的來源 → 解經爭議 — 經文17節；BH/GT討論該隱妻子可能來自亞當夏娃其他兒女及早期近親婚姻問題。

## 互文

- 該隱殺亞伯 → 互文 — 經文8節；來源多次連到約壹3:12、加4:29與義人受逼迫主題。
- 希伯來書11章亞伯因信獻祭 → 互文 — 來源以來11:4解釋亞伯獻祭蒙悅納的關鍵。
- 希伯來書12章亞伯的血與基督的血 → 互文 — 來源以來12:24比較亞伯血與基督血。
- 猶大書該隱的道路 → 互文 — 來源以猶11說明該隱道路代表自意宗教與背離神。
