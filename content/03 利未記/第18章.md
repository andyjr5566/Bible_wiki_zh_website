# 利未記 第18章

1. 耶和華對[[摩西]]說：
2. 你曉諭以色列人說：我是耶和華─你們的神。
3. 你們從前住的埃及地，那裡人的行為，你們[[不可效法外邦惡俗|不可效法]]，我要領你們到的迦南地，那裡人的行為也不可效法，也不可照他們的惡俗行。
4. 你們要遵我的典章，守我的律例，按此而行。我是耶和華─你們的神。
5. 所以，你們要守我的律例典章；人若遵行，[[遵行律例就必活著|就必因此活著]]。我是耶和華。
6. 你們都不可[[亂倫禁令（gillui ervah）|露骨肉之親的下體]]，親近他們。我是耶和華。
7. 不可露你母親的下體，羞辱了你父親。他是你的母親，不可露他的下體。
8. 不可露你繼母的下體；這本是你父親的下體。
9. 你的姊妹，不拘是異母同父的，是異父同母的，無論是生在家生在外的，都不可露他們的下體。
10. 不可露你孫女或是外孫女的下體，露了他們的下體就是露了自己的下體。
11. 你繼母從你父親生的女兒本是你的妹妹，不可露他的下體。
12. 不可露你姑母的下體；他是你父親的[[骨肉之親（she'er besaro）|骨肉之親]]。
13. 不可露你姨母的下體；他是你母親的[[骨肉之親（she'er besaro）|骨肉之親]]。
14. 不可親近你伯叔之妻，羞辱了你伯叔；他是你的伯叔母。
15. 不可露你兒婦的下體；他是你兒子的妻，不可露他的下體。
16. 不可露你弟兄妻子的下體；這本是你弟兄的下體。
17. 不可露了婦人的下體，又露他女兒的下體，也不可娶他孫女或是外孫女，露他們的下體；他們是[[骨肉之親（she'er besaro）|骨肉之親]]，這本是大惡。
18. 你妻還在的時候，不可另娶他的姊妹作對頭，露他的下體。
19. 女人[[行經不潔淨時的禁令|行經不潔淨]]的時候，不可露他的下體，與他親近。
20. 不可與[[鄰舍妻行淫|鄰舍的妻]]行淫，[[玷污自己與玷污地|玷污自己]]。
21. 不可使你的兒女經火歸與[[摩洛（Molech）|摩洛]]，也不可褻瀆你神的名。我是耶和華。
22. 不可[[同性戀禁令（mishkav zakhar）|與男人苟合]]，像與女人一樣；這本是[[可憎（sheqets）|可憎惡的]]。
23. 不可[[獸交禁令|與獸淫合]]，[[玷污自己與玷污地|玷污自己]]。女人也不可站在獸前，與他淫合；這本是[[獸交禁令|逆性的事]]。
24. 在這一切的事上，你們都不可[[玷污自己與玷污地|玷污自己]]；因為我在你們面前所逐出的列邦，在這一切的事上玷污了自己；
25. 連地也玷污了，所以我追討那地的罪孽，那地也吐出他的居民。
26. 故此，你們要守我的律例典章。這一切[[可憎（sheqets）|可憎惡的]]事，無論是本地人，是寄居在你們中間的外人，都不可行，
27. 在你們以先居住那地的人行了這一切[[可憎（sheqets）|可憎惡的]]事，地就玷污了，
28. 免得你們玷污那地的時候，地就把你們吐出，像吐出在你們以先的國民一樣。
29. 無論什麼人，行了其中[[可憎（sheqets）|可憎的]]一件事，必從民中[[剪除（kareth）|剪除]]。
30. 所以，你們要守我所吩咐的，免得你們隨從那些[[可憎（sheqets）|可憎的]]惡俗，就是在你們以先的人所常行的，以致玷污了自己。我是耶和華─你們的神。

## 本章整理

### 引言與總綱（v1-5）
耶和華吩咐摩西曉諭以色列人，開篇便宣告 ==“我是耶和華－你們的神”==，將一切誡命建立在神與子民的聖約關係之上。經文隨即劃出兩道清晰的邊界：以色列人既不可效法他們從前居住的 [[埃及與迦南惡俗|埃及地之人的行為]]，也不可效法即將進入的 [[不可效法外邦惡俗|迦南地之人的惡俗]]。這表明聖潔生活的標準並非地理或文化相對主義，而是唯獨來自神的 [[遵行律例就必活著|律例典章]]。第5節提出一個生命性的應許：==“人若遵行，就必因此活著”==——這裡的“活著”不僅指存活，更指向在約中與神相交的豐盛生命。這五節經文構成了全章的樞紐，將隨後詳列的性倫理禁令置於「分別為聖」的宏大敘事框架中。《串珠》數過本章的兩個副歌：**七次**「不可效法／不可行」迦南惡俗（v3×2、24、26、27、29、30），**六次**「我是耶和華（你們的神）」（v2、4、5、6、21、30）。

KC 把 v3 的兩個地名讀成兩種危險：「**埃及代表沒有神的世界**。埃及有很多人的智慧（徒7:22）……歌羅西書警告那擄掠人的智慧（西2:8）。」而「**迦南是神賜給祂百姓的福地。那裡最大的危險不是智慧、文化與科學，而是在那裡橫行的大敗壞**——正是在描述基督徒福分的以弗所書裡，對各樣的污穢有最強烈的警告（弗4:17-19）」。

v5「人若遵行，就必因此活著」，KC 統計它被舊約引了四次（結20:11、13、21；尼9:29）、新約三次（路10:25-28；羅10:5；加3:12），而羅馬書與加拉太書的引用顯明「**沒有一個人領受了那應許的生命，因為沒有人完全遵行**」。他收在一句對子上：==「**律法說：行這個，你就必活。福音說：活著，你就必行。**」==丁良才同讀：「律法上的應許是本乎行為，福音上的應許是本乎信心（羅10:4-5，加3:9、11-12）。」

### 亂倫禁令的範圍（v6-18）
從第6節開始，經文以“露下體”（[[亂倫禁令（gillui ervah）|gillui ervah]]）這一具體用語，系統性地列舉了禁止的性關係對象。這些禁令的核心原則是保護 [[骨肉之親（she'er besaro）|骨肉之親]] 的界限，禁止在近親圈內發生性行為。清單涵蓋了直系與旁系親屬，包括母親、繼母、姊妹（無論是同父異母或同母異父）、孫女、外孫女、姑母、姨母、伯叔母、兒婦、弟兄妻子等。值得注意的是，第17-18節將禁令擴展至多重關係的疊加：不可同時娶母女或祖孫女，也不可在妻子尚在時另娶其姊妹作對頭。這些規定不僅是性禁忌，更是對家庭結構與人倫秩序的深層保護，防止權力、慾望與嫉妒撕裂神所設立的親屬網絡。逐條整理（刑罰出自利20，實例出自丁良才）：

| 節 | 禁止的對象 | 刑罰（利20）／實例 |
| --- | --- | --- |
| v7 | 母親 | 「羞辱了你父親」——夫婦一體（創2:24） |
| v8 | 繼母（父親的女人） | 治死（利20:11）；流便（創35:22）、押沙龍（撒下16:22）、林前5:1 |
| v9、11 | 姐妹（同父異母／同母異父／繼母所生） | 利20:17；暗嫩與他瑪（撒下13）；亞伯拉罕娶撒拉在律法之前（創20:12） |
| v10 | 孫女、外孫女 | 「露了她們的下體就是露了自己的下體」 |
| v12-13 | 姑母、姨母 | 暗蘭娶約基別在律法之前（出6:20） |
| v14 | 伯叔母 | 「必無子女而死」（利20:20） |
| v15 | 兒婦 | 二人治死（利20:12） |
| v16 | 弟兄之妻 | 利20:21「必無子女」；例外見下欄 |
| v17 | 婦人與其女兒／孫女 | 「這本是大惡」；三人用火焚燒（利20:14） |
| v18 | 妻子的姐妹（妻還在世時） | 雅各娶利亞與拉結（創29-30） |

> [!note] v16 有一個律法自己開的例外：繼代婚姻
> 「不可露你弟兄妻子的下體」有一個各家一致指出的例外——申25:5-10 的兄終弟及：弟兄同居若死了一個、沒有兒子，「死人的妻不可出嫁外人，他丈夫的兄弟當娶他為妻」，所生長子歸死者名下。丁良才指出「當雅各的時候就已經有這風俗（創38:8）」；《舊約背景註釋》也把它列為亂倫禁令的「惟一例外」。《啟導本》補上新約的迴響：施洗約翰責備希律安提帕，正因他在**兄弟還活著時**娶了其妻希羅底（可6:18）——禁令管活人，例外只為死人留後。
>
> v18「不可另娶她的姐妹作對頭」也有實例可指：丁良才與《啟導本》都舉**雅各娶利亞與拉結**二姐妹為殷鑒（創29-30）——「作對頭」就是「使本來相愛的轉為相恨的」。而丁良才引猶太注釋家：**妻子去世後續娶其妹則不在此禁**——禁的是「你妻還在的時候」。

### 其他性倫理與宗教禁令（v19-23）
此段落將禁令從家庭內部延伸至更廣泛的社會與宗教領域。第19節禁止在妻子 [[月經（niddah）|行經不潔淨]] 時與她親近，將生理週期納入聖潔規範。第20節禁止與 [[鄰舍妻行淫|鄰舍的妻行淫]]，守護婚姻盟約的排他性。第21節轉向宗教領域，嚴禁使兒女「經火」歸與[[摩洛（Molech）|摩洛]]。

> [!question] 「經火歸與摩洛」是什麼——火祭說與廟妓說
> 多數來源讀作**焚燒孩童的火祭**：李道生描述那尊神像「有把炭火燃燒放在它的面前……獻祭時置小孩於其手內，使之戰兢落於火中」；《啟導本》記兩種說法（拋入烈焰，或置於燒紅的金屬臂上灼死）；《舊約背景註釋》給出考古佐證：「北非洲之迦太基，以及薩丁尼亞的腓尼基遺址，**都挖掘到以孩童為祭的考古證據**」，並記主前八世紀腓尼基碑文載基利家人臨戰向摩洛獻祭。
> **KC 卻不這樣讀**：「在本段的脈絡裡，這**看來主要不是火祭，而是把孩童奉獻去作廟妓**。那是肉身淫亂底下的屬靈淫亂。」——這個分歧值得保留：火祭說有考古根據，KC 的讀法則扣緊本章通篇的性倫理脈絡。
> 艾基斯另記學界對 molek 一字本身的爭論：艾斯費爾德據迦太基碑文主張它不是神名而是「獻祭」的術語，但馬里文獻的 Muluk 神與亞述的亞得米勒（王下17:31）支持傳統讀法——「摩洛似乎特指一個異教的神，需要把嬰兒作為燔祭獻上」。
>
> 丁良才補「褻瀆你神的名」的所以然：以色列人「若將牛羊獻給神，卻把自己的兒女獻給摩洛，就不免令外邦人說，以色列人以為**耶和華不如摩洛尊貴**」。第22節明確禁止[[同性戀禁令（mishkav zakhar）|男人與男人苟合]]，稱之為==「可憎惡的」==（[[可憎惡的（toevah）|toevah]]——注意這與利11 水族的「可憎」sheqets 是不同的字；《舊約聖經背景註釋》：「第22-29節中『可憎惡的』一語，指出這種行為是**違反神本性的**」）。第23節則禁止人與獸淫合，定性為 ==“逆性的事”==。這一系列禁令勾勒出一個全方位的聖潔框架：性關係被嚴格限定在合法婚姻之內，且不得沾染異教獻祭的惡俗。（《啟導本》提醒：創2:24 的完美婚姻制度「**未在希伯來人社會中完全實行**」——v8《精讀本》指出當時是一夫多妻制、v18 明文規範多妻的情形，故本章的框架是「婚姻的聖潔」而非已然的一夫一妻制。）

### 玷污的後果與聖約責任（v24-30）
經文最後一部分將視野從個體行為拉升至整個土地與民族的命運。神指出，即將被驅逐的迦南列邦正是因為行了上述一切事而 [[玷污自己與玷污地|玷污了自己，連地也玷污了]]。土地被擬人化為有道德反應的主體：==「那地也吐出他的居民」==。《舊約背景註釋》解釋迦南這些行徑的宗教邏輯：「這些性變態行為，不可以單純視為人類本性墮落的結果。**崇拜儀式中進行性交的用意，是促進土地豐收，牲畜和人口興旺**……廟宇人員肯定包括男女娼妓。」——以性求豐收的宗教，最終使地本身不潔。丁良才由此整理出人與地的五重關係：人違主命，地受咒詛（創3:17）；人行惡事，地被玷污；人犯律法，地就衰殘（賽24章）；人蒙恩典，地得福祉；人受報應，地便歡樂（詩96:11-13）——「**世界之景況的好歹是在乎人道德的盛衰**」。這是一個強烈的神學意象——聖潔的土地無法容忍持續行惡的居民。因此，以色列人無論是本地人還是 [[寄居的（ger）|寄居的外人]]，都必須遵守這些律例，否則將面臨同樣的後果：被地“吐出”，並從民中 [[剪除（kareth）|剪除]]。結尾再次回到 ==“我是耶和華－你們的神”==，形成首尾呼應，將一切倫理要求錨定在神的自我啟示與聖約主權之中。

> [!important] 本章樞紐
> 利未記18章是“聖潔法典”（Holiness Code, 利17-26）的核心篇章之一。它將性倫理從社會習俗提升為聖約忠誠的標誌。以色列人之所以要遵守這些律例，不是因為它們“合乎自然”或“普遍理性”，而是因為耶和華是他們的神，且祂已將他們從埃及分別出來。本章的禁令清單與創世記中迦南地的性混亂敘事（如所多瑪、羅得與女兒）形成隱含的敘事呼應，表明神對其子民的倫理要求與對迦南人的審判標準是一致的。
