# 安息日

## 定義

神在第七日賜福、把這日「定為聖日」（*qadesh*，קָדַשׁ），因為神在這日歇了創造之工安息了。原文 *shabbat*（שָׁבַת）是「安息日」的字根。

## 主題發展

### 神學意義／研讀意義

- 安息日的設立是在創造完成時，先於摩西律法
- 神「賜福」＋「聖化」＝分別出來、歸於神
- 安息日豫表基督的福音——惟有主耶穌十字架的工作，給人帶來真正的安息
- 人接受主耶穌救恩 = 滿意神的工作，安息在神的安息裡
- 千年國度時，安息日將恢復為全受造物的實際（來四9）
