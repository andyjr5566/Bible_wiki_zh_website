# 牲畜許願不可更換

## 定義

利未記第27章9-10節規定，人將潔淨牲畜許願獻給耶和華為供物，即成為聖，不可改換、更換（好換壞、壞換好）；若強行以牲畜更換牲畜，原許願牲畜與所換牲畜雙方都成為聖，不可贖回。

- 與本章關聯：確立供物聖潔不可轉移原則，與第33節什一牲畜不可更換呼應。

## 主題發展

此條款防止許願人事後反悔以次充好或私自調換，強調獻給神之物的聖潔不可侵犯；「更換牲畜雙方成聖」的懲罰性規定，確保供物純全。
