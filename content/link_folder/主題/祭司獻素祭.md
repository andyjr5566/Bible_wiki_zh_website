# 祭司獻素祭

## 定義

祭司從婦人手中取嫉妒素祭，在耶和華面前搖一搖，拿到壇前，從中取一把作為紀念燒在壇上，然後叫婦人喝苦水（民5:25-26）。祭司作中保執行儀式，將素祭呈獻神前，啟動審判程序。
