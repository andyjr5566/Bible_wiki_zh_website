# 頒布獻祭與禁血條例

## 定義

耶和華在西奈曠野透過摩西曉諭亞倫、祭司與全體以色列人，頒布宰殺牲畜必須帶到會幕門口獻祭，以及嚴禁食用任何活物之血的律例，違者必被剪除。
