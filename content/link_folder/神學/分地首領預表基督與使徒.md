# 分地首領預表基督與使徒

## 定義

分地首領以利亞撒（大祭司）與約書亞（領袖）及各支派首領（民34:16-29），被視為預表基督（大祭司、元帥）與使徒（教會柱石）分賜屬靈產業。以利亞撒代表基督祭司職分按神公義分賜；約書亞代表基督君王職分帶領得地；支派首領預表使徒根基上建立的教會領袖分派恩賜產業（弗4:11-13）。

- 本章重點：經文第16-29節列出以利亞撒、約書亞及十支派首領名單，構成分地領導層級。
- 與本章關聯：「分地首領名單」人物群的神學預表意義，指向新約基督與使徒職分的產業分派權柄。

## 主題發展

此預表將歷史人物名單（v.16-29）與新約救恩經濟學連結，研經資料常引用此章說明屬靈權柄與產業分派的神聖秩序。
