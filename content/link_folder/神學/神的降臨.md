# 神的降臨

## 定義

神的降臨（the LORD's descent / theophany）是神在西乃山以可見、可聽、可觸的方式向以色列人顯現的事件。出19:9-20記載神在密雲中降臨，伴隨雷轟、閃電、角聲、全山冒煙、遍山大震動。v9神說「我要在密雲中臨到你那裡，叫百姓在我與你說話的時候可以聽見，也可以永遠信你了」。v11「第三天耶和華要在眾百姓眼前降臨在西乃山上」。v16-18記載降臨的景象——雷轟、閃電、密雲、角聲甚大、全山冒煙如燒窯、遍山大震動。v20「耶和華降臨在西乃山頂上」。

CT指出神降臨的目的是叫百姓可以聽見神與摩西說話，永遠信摩西是神的代言人。KC指出神降臨伴隨令人恐懼戰兢的現象——來12:18-21描述西乃山的景象為「火焰、密雲、黑暗、暴風、角聲和說話的聲音」。BH指出神的降臨（theophany）是舊約中神直接向人顯現的重要事件——西乃山的降臨是其中最壯觀的一次，彰顯神的聖潔、能力和威嚴。

### 出埃及記

#### [[02 出埃及記/第19章|第19章]]
- 本章重點：v9-20記載神降臨西乃山的完整過程——神宣告要在密雲中降臨（v9），吩咐百姓自潔三天（v10-15），第三天降臨伴隨雷轟閃電密雲角聲（v16），全山冒煙震動（v18），角聲高而又高（v19），神降臨山頂召摩西上去（v20）。CT指出降臨目的是叫百姓永遠信摩西。KC指出降臨伴隨令人恐懼的現象。BH指出這是舊約中最壯觀的神顯現。
- 與本章關聯：神的降臨是第19章的核心事件——從v9宣告到v20降臨，構成全章的高潮，為第20章頒布十誡預備場景。

#### [[02 出埃及記/第20章|第20章]]
- 本章重點：第二十章記載神在西乃山威嚴降臨頒布十誡。神「從天上」向百姓說話（22節），伴隨雷轟、閃電、角聲和山上冒煙的現象（18節）。百姓因這些現象而發顫，遠遠站立（18節），並請求摩西作中介（19節）。摩西解釋神降臨的目的是「試驗你們，叫你們時常敬畏他，不致犯罪」（20節）。隨後摩西挨近神所在的幽暗之中（21節）。神的降臨在此彰顯了祂的威嚴、聖潔和可畏，同時也顯明祂願意與百姓建立關係。
- 與本章關聯：神在雷轟、閃電、角聲和煙霧中威嚴降臨西乃山頒布十誡（1、18節）。百姓因神的顯現而極其懼怕（18-19節）。摩西解釋神降臨的正面目的：試驗百姓、使他們敬畏神不致犯罪（20節）。神從天上說話（22節），摩西進入幽暗中領受進一步指示（21節）。

## 主題發展

神的降臨在舊約中有多次記載——向亞伯拉罕顯現（創18）、在荊棘火焰中向摩西顯現（出3）、在西乃山降臨（出19）、以雲柱火柱引領（出13:21-22）、在會幕中降臨（出40:34-38）。西乃山的降臨是最壯觀的一次——神以雷轟、閃電、密雲、角聲、火焰、地震顯現，使百姓盡都發顫。來12:18-24對比西乃山的降臨（律法、恐懼、距離）與錫安山的親近（恩典、喜樂、親近）——信徒不是來到西乃山，而是來到錫安山、天上的耶路撒冷、基督這位新約的中保面前。
