# 搖胸歸摩西

## 定義

在承接聖職之禮中，摩西拿第二隻公綿羊的胸作為搖祭，在耶和華面前搖一搖。這搖祭的胸是承接聖職之禮中歸給摩西的分。摩西作為祭司的中保與設立者，代替神領受這份祭物，表明祭司的設立與權柄來自神所膏立的僕人。
