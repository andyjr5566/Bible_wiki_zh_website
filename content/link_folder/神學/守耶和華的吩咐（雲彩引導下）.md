# 守耶和華的吩咐（雲彩引導下）

## 定義

民數記 9:23 「他們遵耶和華的吩咐安營，也遵耶和華的吩咐起行。他們守耶和華所吩咐的，都是憑耶和華吩咐摩西的」。總結全章：以色列人的一切行動都建立在神透過摩西所傳達的命令，雲彩為具體執行的記號。
