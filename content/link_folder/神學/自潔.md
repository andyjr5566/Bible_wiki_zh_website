# 自潔

## 定義

自潔（קָדַשׁ *qādaš*，consecrate / sanctify）是神吩咐以色列人在祂降臨前所作的預備——包括洗衣服、不可親近女人、在第三天預備好。出19:10-15記載神吩咐摩西「叫他們今天明天自潔，又叫他們洗衣服。到第三天要預備好了」。v14-15摩西下山叫百姓自潔，他們就洗衣服；摩西說「到第三天要預備好了，不可親近女人」。CT指出「自潔」意指分別出來歸給神——百姓需要在外在行為上預備自己迎見聖潔的神。KC指出百姓必須聖潔，不能有任何不適合神聖潔的事物——夫妻間的性關係也被暫停，一切必須專注於神的顯現。BH指出自潔包括外在（洗衣服）和內在（不可親近女人）的預備——代表全人的潔淨。

- 本章重點：v10-15記載神吩咐百姓自潔三天——洗衣服（v10,14）、預備第三天（v11,15）、不可親近女人（v15）。CT指出自潔是分別出來歸給神。KC指出一切必須專注於神的顯現，連合法的事也暫停。BH指出自潔包括外在和內在的預備。
- 與本章關聯：自潔是第19章百姓預備迎見神的關鍵步驟——神降臨前必須自潔三天，顯示神的聖潔要求人預備自己才能親近祂。

## 主題發展

自潔的主題貫穿聖經。利11:44「你們要成為聖潔，因為我是聖潔的」。民11:18神吩咐百姓「自潔，預備明天吃肉」。書3:5約書亞吩咐百姓「要自潔，因為明天耶和華必在你們中間行奇事」。新約中，林前7:5保羅說夫妻可以暫時分房「為要專心禱告」——與出19:15「不可親近女人」的原則一致。約壹3:2-3「凡向祂有這指望的，就潔淨自己，像祂潔淨一樣」。彼前1:15-16「那召你們的既是聖潔，你們在一切所行的事上也要聖潔」。自潔的原則從西乃山延續到新約——迎見神需要預備和潔淨。
