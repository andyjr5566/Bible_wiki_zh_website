# 詩篇128篇

## 類型

互文

## 觸發來源

## 目前資料

### 經文結構

> **上行之詩**（第 130-134 篇為上行之詩第 13-17 篇）
>
> 1. **有福的宣告**（1-2 節）：敬畏耶和華、遵行祂道的人有福。
> 2. **家庭蒙福圖景**（3-4 節）：妻如多結果子葡萄樹，兒女如橄欖栽子。
> 3. **錫安祝福**（5-6 節）：願耶和華從錫安賜福，看見耶路撒冷平安、兒孫。

### 逐節釋義

| 經文 | 關鍵意象 | 神學含義 |
|------|----------|----------|
| **1** | 「敬畏耶和華、遵行祂道」 | 敬畏 = 認識神屬性產生的謙卑順服；遵行 = 信心的果效（雅 2:17） |
| **2** | 「吃勞碌得來的」 | 創 3:19「汗流滿面才得糊口」被救贖——勞碌有收成、享福 |
| **3** | **妻：葡萄樹**（多結果子）<br>**兒女：橄欖栽子**（圍繞桌旁） | 家庭為神立約應許傳承單位；妻兒非負擔乃冠冕（箴 12:4；17:6） |
| **4** | 「敬畏耶和華的人必然蒙福」 | 重申第 1 節，形成包含結構 |
| **5** | 「從錫安賜福」 | 錫安 = 神同在、約櫃所在、敬拜中心；福源自神同在 |
| **6** | 「看見兒女的兒女、以色列平安」 | 代代相傳的約應許（創 17:7）；平安 = 豐盛完整 |

### 與創世記雅各家庭的互文呼應

| 詩 128 要素 | 雅各家庭（創 25-50） | 異同 |
|------------|----------------------|------|
| **敬畏耶和華** | 雅各在伯特利、雅博、示劍、伯特利二次築壇、立柱、改名以色列 | 雅各屬靈成長過程，從奪取者→與神摔跤者→敬拜者 |
| **妻如葡萄樹** | 利亞（生六子一女）、拉結（生約瑟、便雅憫） | 利亞被輕看卻多結果子；拉結受愛卻晚結果子 |
| **兒女如橄欖栽子** | 十二子一女 → 以色列十二支派根基 | 兒女有爭競（流便、西緬、利未、猶大、約瑟）但最終成就神旨意 |
| **吃勞碌得來的** | 雅各為拉結服事拉班二十年（創 31:38-41） | 勞碌被神紀念，變為豐富產業（羊群、兒女） |
| **錫安賜福** | 雅各全家下埃及（創 46），神同在「我必與你同下埃及去」（46:4） | 埃及成「母胎」，從 70 人增為百萬民族（出 1:7） |
| **看見兒女的兒女** | 雅各見以法蓮、瑪拿西孫子（創 48:11）「沒想到見你的面，神還讓我見你的兒子」 | 直接應驗詩 128:6 畫面 |

### 神學主題

1. **敬畏耶和華是智慧開端** (箴 1:7) → 家庭蒙福源頭。
2. **約應許的家庭傳承** —— 創 17:7「我與你並你的後裔立約」在家庭層面實現。
3. **勞碌得享福** —— 恢復創 1-2 善工榮耀，逆轉創 3 咒詛。
4. **錫安/耶路撒冷為福流出中心** —— 最終指向新耶路撒冷（啟 21-22）。

### 應用

- **丈夫/父親**：以敬畏神為家庭屬靈帶領核心（弗 6:4「要照著主的教訓養育他們」）。
- **妻子**：多結果子的葡萄樹——生命影響力、屬靈產出（提前 2:15「在生產上得救」）。
- **兒女**：橄欖栽子——圍繞父母桌旁，接受信仰傳承（申 6:6-9）。
- **教會**：作「屬靈家庭」，代代相傳福音（提後 2:2）。
