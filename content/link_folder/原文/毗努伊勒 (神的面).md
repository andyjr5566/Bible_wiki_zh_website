# 毗努伊勒 (神的面)

## 定義

希伯來文「毗努伊勒」（Peniel / Penuel），意為「神之面」、「神的面」（創32:30「我面對面見了神、我的性命仍得保全」）。雅各與神摔跤後命名此地。出33:20「人見我的面不能存活」與本處「面對面見神、性命仍得保全」張力，由對手為「耶和華的使者」（第二位格＝道成肉身前的基督，創16:13；士6:22；13:21-23）解決。

#### [[01 創世記/第33章|第33章]]
- 本章重點：雅各從毗努伊勒出發見以掃（隱含於1節「雅各舉目觀看」之前）。毗努伊勒經歷賜給雅各面對以掃的屬靈資本：見神面存活的見證、以色列新名、大腿扭傷的謙卑記號。雅各說「見你的面如見神的面」（10節），將毗努伊勒見神面經驗投射到以掃身上。
- 與本章關聯：毗努伊勒是本章雅各一切行動（謙卑、勇氣、信心見證）的屬靈根基。何西阿12:4「他在雅博河與天使摔跤、哭泣懇求」確認對手身分。

## 主題發展

- 「神的面」在舊約為神同在、恩典、審判的多重象徵
- 見神面不能存活 vs 見神面得保全：基督論解決
- 毗努伊勒與伯特利並列為雅各兩大神顯現地標
