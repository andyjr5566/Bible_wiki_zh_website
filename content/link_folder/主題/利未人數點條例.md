# 利未人數點條例

## 定義

神吩咐摩西照利未人的宗族、家室數點他們，凡一個月以外的男子都要數點。這與以色列別支派二十歲以上能出去打仗的數點不同，利未人從一個月大就被計算，因他們專歸神服事。摩西照耶和華吩咐數點，總數為22,000名。
