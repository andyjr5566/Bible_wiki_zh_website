# 膳長_baker

## 定義

希伯來原文「שַׂר הָאֹפִים」（sar ha-ofim），字面意「烤餅長官/掌膳長」。「שַׂר」（sar）意「首領、官長」；「אֹפֶה」（ofeh）意「烤餅者、麵包師」，詞根「אָפָה」（afah）「烤、燒」。古埃及宮廷設此職，掌管王的膳食製作、防備毒害。中文和合本譯「膳長」。

- 本章重點：創40:1, 2, 5, 16, 18, 20, 22 多處出現。與「酒政」（שַׂר הַמַּשְׁקִים）並列為法老二大親信。膳長夢中「頭頂三筐白餅」（16節）對應埃及人頭頂搬運習俗（GT、KC、BH）。
- 與本章關聯：膳長作為「烤餅長官」，其夢中「烤餅被鳥吃」象徵職責失效，預表審判結局。

## 主題發展

「膳長」原文在創40章出現7次。與「酒政」形成對比：同囚、同夢、同解、異結局（一生一死）。KC、BH將此對比豫表十字架上兩強盜一得救一滅亡（路23:39-43）。LXX譯「ἀρχιμαγεῖρος」（archimageiros「掌膳長官」）。
