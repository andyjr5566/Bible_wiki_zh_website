# 苦菜

## 定義

出埃及記12:8「當夜要吃羊羔的肉；用火烤了，與無酵餅和苦菜同吃」——希伯來文 מְרֹרִים merōrim「苦菜」。CT指出苦菜作為逾越節筵席的必備三元素之一（烤羊羔、無酵餅、苦菜）——記念以色列人在埃及為奴之苦。BH指出苦菜包含lettuce、chicory、wild endive一類野生苦菜——具體的味覺記號連結為奴歷史記憶。逾越節吃苦菜是信仰教育的感官記號——代代吃記念為奴苦。
