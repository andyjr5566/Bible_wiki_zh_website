# 麥基洗德的豫表

## 定義

麥基洗德作為基督的豫表（typology），是希伯來書的核心論證之一（來七1-28）。麥基洗德無父、無母、無族譜、無生之始、無命之終，與神的兒子相似，永遠為祭司。

### 豫表的要點

| 麥基洗德 | 基督 |
|----------|------|
| 撒冷王（平安王） | 和平之君（賽九6） |
| 公義的王 | 耶和華我們的義（耶廿三6） |
| 無族譜、無始無終 | 從亙古到永遠（彌五2） |
| 至高神的祭司 | 照麥基洗德等次為大祭司（來五6） |
| 帶餅和酒 | 基督的身與血（林前十一23-26） |
| 亞伯蘭納十分之一 | 信徒當納十分之一 |
| 為亞伯蘭祝福 | 基督賜福屬祂的人 |
