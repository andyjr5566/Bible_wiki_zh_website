# 神的恩典與保護

## 定義

神對雅各全方位的眷顧：保守雅各在拉班家二十年不受虧（創31:7,42）、雅博渡口賜福改名（創32:28-29）、預備以掃心意使四百人不攻反抱（創33:4）、雅各全家平安進入迦南（創33:18「平平安安」）。雅各自承「神恩待我，使我充足」（創33:11），承認一切所有皆出於神恩典，非己力所致（申8:18）。

## 按書卷累積

### 創世記
<!-- accumulation:創世記:33:start -->
#### [[01 創世記/第33章|第33章]]
- 本章重點：雅各三次見證神恩——見以掃面如見神面（10節）、神恩待我使我充足（11節）、平安回到迦南地（18節）。以掃四百人而來本是威脅，神使其心意改變（箴21:1），化敵為友。雅各禱告（創32:9-12）在本章得到超乎所求所想的應驗。
- 與本章關聯：本章是雅各「只拿杖過約但河、如今成了兩隊」（創32:10）見證的具體兌現，神的保護超越雅各的人為籌謀（分隊、禮物、謙稱）。
<!-- accumulation:創世記:33:end -->

## 主題發展

- 神的主權保護超越人為計謀
- 恩典非因人配得、單因神信實
- 「平平安安」（Shalom）作為神同在的印證
