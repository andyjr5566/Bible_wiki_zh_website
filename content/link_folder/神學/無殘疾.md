# 無殘疾

## 定義

無殘疾指作為祭牲的動物在身體上必須完美無瑕、沒有任何缺陷或疾病。在利未記第1章的燔祭條例中，無論是獻公牛、綿羊或山羊，都明文規定必須獻上「沒有殘疾」的公羊。此一要求預表了祭牲的純潔與無罪，唯有無瑕疵的祭牲才能代替獻祭者的罪，在耶和華面前蒙悅納，成為獻與神馨香的火祭。

#### [[03 利未記/第1章|第1章]]
- 本章重點：燔祭的公牛或公羊必須是沒有殘疾的，才能在耶和華面前蒙悅納。
- 與本章關聯：本章將「無殘疾」確立為祭牲蒙悅納的絕對神學前提，彰顯神對聖潔的要求。
