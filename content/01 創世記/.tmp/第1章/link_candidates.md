# 創世記 第1章 — Link Candidates

## 神學
- 創造（從無到有） → 神學 — 1、21、27節；BH、CT、GT、KC 均指出神是萬有創造主，*bara* 是神獨有的創造行動，強調從無到有與神主權。
- 神的靈的角色 → 神學 — 2節；BH、CT、GT、KC 討論神的靈運行／覆翼／賦生命，與創造、更新、重生相關。
- 神的話（創造媒介） → 神學 — 3、6、9、11、14、20、24、26節；來源反覆指出「神說」帶出創造命令，神的話立即成就。
- 三位一體在創造中的隱現 → 神學 — 1-3、26節；來源提及 Elohim、神的靈、神說，以及「我們要」所暗示的三一工作。
- 形像與樣式 → 神學 — 26-27節；來源討論人按神形像樣式被造，涉及人的性情、代表權柄、與神關係。
- 人的尊嚴與神的形像 → 神學 — 26-27節；BH、GT、KC 強調男女同按神形像被造，賦予人獨特尊嚴與價值。
- 光與黑暗的分別 → 神學 — 3-5節；BH、CT、KC 將光暗分開連於神帶出秩序、真理、生命與屬靈光照。

## 人物
- 人 → 人物 — 26-28節；人是創造高峰，男女同被造，承受神的賜福與治理託付。

## 背景／結構
- 空虛混沌的原意 → 背景 — 2節；來源討論 *tohu wa-bohu*，有「未成形、無居住」與「審判後荒涼／變成」兩種處理。
- 六日復造的結構 → 背景 — 3-31節；來源按六日整理神創造／復造秩序，並指出每天「晚上、早晨」的結構。
- 創造日長度問題 → 解經爭議 — 5、8、13、19、23、31節；BH 提到創造日長度有 literal 24-hour 與象徵／長時期等不同解讀，KC 與 CT/GT 偏向普通一日。
- 創世記1章第1-2節間隔問題 → 解經爭議 — 1-2節；CT、GT 採間隔／復造說，KC 明確說曾接受此說但後來改採連續創造敘述，BH 較偏向神使未成形狀態進入秩序。
- 光體作記號定節令 → 背景 — 14-19節；來源討論日月星辰用於分晝夜、定節令、日子、年歲，並受神管轄而不可被敬拜。

## 原文／文化
- 各從其類 → 原文 — 11-12、21、24-25節；CT、BH 指出此語反覆出現，強調神創造中的種類、界限與秩序。
- 治理與管理 → 原文 — 26、28節；來源說明「治理／管理」是神給人的代表性權柄與管家責任。
- 神命名與主權 → 文化 — 5、8、10節；BH、CT 指出神稱光暗、穹蒼、旱地與海，命名表明主權與界定秩序。

## 主題
- 生養眾多與治理 → 主題 — 28節；神賜福人類生養、遍滿、治理，來源將其視為人的第一份託付。
- 神看為好（甚好） → 主題 — 4、10、12、18、21、25、31節；來源指出神反覆宣告「好」，造人後總結「甚好」。
- 原初素食與受造界和平 → 主題 — 29-30節；KC、CT、GT、BH 均指出墮落／洪水前神給人與動物植物作食物，呈現未流血的受造界秩序。
- 六日創造與信徒重生經歷 → 主題 — 3-31節；KC、CT 以六日創造次序作屬靈應用，對應光照、分別、結果、發光、試煉、成熟。

## 互文
- 詩篇33篇神話語創造 → 互文 — 3節；來源以詩33:6、9 說明神說有就有。
- 約翰福音1章道與創造 → 互文 — 1-3節；BH、KC、CT 將「起初」與神的話／道連於約1:1-3。
- 詩篇8篇人的治理權柄 → 互文 — 26-28節；BH、KC 以詩8篇說明人被立為受造界代表。
- 創世記9章洪水後准吃肉 → 互文 — 29-30節；來源以創9:3 對照原初植物作食物。

## 暫不建立
- 空氣／天／地／海 → 普通詞；本章有神命名與秩序意義，但單詞本身不作裸 target。
- 青草／菜蔬／樹木／魚／飛鳥／牲畜／昆蟲／野獸 → 普通受造物；來源多為逐節解說，不足以各自建立正式條目。
