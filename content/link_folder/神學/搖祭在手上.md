# 搖祭在手上

## 定義

搖祭在手上指祭司承接聖職時，將脂油、右腿及無酵餅等祭物放在亞倫和他兒子的手上，在耶和華面前搖一搖作為搖祭的動作。此舉象徵將祭物獻給神並由神賜回給祭司，表明祭司全人奉獻並從神領受事奉的恩典與供應。

#### [[03 利未記/第8章|第8章]]
- 本章重點：摩西取脂油、右腿和無酵餅放在亞倫和他兒子的手上作搖祭，在耶和華面前搖一搖，隨後拿下來燒在壇上。
- 與本章關聯：本章透過將祭物放在祭司手上作搖祭的動作，具體展現承接聖職的奉獻與神對祭司事奉的接納。
