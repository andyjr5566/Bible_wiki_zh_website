# 創十五14 亞伯拉罕之約的財物應驗

## 定義

出埃及記11:2-3「你要傳於百姓的耳中，叫他們男女各人向鄰舍要金器銀器。耶和華叫百姓在埃及人眼前蒙恩，並且摩西在埃及地、法老臣僕，和百姓的眼中看為極大」——BH明確指出此處應驗創世記15:14 神對亞伯拉罕的預言「他們所要服事的那國，我要懲罰，後來他們必帶著許多財物從那裡出來」。創世記15:13-14 是亞伯拉罕之約中最具體的「後裔寄居人地四百年」預言，包括三部分：① 後裔寄居外人地；② 服事那國受苦待；③ 神懲罰那國；④ 後裔帶著許多財物出來。出第11章 v2-3 的「要金銀器物」吩咐、第12章 v35-36 的「埃及人給以色列人所要的」具體成就 創15:14c「帶著許多財物出來」——預言具體實現。

### 出埃及記

- 本章重點：v2-3 吩咐要金器銀器 + 百姓在埃及人眼前蒙恩 + 摩西看為極大 — BH指出應驗創15:14；宣告在 v2 但執行在出12:35-36

## 主題發展

- 創15:13-14 預言四部分:①後裔寄居外人地→應驗於出1:1-7；②服事那國苦待→應驗於出1:8-14；③神懲罰那國→應驗於出7-12 十災；④後裔帶財物出來→應驗於出11:2-3、出12:35-36
- 同一預言在創15:14b「所得的財物都從那裡出來」+出3:21-22作預告（首次明示），本章11:2-3宣告吩咐，出12:35-36具體成就——三次漸進式應驗
- 對新約的預表：保羅在加3:8「既然神曉得萬國都因信稱義，便預先傳福音給亞伯拉罕說『萬國都必因你得福』」——亞伯拉罕之約延伸到萬國因信稱義，本章是亞伯拉罕之約具體應驗的第一階段
- 物理應驗:出3:21-22 (吩咐) → 出11:2-3 (重申宣告) → 出12:35-36 (具體取得)
