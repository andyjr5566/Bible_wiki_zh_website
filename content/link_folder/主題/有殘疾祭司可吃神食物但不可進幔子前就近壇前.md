# 有殘疾祭司可吃神食物但不可進幔子前就近壇前

## 定義

利未記21章22-23節規定，有殘疾的祭司雖可吃聖物（神的食物），但不可進到幔子前，也不可就近壇前，免得褻瀆聖所。

- 本章重點：第22-23節說明有殘疾祭司可吃聖物，但不得進幔子前、就近壇前。
- 與本章關聯：顯示聖潔與服事的界限，殘疾不影響食用聖物資格，但限制接近至聖所。
