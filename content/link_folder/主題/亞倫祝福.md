# 亞倫祝福

## 定義

民數記6:23-27記載亞倫和他兒子奉耶和華名為以色列人祝福的具體詞句，即「三重祝福」。此祝福詞由神親自啟示，授權祭司代宣，應許神必賜福給被祝福的百姓。

## 主題發展

成為猶太教會堂與基督教禮拜結束時的核心祝福禮儀，象徵神與百姓立約關係的恩典確據。
