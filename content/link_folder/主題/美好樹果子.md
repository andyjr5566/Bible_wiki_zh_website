# 美好樹果子

## 定義

利未記23章40節規定，住棚節第一日要拿美好樹上的果子、棕樹枝、茂密樹枝條、河旁柳枝，在耶和華面前歡樂七日，作為住棚節慶典的重要象徵，表達對神供應的感恩與喜樂。
