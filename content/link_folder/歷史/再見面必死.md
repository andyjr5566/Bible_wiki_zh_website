# 再見面必死

## 定義

出十28-29 記載法老對摩西的最後逐客令並死亡威脅：「你離開我去吧，你要小心，不要再見我的面！因為你見我面的那日你就必死！」摩西回：「你說得好！我必不再見你的面了。」雙方談判破裂，預備進入第十一章神單方面宣告長子之死與第十二章逾越節之施行。

## 主題發展

BH 指出在古代近東，被王拒見是嚴厲的懲罰，更何況是死刑威脅。「見我面的那日你就必死」是王子對平民公僕的死亡宣告——法老從外交淪為公開敵對。

CT 指出此話帶著諷刺——法老威脅摩西死，自己卻不知十日內將痛失長子（出十一5，十二29-30），他的軍隊日後也葬身紅海（出十四28）。預告摩西：

- 當晚摩西就不再能見法老，但長子之死將臨到埃及
- 法老當夜將主動召摩西來（出十二31），但「見面」已非協商場面而是送別場面
- 最後在紅海邊，法老本人追趕而至，但最終「連一個也沒有剩下」（出十四28）

GT 指出「你說得好」原文 *kên*「如此、對」——既是同意也是預言。摩西的確不再以談判者身分見法老的面（中間隔了出十一1-10 神對摩西的私下曉諭，和十二章的死災臨到）。BH 指出摩西的最後一句話類似基督對猶大之「你所做的，快做吧」（約十三27），之後是黑夜的行動。

KC 延伸：摩西自此不再「進去見法老」，11 章起改為向百姓宣告；12 章改為「你們…都要…」，全民動員；13 章改為「神帶領」。3 個主詞的轉換：法老宣告→摩西宣告→神帶領。從談判崩潰到救贖完成的運動軌跡清晰可見。

預表：法老以死威脅神的使者，正是約翰福音十五18-25「僕人不能大於主人，他們若逼迫了我，也要逼迫你們」之先例。摩西被逐之後神伸手行奇事，對照基督被逐出耶路撒冷（路十三33「先知在耶路撒冷之外喪命是不能的」）之後十字架與復活。
