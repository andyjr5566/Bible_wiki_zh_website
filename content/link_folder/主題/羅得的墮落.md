# 羅得的墮落

## 定義

羅得是跟隨亞伯蘭的侄兒，因選擇約旦河平原而逐步走向墮落。他的墮落是漸進式的，從相爭到看見、選擇、直到住在所多瑪。

## 主題發展

### 墮落步驟

1. 眼錯——看見約旦河全平原
2. 心錯——選擇約旦河全平原
3. 腳錯——往東遷移
4. 漸近罪惡——漸漸挪移帳棚直到所多瑪

### 屬靈警告

- 羅得是所有信仰墮落者的預表——被世俗富裕吸引，離開屬靈保護
- 他的選擇最終導致被擄、家產被毀、喪失妻子和女婿
- 來源：CT, GT, KC, BH
