# 哈洗遜他瑪

## 定義

哈洗遜他瑪（Hazezon Tamar），意為「棕樹的分枝」或「他瑪的剪枝」，即後來的隱基底（En Gedi）。位於死海西岸，為一處有泉水和棕樹的綠洲。
