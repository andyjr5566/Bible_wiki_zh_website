# 彈膏油和血在衣服上

## 定義

在承接聖職之禮中，摩西取點膏油和壇上的血，彈在亞倫和他的衣服上，並他兒子和他兒子的衣服上，使他和他們的衣服一同成聖。膏油代表聖靈的恩膏，血代表贖罪與潔淨，兩者結合彈在衣服上，表明祭司及其事奉的記號（衣服）皆被分別為聖，合乎主用。
