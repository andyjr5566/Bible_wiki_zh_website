# 三百一十八人

## 定義

亞伯蘭家中生養的練丁三百一十八人（創十四14），是一隊受過的親兵，亞伯蘭用以擊敗四王聯盟。傳統解經認為「三百一十八」的數值（גימטריה）相等於「以利以謝」（אֱלִיעֶזֶר，意為「神是我的幫助」）的名字數值，暗示亞伯蘭的僕人領袖即是以利以謝（創十五2）。
