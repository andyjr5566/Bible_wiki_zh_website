# 鬼魔（se'irim，公山羊）

## 定義

希伯來文 se'irim，字面意為「公山羊」，在利未記17：7指以色列人曾在埃及隨從敬拜的半山羊半人形鬼魔偶像。埃及與古代近東民族常以公山羊為神祇標號，敬拜時伴隨邪淫禮儀，因此聖經以「行邪淫」描述此背道行為。神定例禁止再向此等鬼魔獻祭，成為世世代代永遠的定例。
