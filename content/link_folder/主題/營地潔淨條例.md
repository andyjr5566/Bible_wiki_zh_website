# 營地潔淨條例

## 定義

耶和華吩咐摩西，命以色列人使一切長大痲瘋的、患漏症的、並因死屍不潔淨的都出營外去，免得污穢神所住的營（民5:1-3）。此條例確立營地聖潔原則：神住在營中，營地不可被污穢，不潔淨者必須隔離在營外。

- 本章重點：耶和華曉諭摩西，吩咐以色列人使不潔淨者出營外，以色列人照著行了。
- 與本章關聯：本章開篇確立營地聖潔的基本條例，為後續贖罪、試驗條例奠定神學基礎。
