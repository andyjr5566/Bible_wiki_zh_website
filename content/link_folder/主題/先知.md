# 先知

## 定義

聖經中第一次出現「先知」一詞在創世記二十章7節，原文 נָבִיא（nabi）。亞伯拉罕被稱為先知，強調他作為神代言人的角色——一面為神向人說話，一面為人向神禱告。

先知不是主要預測未來，而是傳達神的話語。亞伯拉罕的三重尊榮：君王（創十四14–16）、祭司（創十八23–33）、先知（創二十7）。先知最重要的職事是為人禱告。

## 主題發展

先知在舊約中逐漸發展為正式職分（摩西、撒母耳、以利亞等），新約中基督是終極先知（來一1–2）。
