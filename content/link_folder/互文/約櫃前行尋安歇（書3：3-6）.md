# 約櫃前行尋安歇（書3：3-6）

## 定義

民數記 10:33 約櫃在前行三天為百姓尋找安歇處，約書亞 3:3-6 記載約櫃先行過約旦河，帶領百姓進迦南，兩者同顯約櫃引導神子民的主題。
