# 摩西亞倫進會幕再出來祝福

## 定義

「摩西、亞倫進入會幕，又出來為百姓祝福，耶和華的榮光就向眾民顯現」（利9:23）。這是當日第二次祝福，由摩西與亞倫一同進入會幕之後發出；祝福話語落下，耶和華的榮光隨即向眾民顯現，接著有火從耶和華面前出來燒盡祭物（v24）。
