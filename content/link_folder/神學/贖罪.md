# 贖罪

## 定義

在利未記第1章的燔祭條例中，「贖罪」指透過獻祭程序來遮蓋或潔淨獻祭者的罪咎。經文記載，當獻祭者按手在無殘疾的燔祭牲頭上，祭牲作為替代物被宰殺、流血並焚燒獻與神後，燔祭便蒙悅納，達成「為他贖罪」的功效，使獻祭者在神面前重新恢復正確的關係。
