# 為百姓獻平安祭公牛公綿羊

## 定義

亞倫宰了那給百姓作平安祭的公牛和公綿羊（利9:18，參v4）。他兒子把血遞給他，他就灑在壇的周圍；又把公牛和公綿羊的脂油、肥尾巴，並蓋臟的脂油與腰子，和肝上的網子，都遞給他（v19）；把脂油放在胸上，他就把脂油燒在壇上（v20）；胸和右腿，亞倫當作搖祭，在耶和華面前搖一搖，都是照摩西所吩咐的（v21）。
