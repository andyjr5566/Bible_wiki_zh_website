# 以掃的怨恨

## 定義

以掃因被雅各騙取祝福而心懷怨恨，計劃在父親死後殺害雅各。這怨恨成為雅各逃亡的導火索，也預示了以東與以色列的長期衝突。

- 以掃聽了父親的話後放聲痛哭（34節），再求祝福（38節）
- 以掃說「他名雅各，豈不是正對嗎？」（36節）——承認雅各名如其人
- 以掃心裡說「為我父親居喪的日子近了，到那時候我要殺我的兄弟雅各」（41節）
- 以掃的怨恨不只是氣憤，而是實際的殺機（BH）
- 以掃的後裔（以東人）與雅各的後裔（以色列人）長期為仇
