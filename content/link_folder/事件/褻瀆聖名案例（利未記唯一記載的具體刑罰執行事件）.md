# 褻瀆聖名案例（利未記唯一記載的具體刑罰執行事件）

## 定義

利未記第二十四章記載了一個以色列婦人與埃及人所生的兒子，在營中與以色列人爭鬥時褻瀆了聖名並且咒詛。由於當時尚無明確的處置條例，摩西將他收在監裡等候神的指示。神明確指示必須將他帶到營外，聽見的人要按手在他頭上作見證，全會眾用石頭打死他。這是利未記中唯一具體記載刑罰執行的案例，確立了凡咒詛神、褻瀆耶和華名的必被治死的法規。
