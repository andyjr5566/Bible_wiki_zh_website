# 耶和華永生神

## 定義

創世記二十一章33節，亞伯拉罕在別是巴栽垂絲柳樹，求告「耶和華永生神」的名。原文 אֵל עוֹלָם（El Olam），意為「永恆的神」或「永活的神」。

亞伯拉罕對神的認識逐步加深：至高的神（創十四22）→ 全能的神（創十七1）→ 永生神（創二十一33）。El Olam 強調神是永遠活著、永不改變的——祂說過的話必然應許，永不落空。

## 主題發展

神的名逐步向人啟示：El（神）→ El Shaddai（全能的神）→ El Olam（永生神）→ El Elohe Israel（以色列的神）。每一次更深認識都伴隨著更深的經歷。
