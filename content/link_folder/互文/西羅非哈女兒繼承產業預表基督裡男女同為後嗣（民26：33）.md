# 西羅非哈女兒繼承產業預表基督裡男女同為後嗣（民26：33）

## 定義

民數記 26:33 記載西羅非哈無子只有五女，後在民27得承繼產業，預表在基督裡男女同為後嗣（加拉太3:28）。
