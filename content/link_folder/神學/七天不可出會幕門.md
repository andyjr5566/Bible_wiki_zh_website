# 七天不可出會幕門

## 定義

亞倫和他兒子在承接聖職期間，被要求七天不可出會幕的門，要晝夜住在會幕門口，遵守耶和華的吩咐，直到承接聖職的日子滿了。這七天的等候表明祭司的成聖需要時間，並且他們必須完全順服神的命令，停留在神的同在中，免得他們死亡。這也象徵祭司完全與世界隔離，全然歸屬於神。
