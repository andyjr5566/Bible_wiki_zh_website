# 巴珊王噩

## 定義

巴珊王，率領眾民在以得來與以色列交戰（民21:33）。耶和華吩咐摩西不要怕他，因神已將他和他的地交在以色列手中（民21:34）。以色列殺了他、他的眾子並眾民，無一倖免，得了他的地（民21:35）。
