# 燔祭灰

## 定義

祭司在壇上焚燒燔祭後所留下的灰燼。處理灰燼有嚴格的儀式規範，祭司必須先穿上特定的細麻布衣服將灰收起來倒在壇旁，隨後要換上別的衣服，將灰拿到營外潔淨之處，以維護聖俗之間的界線與潔淨。

- 與本章關聯：處理燔祭灰的條例彰顯了祭司事奉中聖與俗的區分及潔淨的要求。
