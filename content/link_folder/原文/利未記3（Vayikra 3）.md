# 利未記3（Vayikra 3）

## 定義

利未記第三章記載平安祭的條例，規定獻牛、羊、山羊作平安祭，獻祭者按手宰殺，祭司灑血，將脂油、腎、肝網子等內臟脂肪在壇上焚燒作為馨香之氣，胸搖祭和右腿舉祭歸祭司，其餘肉由獻祭者與祭司同吃，象徵與神和好共餐。

## 主題發展

本章確立平安祭為自願獻祭（3:1）、脂肪歸神（3:16-17）、祭司分胸腿（7:29-34）、信徒共餐的團契祭，預表基督成就和好（西1:20）。
