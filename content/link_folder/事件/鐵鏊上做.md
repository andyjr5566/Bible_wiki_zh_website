# 鐵鏊上做

## 定義

指以鐵鏊（平底的烘烤器具）來製作素祭的方式。獻祭者要用調油的無酵細麵，分成塊子並澆上油，以此作為素祭獻給耶和華。

本章第5至6節具體記載此法的程序，為素祭多樣化製作方式之一，強調無酵與調油的基本要求。
