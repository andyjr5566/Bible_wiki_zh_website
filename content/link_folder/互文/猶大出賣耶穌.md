# 猶大出賣耶穌

## 定義

猶大用親嘴作為記號出賣耶穌（太二六49），與雅各用親嘴騙取父親祝福（創二十七26-27）形成聖經文學上的平行。

- 雅各與父親親嘴是欺騙的記號（創二十七26-27）
- 猶大與耶穌親嘴是出賣的記號（太二六49）
- 親嘴原代表愛和信任，竟成為欺騙和出賣的途徑（CT、GT、BH）
- 出賣者最終都面臨審判：雅各受苦二十年，猶大自殺身亡
