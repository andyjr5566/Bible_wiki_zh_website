# 每日獻祭（常獻燔祭）

## 定義

神規定祭司每天早晨和黃昏各獻一隻一歲的羊羔為燔祭，並配合素祭與奠祭。這常獻的祭象徵以色列人對神持續不斷的敬拜、感恩與完全奉獻，也表明神持續與祂的子民同在。這每日的獻祭預表了基督一次且永遠有效的贖罪祭，以及信徒應將身體當作活祭獻上的屬靈原則。
