# 過紅海

## 定義

過紅海是出埃及記第14章的核心歷史事件，也是整個出埃及敘事的最高潮。事件包括：神吩咐以色列人轉回安營在海邊 → 法老率全軍追趕 → 以色列人懼怕發怨言 → 摩西宣告救恩 → 雲柱火柱轉到後邊保護 → 摩西伸杖、神用大東風一夜分開海水 → 以色列人下海中走乾地、水在左右作牆垣 → 埃及軍兵追入海中 → 神混亂埃及軍兵 → 摩西再次伸杖、海水復原淹沒埃及全軍 → 以色列人敬畏信服神。各來源（CT/GT/KC/BH）均以「過紅海」為本章標題。
