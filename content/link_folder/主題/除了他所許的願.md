# 除了他所許的願

## 定義

指拿細耳人除了為離俗所許願應獻的規定供物外，若他還有能力、甘心樂意再獻上的供物（民6:21「和他以外所能得的獻給耶和華」）。這顯示敬拜不僅止於最低要求，更包含自願的豐盛回應。
