# 漸漸的墮落

## 定義

亞伯蘭從伯特利「漸漸遷往南地」（創十二9），再下埃及。墮落往往是漸漸的，非一蹴可幾。

## 主題發展

### 亞伯蘭的漸漸墮落

1. 在伯特利有祭壇（8節）— 屬靈高峰
2. 「漸漸遷往南地」（9節）— 開始走下坡
3. 下埃及逃荒（10節）— 完全離開迦南
4. 撒謊保己（11-13節）— 道德墮落

### 屬靈原則

- 南地既非迦南也非埃及，豫表**邊界地帶**
- 許多基督徒住在「南地」——不上不下，是「邊界的基督徒」
- 墮落往往是**漸漸**的
- 亞伯蘭在埃及大大失敗，但在「漸漸遷往南地」時就已經失敗了
