# 耶和華豈有難成的事

## 定義

創世記十八章14節的核心信心宣告。原文作「耶和華豈有太奇妙（pālā')的事嗎？」，pālā' 意為「超越常理、奇妙、不可測度」。神用反問方式強調：在祂沒有難成的事。

這句話是神對撒拉不信的直接回應。撒拉從人的有限出發（「我既已衰敗」），神從祂的全能出發——問題不在事情難易，乃在神有沒有說話（路一37）。
