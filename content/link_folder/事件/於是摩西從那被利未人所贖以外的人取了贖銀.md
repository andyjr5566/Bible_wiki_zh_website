# 於是摩西從那被利未人所贖以外的人取了贖銀

## 定義

摩西執行神命，從未被利未人贖回的頭生男子中收取贖銀（民3:49）。

此行動完成贖回程序，並將銀兩轉交祭司，顯示順服與行政執行。
