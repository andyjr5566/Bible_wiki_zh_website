# 沿海大道（Via Maris）

## 定義
沿海大道（Via Maris，拉丁文「海的路」）是古代從埃及東北邊通往迦南（經今日以色列沿海平原）通往大馬士革及美索不達米亞的主要國際貿易與軍事路線，因沿地中海東岸而行而得名。其埃及至迦南段又稱「非利士地的道路」（Derech Eretz Plishtim），因為經過非利士人控制的沿海地區。出埃及記 第13章 17節「非利士地的道路雖近，神卻不領他們從那裡走」即指此路——從疏割走西乃半島北部的沿海大道，經非利士五城（迦薩、亞實突、亞實基倫、迦特、以革倫）直到迦南地，全程約 400 公里，約兩週行程，是出埃及到迦南最短的路徑。但神選擇繞道紅海曠野，因為此路沿途有埃及設立的要塞駐軍鏈（邊境防禦），未成熟的以色列百姓遇見打仗會後悔回埃及。

## 按書卷累積

### 出埃及記
<!-- accumulation:出埃及記:13:start -->
#### 第13章
- 本章重點：非利士地的道路（沿海大道 Via Maris）是出埃及到迦南最短的路——約400公里沿海岸線。但神不領百姓走此路因沿途埃及要塞鏈，百姓未預備打戰（v17）。
- 與本章關聯：v17「非利士地的道路」即指沿海大道，是繞道而行決策的地理基礎。
