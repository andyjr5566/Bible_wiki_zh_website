# 四百舍客勒

## 定義

以弗崙向亞伯拉罕開出的麥比拉洞地價：「值四百舍客勒銀子的一塊田」（創廿三15），亞伯拉罕不還價，用「買賣通用的銀子」平足付清（16節）。這個數目遠超當時的土地行情——大衛買禾場五十舍客勒、耶利米購地十七舍客勒，一名奴隸約值二十舍客勒，一年工資約十舍客勒。以弗崙口頭上說要白送，開的卻是高價；亞伯拉罕仍照價付足，為的是取得無可爭議的永久產權、不給後人留下追討的口實，也顯明他不佔別人便宜的見證。靈意上「四百」＝四十乘十，被讀作完全的試煉之數（CT）。

#### [[01 創世記/第23章|第23章]]
- 本章重點：以弗崙開價四百舍客勒銀子（創廿三15），遠高於大衛買禾場的五十舍客勒與耶利米購地的十七舍客勒；亞伯拉罕不還價，用買賣通用的銀子平足給他（16節）。
- 與本章關聯：本章是此度量衡條目的出處：全額付款確保永久產權、避免後人追討口實，也顯出亞伯拉罕不佔別人便宜的見證。
