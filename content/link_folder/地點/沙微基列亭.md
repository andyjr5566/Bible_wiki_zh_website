# 沙微基列亭

## 定義

沙微基列亭（Shaveh Kiriathaim），[[以米人]]居住之地，位於摩押平原。「基列亭」意為「雙城」，「沙微」意為「平原」或「平等」。後來此地歸屬流便支派（書十三19）。
