# 神的旨意與人的詭詐

## 定義

神通過歷史中的具體事件成就自己的預言，即使人的手段不正當，神仍能用人的錯誤推進祂的救贖計畫。這是創世記第27章的核心神學主題。

- 神曾預言「大的要服事小的」（創二十五23），以撒卻想阻止
- 利百加和雅各用詭詐騙取祝福，但神仍通過他們的錯誤成就旨意（CT）
- 以撒大大戰兢後說「他將來也必蒙福」——承認鬥不過神（33節）
- 神不因人的罪而廢掉自己的計畫（羅九16；徒四28）
- 目的正當不能使手段正當（CT，羅三8）

## 主題發展

- 神的主權高過人的自由意志（羅九10-23）
- 人的罪不能廢掉神的信實
- 信徒應當等候神，不憑自己的手段來成就神的旨意
