# 利未人代替頭生歸神

## 定義

神從以色列人中揀選利未人，代替以色列人一切頭生的（頭生男子與頭生牲畜）歸屬神。這根基於神在埃及擊殺一切頭生之日，將以色列中一切頭生的分別為聖歸神。利未人總數（一個月以上男子）為22,000，以色列頭生男子為22,273，多出的273人需按聖所的平每人贖銀五舍客勒（一舍客勒二十季拉），贖銀交給亞倫和他的兒子。
