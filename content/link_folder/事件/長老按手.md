# 長老按手

## 定義

在利未記第4章中，當以色列全會眾誤犯罪並獻上公牛犢為贖罪祭時，會 中的長老要在耶和華面前按手在牛的頭上，然後將牛宰於耶和華面前。此 按手動作代表會眾的代表（長老）將全會眾的罪轉移到祭牲身上，藉由祭 牲的代死來為會眾贖罪。
