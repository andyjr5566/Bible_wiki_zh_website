# 初熟節條例（搖祭禾捆）

## 定義

利未記 23 章 9-14 節規定進入迦南地後，收割初熟莊稼時將一捆禾捆帶給祭司，祭司在安息日的次日搖祭，並獻上一歲無瑕公綿羊羔為燔祭、調油細麵為素祭、酒為奠祭；此日不可吃餅、烘子粒或新穗，直等獻供物之日才可吃，作為世世代代永遠的定例。
