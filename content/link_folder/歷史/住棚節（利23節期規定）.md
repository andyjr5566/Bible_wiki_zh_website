# 住棚節（利23節期規定）

## 定義

利未記23章34至43節規定，七月十五日起要守住棚節七日，第一日與第八日皆為聖安息。以色列人要住在用樹枝搭成的棚裡七日，紀念神領以色列人出埃及時曾使他們住在棚裡的曠野生活。此節期正值秋天收藏地的出產，故又稱收藏節。預表上，住棚節表徵享受國度的安息，預示千禧年國度中神與人同住的歡樂與豐滿收成。
