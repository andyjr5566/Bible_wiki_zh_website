# 各按本位，各歸本纛

## 定義

指每個支派在安營與起行時，都固定在自己的方位（本位），並聚集在自己的旗幟（本纛）下。民數記 2:2 吩咐：「以色列人要各歸自己的纛下，在本族的旗號那裡，對著會幕的四圍安營」；2:17 再次強調：「他們怎樣安營就怎樣往前行，各按本位，各歸本纛」；2:34 總結：「以色列人就這樣行，各人照他們的家室、宗族歸於本纛。」這體現神對祂百姓編組的精確要求：位置不亂、歸屬不混、旗幟分明。

- 本章重點：三次提及「各歸本纛/各按本位」（2:2, 17, 34），貫穿安營、起行、順服全過程。
- 與本章關聯：本章以此為核心組織原則，串聯四面安營、起行次序與百姓順服。
