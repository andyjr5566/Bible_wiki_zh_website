# 三重祝福

## 定義

民數記6:24-26三句平行詩句構成的祝福：(1)「願耶和華賜福給你，保護你」；(2)「願耶和華使他的臉光照你，賜恩給你」；(3)「願耶和華向你仰臉，賜你平安」。神學上呈現神三重恩典作為：保護（賜福保護）、恩典（光照恩待）、和好（仰臉賜平安），後為基督教三一論預表依據之一。

## 主題發展

此三重結構在舊約獨特，展現神作為施恩主、光照者、賜平安者的屬性，奠定聖經祝福神學基礎。
