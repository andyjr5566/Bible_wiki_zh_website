# 素祭為祭司收入

## 定義

民數記7章各支派供物中均包含「調油的細麵作素祭」（13、19、25等節），按利未記2:3、6:14-18規定，素祭的大部分歸祭司（亞倫子孫）為食物，作為他們在聖所事奉的供給。因利未人（含祭司）在以色列中無產業分（民18:20-24），神指定祭物中的素祭、贖罪祭、贖愆祭等為祭司收入（民18:8-10），體現「神為他們的產業」。

## 按書卷累積

### 民數記

<!-- accumulation:民數記:7:start -->
#### [[04 民數記/第7章|第7章]]
- 本章重點：每支派供物皆含「盛滿了調油的細麵作素祭」（13、19、25、31、37、43、49、55、61、67、73、79節），總計十二個銀盤、十二個銀碗的細麵（84-85節），皆屬素祭一部分。
- 與本章關聯：本章記載大量素祭被獻上，配合利未記、民數記18章祭司條例，證明素祭為祭司主要收入來源之一。
