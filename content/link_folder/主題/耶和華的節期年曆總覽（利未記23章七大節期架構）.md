# 耶和華的節期年曆總覽（利未記23章七大節期架構）

## 定義

利未記第23章記載了耶和華神親自設立並吩咐以色列人遵守的七個節期，分別為逾越節、無酵節、初熟節、五旬節（七七節）、吹角節、贖罪日和住棚節。這些節期標明了農曆年的各個階段，慶賀收成並歸榮耀給神，同時也與以色列人的歷史事件緊密相連。在預表意義上，這七個節期描述了神藉著基督的救贖，將祂的子民從十字架帶入千禧年國度之安息的完整計畫。
