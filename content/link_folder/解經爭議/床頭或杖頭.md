# 床頭或杖頭

## 定義

創世記四十七章三十一節可讀作以色列在「床頭」上敬拜，也可讀作扶著「杖頭」敬拜。

### 文本原因

古希伯來文原先只寫輔音，表示「床」與「杖」的輔音字母相同。馬所拉文本所加母音形成「床」的讀法；七十士譯本採「杖」，希伯來書十一章二十一節也沿用該希臘讀法。

### 評估

兩種讀法涉及支撐雅各身體的是床頭或手杖，不改變句子的中心：雅各在約瑟保證歸葬應許地後敬拜神。
