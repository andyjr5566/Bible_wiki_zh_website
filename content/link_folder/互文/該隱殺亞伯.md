# 該隱殺亞伯

## 定義

該隱因嫉妒殺弟弟亞伯（創四8），是聖經中第一起殺人事件。以掃怨恨雅各並計劃殺他（創二十七41）是兄弟相殘的平行。

- 該隱因神悅納亞伯的供物不喜悅自己的，心生嫉妒殺弟弟（創四5-8）
- 以掃因雅各奪去祝福，計劃在父親死後殺他（創二十七41）
- 兩者都是弟弟被哥哥追殺
- 嫉妒和怨恨是殺人的根源（CT 該隱平行）
