# 公牛犢

## 定義

在利未記第4章的贖罪祭條例中，公牛犢被指定為受膏的祭司（使百姓陷在罪裡時）以及以色列全會眾誤犯罪時所獻的祭牲。 這反映了古代近東的文化背景中，牛是極具價值的財產，代表著最高規格的贖罪祭。 祭司或長老需按手在公牛犢頭上，宰殺後將血帶進會幕彈血七次並抹在香壇四角， 其脂油燒在燔祭壇上，其皮、肉、頭、腿等則需搬到營外潔淨之處燒盡。
