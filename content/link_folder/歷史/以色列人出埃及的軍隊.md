# 以色列人出埃及的軍隊

## 定義

出埃及記第13章18節記載「以色列人帶著兵器上去」（chamushim），第14章 v8 記載「以色列人是昂然無懼地出埃及」。CT 指出以色列人被神稱為「我的軍隊」（出七4「將我的軍隊以色列民從埃及地領出來」）——他們雖是奴隸出身，神卻視他們為軍隊。v8「昂然無懼」原文為「高舉的手」（ramah yad），表示得勝者高昂的姿態。法老因以色列人這種昂然無懼的姿態而心有不甘，決定追趕。
