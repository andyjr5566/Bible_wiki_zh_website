# 潔淨不潔淨條例（創7：2-3）

## 定義

創世記第7章第2至3節記載挪亞進方舟時，神吩咐他將潔淨的畜類每樣帶七公七母，不潔淨的畜類每樣帶一公一母。這顯示在利未記第11章正式頒布飲食條例之前，早在洪水時代，動物就已經存在「潔淨」與「不潔淨」的分類。利未記第11章並非首創此分類，而是將洪水前既有的區分標準化、律法化，作為以色列人日常飲食的規範。
