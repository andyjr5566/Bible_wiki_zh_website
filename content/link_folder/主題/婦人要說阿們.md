# 婦人要說阿們

## 定義

婦人在祭司宣讀咒詛條款後，必須雙重回應「阿們，阿們」（民5:22），表示自願接受咒詛的約束，承認神審判的公義，並確認誓約的法律效力。

### 民數記

#### [[04 民數記/第5章|第5章]]
- 本章重點：祭司叫婦人發咒起誓後，婦人要回答說：阿們，阿們（5:22）。
- 與本章關聯：「阿們」意為「誠心所願、確信不疑」，婦人的雙重阿們使苦水試驗成為具有法律約束力的自我宣誓儀式。
