# 米拉利

## 定義

利未的三子，利未三大家族之一的始祖。其子孫分抹利族、母示族，負責帳幕板、閂、柱子、帶卯的座與器具、院子柱子座橛繩子的看守，安營在帳幕北邊。

- 本章重點：民4:29,30,31,33,34,42,43,44 你要照米拉利子孫的宗族、家室數點他們...米拉利子孫所辦的事、所擔的擔...米拉利子孫在會幕中所辦的事...摩西、亞倫、會眾首領照耶和華吩咐數點米拉利子孫...米拉利子孫...共有三千二百名。米拉利為利未三子，負責抬會幕板柱。
