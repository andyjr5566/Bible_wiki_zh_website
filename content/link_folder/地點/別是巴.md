# 別是巴

## 定義

別是巴（Beersheba），原文由「別」（井）和「是巴」（盟誓/七）組成，兼具「盟誓的井」和「七之井」之意。位於迦南地南端，希伯崙以南約80公里。亞伯拉罕和以撒都曾在此寄居並與非利士王立約。
