# 烏薛的兒子以利撒反

## 定義

烏薛的兒子，哥轄孫子，利未曾孫。作哥轄宗族家室的首領（民3:19, 30），負責看守約櫃、桌子、燈臺、兩座壇與聖所內使用的器皿等至聖物（民3:31）。

- 與本章關聯：體現利未人哥轄族分工看守聖物的制度。
