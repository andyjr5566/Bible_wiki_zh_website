# 利未記 第8章

1. [[耶和華曉諭摩西（利6：1）|耶和華曉諭摩西]]說：
2. 你將[[亞倫和他兒子（祭司）|亞倫和他兒子]]一同帶來，並將[[聖衣（祭司聖服）|聖衣]]、[[膏油（聖膏油）|膏油]]，與[[贖罪祭]]的一隻公牛、兩隻[[公綿羊]]、一筐[[無酵餅]]都帶來，
3. 又[[招聚會眾到會幕門口]]。
4. [[摩西]]就照耶和華所吩咐的行了；於是[[招聚會眾到會幕門口|會眾聚集在會幕門口]]。
5. [[摩西]]告訴會眾說：這就是耶和華所吩咐當行的事。
6. [[摩西]]帶了[[亞倫和他兒子（祭司）|亞倫和他兒子]]來，[[用水洗了他們]]。
7. 給亞倫穿上內袍，束上腰帶，穿上外袍，又加上[[以弗得]]，用其上巧工織的帶子把以弗得繫在他身上，
8. 又給他戴上[[胸牌（決斷胸牌）|胸牌]]，把[[烏陵和土明]]放在胸牌內，
9. 把[[冠冕（裹頭巾）|冠冕]]戴在他頭上，在冠冕的前面釘上金牌，就是聖冠，都是照耶和華所吩咐[[摩西]]的。
10. [[摩西]]用[[膏油（聖膏油）|膏油]]抹[[會幕（帳幕整體）|帳幕]]和其中所有的，使他成聖；
11. 又用[[膏油（聖膏油）|膏油]]在壇上彈了七次，又抹了壇和壇的一切器皿，並[[洗濯盆（銅盆）|洗濯盆和盆座]]，使他成聖；
12. 又把[[膏油（聖膏油）|膏油]]倒在亞倫的頭上膏他，使他成聖。
13. [[摩西]]帶了亞倫的兒子來，給他們穿上內袍，束上腰帶，包上[[冠冕（裹頭巾）|裹頭巾]]，都是照耶和華所吩咐摩西的。
14. 他牽了[[贖罪祭]]的公牛來，[[亞倫和他兒子（祭司）|亞倫和他兒子]][[按手（samak）|按手]]在贖罪祭公牛的頭上，
15. 就宰了公牛。[[摩西]]用指頭蘸[[血]]，抹在壇上四角的周圍，使壇潔淨，把血倒在壇的腳那裡，使壇成聖，壇就潔淨了；
16. 又取臟上所有的[[脂油]]和肝上的網子，並兩個腰子與腰子上的脂油，都燒在壇上；
17. 惟有公牛，連皮帶肉並糞，用火燒在營外，都是照耶和華所吩咐[[摩西]]的。
18. 他奉上[[燔祭（olah）|燔祭]]的[[公綿羊]]；[[亞倫和他兒子（祭司）|亞倫和他兒子]][[按手（samak）|按手]]在羊的頭上，
19. 就宰了公羊。[[摩西]]把[[血]]灑在壇的周圍，
20. 把羊切成塊子，把頭和肉塊並[[脂油]]都燒了。
21. 用水洗了臟腑和腿，就把[[全羊燒在壇上]]為[[馨香（reach nichoach）|馨香]]的[[燔祭（olah）|燔祭]]，是獻給耶和華的[[火祭（isheh）|火祭]]，都是照耶和華所吩咐[[摩西]]的。
22. 他又奉上第二隻[[公綿羊]]，就是[[承接聖職之禮（millu'im）|承接聖職之禮]]的羊；[[亞倫和他兒子（祭司）|亞倫和他兒子]][[按手（samak）|按手]]在羊的頭上，
23. 就宰了羊。[[摩西]]把些[[血]]抹在亞倫的右耳垂上和右手的大拇指上，並右腳的大拇指上，
24. 又帶了亞倫的兒子來，把些[[血]]抹在他們的右耳垂上和右手的大拇指上，並右腳的大拇指上，又把血灑在壇的周圍。
25. 取[[脂油]]和肥尾巴，並臟上一切的脂油與肝上的網子，兩個腰子和腰子上的脂油，並右腿，
26. 再從耶和華面前、盛[[無酵餅]]的筐子裡取出一個無酵餅，一個油餅，一個薄餅，都放在[[脂油]]和右腿上，
27. 把這一切放在亞倫的手上和他兒子的手上作搖祭，在耶和華面前搖一搖。
28. [[摩西]]從他們的手上拿下來，燒在壇上的[[燔祭（olah）|燔祭]]上，都是為[[承接聖職之禮（millu'im）|承接聖職]]獻給耶和華[[馨香（reach nichoach）|馨香]]的[[火祭（isheh）|火祭]]。
29. [[摩西]]拿羊的胸作為搖祭，在耶和華面前搖一搖，是[[承接聖職之禮（millu'im）|承接聖職之禮]]，歸摩西的分，都是照耶和華所吩咐摩西的。
30. [[摩西]]取點[[膏油（聖膏油）|膏油]]和壇上的[[血]]，彈在亞倫和他的衣服上，並他兒子和他兒子的衣服上，使他和他們的衣服一同成聖。
31. [[摩西]]對[[亞倫和他兒子（祭司）|亞倫和他兒子]]說：把肉煮在[[會幕（帳幕整體）|會幕]]門口，在那裡吃，又吃[[承接聖職之禮（millu'im）|承接聖職]]筐子裡的餅，按我所吩咐的說（或作：按所吩咐我的說）：這是亞倫和他兒子要吃的。
32. 剩下的肉和餅，你們要用火焚燒。
33. 你們七天不可出[[會幕（帳幕整體）|會幕]]的門，等到你們[[承接聖職之禮（millu'im）|承接聖職]]的日子滿了，因為主叫你們七天承接聖職。
34. 像今天所行的都是耶和華吩咐行的，為你們[[贖罪]]。
35. 七天你們要晝夜住在[[會幕（帳幕整體）|會幕]]門口，遵守耶和華的吩咐，免得你們死亡，因為所吩咐我的就是這樣。
36. 於是[[亞倫和他兒子（祭司）|亞倫和他兒子]]行了耶和華藉著[[摩西]]所吩咐的一切事。

## 本章整理

### 神吩咐、會眾觀禮：公開設立的祭司職（v1-5）

本章以[[耶和華曉諭摩西（利6：1）|耶和華曉諭摩西]]起頭，把一至七章的條例轉為歷史敘事——祭司制度正式設立。KC 指出，這場設立完全按出埃及記二十九章的預告執行；《串珠聖經註釋》進一步解開編排上的疑問：按時序，八至十章本應緊接出埃及記四十章的會幕落成，但祭司必須先握有整套獻祭條例、又必須先為自己獻祭才能供職，所以利未記一至七章的條例必須先行。[[摩西]]奉命帶來的清單本身就是全章的縮影：[[亞倫和他兒子（祭司）|亞倫和他兒子]]、[[聖衣（祭司聖服）|聖衣]]、[[聖膏油|膏油]]、[[贖罪祭]]的公牛、兩隻[[公綿羊]]、一筐[[無酵餅]]；丁良才指出，兩隻公綿羊一隻作燔祭（18-21節），一隻作承接聖職之禮的祭（22-29節）。

神又特意吩咐[[招聚會眾到會幕門口]]。CT 解為「為要觀禮以見證他們的任職出於神」；丁良才則點出這公開性的防衛作用——==這職分不是亞倫自取的（來五4），也不是摩西徇私授予哥哥的==，日後哥拉黨的反抗（民十六3）正顯明會眾親眼見證何等要緊。至於「會眾」的實指，丁良才、《串珠》與 BibleHub 三家一致認為是長老與百姓的代表，因為[[會幕（帳幕整體）|會幕]]院子容不下全民；丁良才並補充，承接聖職之禮既歷時七天，百姓或可輪流觀禮、或在鄰近山坡遠觀。摩西一句「這就是耶和華所吩咐當行的事」為底下每個動作定了調：全章反覆出現「都是照耶和華所吩咐摩西的」（9、13、17、21、29、34-36節），這句副歌式的短語才是本章真正的骨架。

### 三重潔淨的起頭：水、聖衣與膏油（v6-13）

CT 為本章的成聖之禮歸納出一條清楚的線索：==祭司要經過三重潔淨——水的潔淨（6節）、膏油的潔淨（12節）、血的潔淨（24節）==，正應約壹五8「作見證的原來有三：就是聖靈，水，與血」。[[用水洗了他們]]是第一步。丁良才用原文歸納法細考「洗」字在利未記其餘二十五處的用法，判定此處是洗全身而非僅洗手腳，並推測是在圍子或旁屋中進行、不在眾人面前；啟導本認為洗禮之處即前院的[[洗濯盆（銅盆）|銅洗濯盆]]，情形有似新約的水禮，BibleHub 亦以此預表洗禮。值得注意的是 KC 提出一個略微不同的角度：這裡的洗不是要洗去什麼污穢，而是要藉著話語的水==保持==潔淨——洗，是事奉的常態，不是一次性的補救。

接著是穿衣。CT 與《聖經精讀本》都注意到大祭司與一般祭司的服飾規格有別，這差別本身就標示了職分的層級：

| | 大祭司亞倫（v7-9） | 亞倫的兒子（v13） |
| :--- | :--- | :--- |
| 件數 | 七件 | 三件（丁良才計入褲子作四件） |
| 內容 | 內袍、腰帶、外袍、[[以弗得]]、[[胸牌（決斷胸牌）]]、冠冕、金牌 | 內袍、腰帶、裹頭巾 |
| 頭飾 | 冠冕加[[金牌聖冠（nezer hakodesh）]] | [[冠冕（裹頭巾）]]，前面沒有金牌 |

[[烏陵和土明]]放在胸牌內（v8），字義分別是「光明」與「全備」。各家在此不約而同地保持節制：CT 說解經家的各種推測「不足採信」，啟導本指出此二物在被擄時失蹤、聖經最後提及是尼希米記七65；《舊約聖經背景註釋》則從古代近東取證——巴比倫的塔米圖文獻與美索不達米亞的石子占卜（以鮮明與烏黑的石塊表肯定與否定）顯示這類求問方式廣為人知，烏陵之為「光」大概對應白石，土明或與蘇美語的「真實石」相關。

膏油隨後臨到人與物。摩西先膏抹帳幕與其中所有的，再[[彈膏油七次]]於壇上。CT 特別點明：會幕一切器具中==僅有燔祭壇例外——除了抹油，還要在壇上彈七次==；丁良才申論其意義：這顯明[[銅壇（燔祭壇）|燔祭壇]]雖安置在會幕以外，卻仍是至聖的，惟其至聖，才能使一切獻在壇上的供物成聖（出二十九37；太二十三19）。最後是[[倒膏油在頭上]]（v12）——不是彈，不是抹，而是「倒」。丁良才由此讀出份量的差別：亞倫所受的膏油比他兒子所受的更多（比較12節的「倒」與30節的「彈」），故利二十一12說「耶和華膏油的冠冕在亞倫的頭上」，論眾祭司卻只說「耶和華的膏油在他們身上」（利十7）。啟導本補充，舊約受膏的職分有祭司、君王、先知三種，卻無人兼領三職；「彌賽亞」與「基督」都是「受膏者」之意，三職最終集於基督一身。

> [!important] 本章樞紐：受膏與抹血的先後
> ==亞倫是在尚未抹血之前就受膏（v12），他的兒子卻是先抹血（v23-24）、後受彈膏油（v30）。==丁良才與 KC 各自獨立指出這個次序差異，並給出同一個解釋：亞倫在此預表基督——聖靈可以直接臨到那位無罪的完全人；我們卻是罪人，聖靈只能臨到已被基督之血潔淨的人身上。次序本身就是神學。

### 贖罪祭與燔祭：先潔淨壇，再全然獻上（v14-21）

三祭的次序不是隨意的。丁良才讀出其中的救恩邏輯：==先有[[贖罪祭]]（14節，稱義），次有[[燔祭（olah）|燔祭]]（18節，成聖），後有平安祭性質的承接聖職之祭（22節，與主相交）==；《聖經精讀本》的說法相合——先獻贖罪祭使罪得赦免，再獻燔祭表示完全的奉獻與忠心，最後獻平安祭將感謝頌讚歸給神。三祭都以[[按手（samak）|按手]]開始：KC 指出按手即認同，[[按手在贖罪祭公牛頭上]]意味著亞倫父子承認自己的罪已歸到祭牲身上；他們作祭司的資格不出於自身，而全然出於他們所認同的那一位。

也正因祭司尚未就職，全章的獻祭動作都由摩西執行。KC 對此有一句要緊的話：==摩西不需要承接聖職，他能作這一切，因為他預表那位「公義的教師」==；丁良才則說摩西此時「作了臨時的祭司」，是神所認可的立約中保（詩九十九6）。[[血]]先臨到壇：摩西用指頭[[抹血在壇四角]]，使壇潔淨，再把餘血倒在壇腳（v15）。《串珠》比較四章的贖罪祭禮儀，指出差別在於四章要在聖所幔子前彈血、抹香壇四角，這裡卻只抹燔祭壇的四角；丁良才解釋原因——這祭並非為亞倫某一件特定的罪而獻。CT 由此推出獻祭的空間邏輯：==必須先潔淨壇，才能在壇上獻祭==。

至於公牛的其餘部分，則[[營外焚燒公牛|連皮帶肉並糞用火燒在營外]]（v17）。丁良才給出兩個具體理由：其一，按定例祭司不能吃自己的贖罪祭肉；其二，摩西是亞倫的哥哥、又是家長，同樣不能吃。CT 與 BibleHub 則同聲指向新約：脂油燒在壇上成為馨香蒙神悅納，皮肉糞燒在營外卻是審判的火，預表基督在城門外受苦——「我們也當出到營外，就了祂去，忍受祂所受的凌辱」（來十三13）。接著獻上燔祭的公綿羊，[[全羊燒在壇上]]為[[馨香（reach nichoach）|馨香]]的[[火祭（isheh）|火祭]]（v21）。丁良才注意到此處的血只「灑在壇的周圍」而不再抹四角，因為壇已被贖罪祭牲的血潔淨了；CT 補充燔祭牲須先放血、切塊，並用水洗淨臟腑與腿，才能整隻焚燒。燔祭表徵蒙神悅納的完全奉獻，KC 說得更直白：我們承接聖職的獻上，必須與燔祭同一性質，也就是全人的交付。

### 承接聖職之禮：血在耳手腳、祭物在手上（v22-30）

第二隻公綿羊是[[承接聖職之禮（millu'im）|承接聖職之禮]]的羊。KC 與《聖經精讀本》都指出這是[[平安祭（shelamim）|平安祭]]的一種特殊形式，而它的特殊，恰恰顯在血的用法上。摩西把血[[抹血在右耳垂、右手大拇指、右腳大拇指|抹在亞倫和他兒子的右耳垂、右手的大拇指、右腳的大拇指上]]（v23-24）。CT、丁良才、啟導本與 BibleHub 給出高度一致的解讀：==耳聽神的話、手作神的工、腳行神的路==；KC 更點出次序的意義——血先抹在耳上，因為一切真實的祭司事奉都從「聽」開始，先聽明主要什麼，手的行動與腳的行走才有方向。《舊約聖經背景註釋》則從儀式學補上另一層：古代近東的抹血常集中在身體的邊緣與入口，血在此兼有潔淨與保護的雙重功用。丁良才並提供一條漂亮的互文：長大痲瘋者得潔淨時，祭司也照樣把血抹在他的右耳、右手與右腳上（利十四14-17）——同一個記號，同時說出「歸神為聖」與「得潔淨」兩件事。

隨後是[[搖祭在手上]]（v27）。摩西把[[脂油]]、肥尾巴、腰子、右腿，連同筐子裡的三種餅（無酵餅、油餅、薄餅）一併放在亞倫和他兒子的手上搖一搖。「承接聖職」的原文意象在此顯露無遺：《串珠》說原意是「充填物」，《聖經精讀本》直譯為「裝滿」，CT 則譯作 to fill the hand——手本來是空的，如今被裝滿了。《舊約聖經背景註釋》指出「充滿其手」這片語在亞喀得文獻中既可用於祭司也可用於君王：亞述王阿達德尼拉里二世即位時，正是用這話描述權杖被放進他手中。奉獻的意象因此是雙關的——手被充滿，同時也是權柄被授予。KC 則把目光放在被托起的物件上：脂油與餅分別說出主的精力與完全的生命，右腿說出祂在地上行走的能力；這一切被舉在神面前搖一搖，是把神在基督生命中所看見的榮耀，再次呈到祂面前。

這祭的分配方式處處與尋常平安祭相左，丁良才與《利未記雷氏研讀本》把差異點得很清楚：

| 項目 | 尋常平安祭（利七） | 承接聖職之禮（利八22-32） |
| :--- | :--- | :--- |
| 胸 | 歸亞倫子孫（七31、34） | 歸摩西——他此時代行祭司之職 |
| 右腿 | 歸灑血的祭司（七33） | 連同三餅燒在壇上，不歸祭司 |
| 餅 | 感謝祭可用有酵餅（七13） | 只用無酵餅 |
| 吃的人 | 獻祭者與親友皆可（七15） | 唯獨亞倫和他兒子 |
| 吃的地點 | 任何潔淨之處 | 會幕門口 |

理由其實只有一個：==亞倫父子這一次是「被獻」的一方，不是收祭的一方==。丁良才說，本應歸祭司的右腿和餅這次都燒在壇上，因為他們在這祭上「有份」；雷氏研讀本同樣指出，他們不能吃這搖祭，因為這祭正是為他們而獻的。至於[[搖胸歸摩西]]，《串珠》讀出一層意味深長的暗示：搖祭的胸歸摩西，舉祭的腿歸神，==亞倫的被按立是神與摩西共同完成的==。全禮以[[彈膏油和血在衣服上]]作結（v30）——CT 指出衣服代表的是職任，人與所穿的職分要一同成聖；KC 則說油與血並用，正說明聖靈與基督的寶血是我們得以在神面前有份的雙重根基（彼前一2）。

### 七天不出門：從一日之禮到一生之職（v31-36）

承接聖職不是一天的典禮，而是[[七天不可出會幕門|七天晝夜不出會幕門]]。摩西吩咐他們[[煮肉在會幕門口吃]]，又吃筐子裡的餅，剩下的用火焚燒（v31-32）——丁良才解釋，剩餘必須焚燒，是因為七天之內天天要獻同樣的祭。KC 對這頓飯有極貼切的說明：==前面一切禮儀把他們擺在祭司的地位上，這頓飯卻給他們供職的力氣==；沒有這食物，就沒有真實的祭司事奉。而「七天」的份量，各家都指向「完全」：CT 說七天表徵完全，即一生活在聖潔的原則之下；KC 說七是完全之數，第八天（利九1）開啟新的一頁，所以承接聖職的期限其實就是我們的一生——我們是「常常」獻上讚美之祭的祭司（來十三15）。摩西也重申，像今天所行的都是耶和華吩咐行的，為要為他們[[贖罪]]（v34）。

「免得你們死亡」（v35）這句警告，把本章與後面兩章牢牢鎖在一起。《舊約聖經背景註釋》指出，古代近東確有類似的禁足觀念——蘇美的恩圖女祭司在特定期間一步也不可踏出廟宇；祭司留在聖所範圍內能承受聖物的能力，一旦離開便有性命之危。但 KC 的觀察更直指要害：本章末了「亞倫和他兒子行了耶和華藉著摩西所吩咐的一切事」（v36）是何等美好的順服，==然而利未記十1 隨即記下拿答、亞比戶獻上凡火而死==。同一雙手，可以完全遵行，也可以擅自行動；第八章反覆的「都是照耶和華所吩咐摩西的」與第十章的「這是耶和華沒有吩咐他們的」，恰成一組刺目的對照。

### 跨章脈絡與預表：從出二十九到來十三

承接聖職之禮的各個動作，構成一條從潔淨、授職到獻祭、供養的完整鏈條：

  | 步驟 | 經文 | 意義 |
  | :--- | :--- | :--- |
  | 洗身 | v6 | 除去污穢；KC：不只為潔淨，更為**保持**潔淨 |
  | 穿聖衣 | v7-9、13 | 職分的尊榮不出於自己（出28:2） |
  | 膏抹會幕與壇 | v10-11 | 器具也須分別為聖，才可與神相連 |
  | 倒膏油在亞倫頭上 | v12 | 盡大祭司職任的能力是從聖靈來的（丁良才） |
  | 贖罪祭 | v14-17 | 稱義 |
  | 燔祭 | v18-21 | 成聖 |
  | 承接聖職之祭 | v22-29 | 與主相交 |
  | 抹血在耳手腳 | v23-24 | 聽神的話、作神的工、行神的路 |
  | 彈膏油和血 | v30 | 聖靈與寶血是站立的雙重根基（彼前1:2） |
  | 吃祭肉七天 | v31-35 | 供職的力氣；一日之禮化為一生之職 |

  丁良才在本章開頭為以色列人的敬拜訂出**四個要點**，本章正好把四點齊備：「（一）須在一定的**地方**（就是在會幕那裡）；（二）按一定的**時期**（就是每日，每安息日，並諸節期）；（三）用一定的**禮節**（就是獻各祭的條例）；（四）藉一定的**中保**（就是大祭司和眾祭司）。」

  他並列出祭司的**七層責任**——這份清單說明祭司遠不只是「獻祭的人」：

  1. 替人辦理屬神的事，即獻上禮物和贖罪祭（來5:1）
  2. 為百姓祝福（民6:23-27）
  3. 在神和人中間作中保（民16:46-48）
  4. **將神的典章教訓以色列人**（利10:11，申33:10，瑪2:6）
  5. **判斷官長所不能斷的訟事**（申21:5，代下19:8-10）
  6. 領百姓禮拜神（珥2:17）
  7. 管理會幕中一切的事（民18:7）

  《舊約聖經背景註釋》把這職分放進古代近東的框架裡看：「祭司是古代近東**所有文化共有**的發展。沒有專人擔任祭司職務的**只有貝督因部落**。」該書描述其階層：「最高級者是大祭司，**他的勢力有時堪與君王匹敵**。其次是負責日常儀式和祭禮的中等祭司，再其次是樂師，最下等則是廟役。」而以色列與四鄰的關鍵差異在於膏抹的意涵：「按照以色列的慣例，膏抹是**揀選**的象徵，經常和聖靈的賞賜有密切關係……**埃及和赫人相信膏抹能夠保護人不受冥界神祇勢力之害**。」

  KC 則把整章拉到新約的應用上：「在亞倫和他兒子身上，我們看見主耶穌作大祭司、以及信徒作祭司的預表或影兒。**所有信徒都是祭司**（彼前2:5）。那是原則。**可惜並非教會中所有的信徒都在盡祭司的功用。**舊約的圖畫顯明我們如何能實際地活出這祭司的職分。」他並指出這職分不是私人的事：「祭司的事奉不只是個人的事，它關乎整個會眾……**信徒的祭司職分，特別表現在信徒作為教會聚集在神所居之處的時候**（太18:20）。」

  丁良才在全章末尾給了一段最遼闊的預表——他把會幕的三層空間讀成三個世界：「有人以為**會幕的院子是預表世界，會幕的聖所是預表高天，至聖所是預表神的住處**。一切祭牲是在會幕的院子眾人面前宰的，照樣耶穌也是在耶路撒冷城外眾人面前捨命的……大祭司一年一次帶著贖罪祭的血經過了聖所，進到至聖所的幔內，將血奉獻在法櫃施恩座前；照樣**基督一次經過了高天，進到神面前，奉獻自己的寶血，就贖了萬世萬代人的罪**（來4:14，9:11-12）。」

  而他由來9:23「天上的本物自然當用更美的祭物」推出一個大膽的結論：「這大概是因為==**高天曾被離了本位之天使的罪污穢了**==（猶6）。基督既已獻上自己的血，**不但贖了人的罪……也是潔淨了高天**；神是藉著祂叫萬有，無論是在地上的、天上的，都與自己和好了（西1:20）。」
