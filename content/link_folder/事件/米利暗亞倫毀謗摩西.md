# 米利暗亞倫毀謗摩西

## 定義

米利暗和亞倫因摩西娶古實女子為妻，毀謗摩西說：「難道耶和華單與摩西說話，不也與我們說話嗎？」，這話耶和華聽見了（民12:1-2）。
