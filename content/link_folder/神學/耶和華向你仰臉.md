# 耶和華向你仰臉

## 定義

民數記 6:26 「願耶和華向你仰臉，賜你平安。」三重祝福的第三句。「仰臉」原文 nasa panim 指神轉臉注視、接納、不掩面（相對於「掩面」表棄絕）；「平安」shalom 涵蓋完全、和睦、興旺、屬靈安息的整全狀態。此句將祝福推向高潮：神以完全的接納賜下整全的平安。

## 主題發展

祭司祝福三句呈遞進結構：賜福保護 → 恩典同在 → 接納平安。猶太禮儀視此為最高祝福；基督教視基督為「我們的平安」（弗 2:14），成就此祝福的終極應驗。
