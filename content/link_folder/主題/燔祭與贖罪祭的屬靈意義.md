# 燔祭與贖罪祭的屬靈意義

## 定義

產婦滿了潔淨日子需獻一歲羊羔為燔祭，雛鴿或斑鳩為贖罪祭。燔祭象徵全然奉獻與神悅納，贖罪祭則為了潔淨產血的不潔並贖罪，使產婦的血源得潔淨，恢復與神的交通。
