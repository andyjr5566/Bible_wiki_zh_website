# 派

## 定義

指祭司亞倫和他兒子按耶和華吩咐，將具體職任、抬擔物件分派給各利未人（民4:19, 27）。第19節「派他們各人所當辦的，所當抬的」；第27節「他們所當抬的，要派他們看守」。此為維持會幕服事秩序、防止混亂與觸犯聖潔的行政動作。

- 本章重點：經文記載亞倫和他兒子派定哥轄人各人職任與抬擔（第19節）、革順人所當抬的要派他們看守（第27節）。
- 與本章關聯：派定職任是「不可使哥轄人被剪除」保護措施的核心環節，確保各人按序服事，聖物得以有序遮掩抬運。
