# 基督頌

## 類型

互文

## 觸發來源

## 目前資料

### 定義

指新約中呈現基督「降卑—升高」榮耀軌跡的詩歌式神學陳述，最經典為 **腓 2:6-11**（肯西斯頌），此外約 1:1-18、西 1:15-20、來 1:1-4 亦屬此類。

### 核心經文對照

| 經文 | 重點 | 關鍵詞 |
|------|------|--------|
| **腓 2:6-11** | 肯西斯：雖有神的形像，不以此為強奪，虛己取奴僕形像，死在十字架；神將祂升為至高，賜名超萬名 | 降卑、虛己、奴僕形像、十字架、升高、超萬名 |
| **約 1:1-18** | 道成肉身：太初有道，道就是神，道成了肉身，住在我們中間 | 道、成肉身、榮光、恩典真理 |
| **西 1:15-20** | 萬有之首：是不能看見之神的像，首生者，萬有靠祂立住，藉十字架血使萬有與神和好 | 神的像、首生者、和好、十字架血 |
| **來 1:1-4** | 兒子優越：神藉著兒子曉諭我們，承受萬有，洗淨了罪，坐在至高者右邊 | 承受萬有、神榮耀發光、洗淨罪、坐在右邊 |

### 腓 2:6-11 結構分析

```
第 1 節（6）  降卑前狀態：本有神的形像，不以與神同等為強奪
第 2 節（7-8） 降卑過程：虛己 → 取奴僕形像 → 成人樣式 → 死在十字架
第 3 節（9-11） 升高結果：神將祂升為至高 → 賜名超萬名 → 萬膝跪拜、萬口承認
```

### 神學三大支柱

1. **肯西斯** —— 基督自願放下神性榮耀特權，非放棄神性本質。
2. **道成肉身** —— 完全神、完全人，二性合一位格。
3. **代贖與榮耀** —— 十字架死是順服極點，復活升天是神對順服的肯定。

### 與創世記關聯

- **創 1:26-27** —— 人照神形像被造；基督是「不能看見之神的像」（西 1:15），恢復人失喪的形像。
- **創 3:5-6** —— 亞當謀取「如神知道善惡」；基督「不以與神同等為強奪」（腓 2:6），扭轉悖逆。
- **創 28:12** —— 雅各夢見天梯；基督是連接天地的真天梯（約 1:51）。

### 謙卑像基督 (腓 2:5-8) 實踐

> 「你們當以基督耶穌的心為心」（腓 2:5）
- 不爭競、不貪圖虛浮榮耀（2:3）
- 看別人比自己強（2:3）
- 不顧自己的事，顧別人的事（2:4）
- 願意降卑、服事、捨己（2:7-8）
