# 曉諭

## 定義

本章四次出現（v.1, 5, 11, 12），均為「耶和華曉諭摩西說」或「你曉諭以色列人說」。指神將律法條例透過摩西傳達給百姓的啟示-傳遞機制，強調律法源自神、經由先知中介、針對會眾。

## 主題發展

四次曉諭分別對應四大段落：營地潔淨（v.1-4）、犯罪賠償（v.5-10）、嫉妒條例引言（v.11-12）、嫉妒條例細則（v.12以下）。結構上呈現「神→摩西→百姓」的權威鏈條。
