# 聖俗潔淨不潔淨的分別

## 定義

利未記10章10節指出祭司禁酒的目的，是「使你們可以將聖的、俗的，潔淨的、不潔淨的，分別出來」。此為祭司核心職能：在禮儀與道德上辨別屬神（聖、潔淨）與屬世界/污穢（俗、不潔淨）的界線。拾穗註釋引述「聖潔範圍」概念：至聖所為軸心，向外圈圈遞減，祭司負責維持正確聖潔與潔淨程度。BH研經註解指出此原則的新約應用是屬靈分辨的比喻——保持清醒才能分辨敬拜中何為屬神、何為世俗影響。
