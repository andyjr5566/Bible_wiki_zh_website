# 「撒冷」名字含義

## 定義

「撒冷」（希伯來文 *Shalem* / *Shalom*）意為「平安」（peace）、完全、和好。[[麥基洗德]]作撒冷王，即「平安之王」。希伯來書七2將「撒冷王」解釋為「平安王」。

## 主題發展

### 字義分析

- *Shalem*（שָׁלֵם）/ *Shalom*（שָׁלוֹם）= 平安、完全、和好、繁榮
- *Shalem* 的字根含義：完整、完滿、和平
- 希伯來書七2：「又名撒冷王，就是平安王的意思」
- 「耶路撒冷」（*Yerushalayim*）= 耶路 + 撒冷 = 「平安的根基」或「平安之城」

### 屬靈意義

- 撒冷 = 平安，是神賜給人的终极恩典
- 公義在先，平安在後——人必須先因信稱義（公義），才能得著與神和好（平安）
- 麥基洗德為「公義王」又作「平安王」——豫表基督先成就公義，再賜平安
- 以賽亞書三十二17：「公義的果效必是平安，公義的效驗必是平穜，直到永遠」
- 撒冷可能即日後的耶路撒冷——「平安之城」至今仍等候真正的平安之君

### 待確認事項

- 撒冷是否即日後耶路撒冷的前身，學者意見不一
