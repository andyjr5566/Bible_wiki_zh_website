# 拿單

## 定義
大衛朝先知（撒下7:2；代上17:1），神用以責備大衛犯罪（撒下12）、傳達大衛之約（撒下7）、輔助所羅門登基（王上1）。名字意為「賜予者」或「神賜予」。

## 主題發展
- 先知三重職事：傳達神約（撒下7）、責備君王罪（撒下12）、決定王位繼承（王上1）
- 「先知不畏權勢、忠心傳達神語」典範，非阿諛奉承（對比亞哈四百先王先知王上22）
- 拿單詩歌被聖殿敬拜採用（代下29:30），屬靈影響跨代
