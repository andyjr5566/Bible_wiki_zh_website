# 以撒偏愛以掃

## 定義

以撒偏愛長子以掃，因常吃他作的野味，不理會神「大的要服事小的」的旨意，堅持要給以掃祝福。這種偏愛導致家庭分裂。

- 以撒愛以掃因常吃他作的野味（創二十五28；二十七4、9、14）
- 以撒137歲時仍想祝福以掃，無視神諭（CT）
- 以撒說「不知道哪一天死」，急於祝福以掃（2節）
- 以撒的偏愛是造成兒女不和睦的主要因素（CT）
- 以撒的錯：偏愛、貪吃美味、不理會神的旨意、糊裡糊塗（CT）
