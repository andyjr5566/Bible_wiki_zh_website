# 脂油和血都不可吃

## 定義

利未記第3章第17節頒佈了一項嚴格的飲食與獻祭定例：「在你們一切的住處，脂油和血都不可吃；這要成為你們世世代代永遠的定例。」此規定結合了兩項核心神學原則：脂油代表動物身上最精華的部分，歸屬於耶和華；血則代表動物的生命，用於贖罪。禁止食用脂油是承認神對受造物精華的主權；禁止吃血則是尊重生命的神聖性，因為血是生命，且已被指定用於祭壇贖罪。這項定例適用於以色列人一切的住處，跨越時空成為世世代代永遠的定例。
