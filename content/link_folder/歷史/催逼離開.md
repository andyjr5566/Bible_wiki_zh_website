# 催逼離開

## 定義

出埃及記11:1「他容你們去的時候，總要催逼你們都從這地出去」。CT指出原文「催逼」(khrṣ / 希伯來文 ḥāraṣ 在某些語境表「強迫驅趕」) 含有無法容忍而以暴力驅趕的意思——從前九章「不容」到十章容讓、再到第十一章「催逼」是法老態度的最大反轉。CT原文釋：「都從這地出去是一人一蹄都不留——全部男女老少和牛群羊群，埃及人都要把他們驅趕離開」。「催逼」與第10章 v28 法老對摩西的逐客令形成呼應並反轉：原本法老威脅「你再來見我必死」 threatening 摩西離開；末災後法老反過來主動驅趕以色列全族。

- 本章重點：v1 預告末災後法老會「催逼」以色列人離開——CT指出從「不容」到「催逼」是世人「不見棺材不流淚」的寫照，也是神懲罰迫使法老轉變態度

## 主題發展

- 法老態度演變時間軸：第5章「不容」（出5:2）→ 第8章「容你們去獻祭」（出8:8 妥協）→ 第10章「容你們去」（出10:8 條件性容讓）→ 第11章「催逼」彻底反轉（出11:1）
- CT屬靈意義：世人「不見棺材不流淚，不到最後關頭不醒悟」——法老是世人剛硬心腸的屬靈寫照
- 應驗：第12章逾越節當晚法老半夜召摩西亞倫說「起來！連你們帶以色列人都從我民中出去」（出12:31）
- 呼應第6章以來「容我的百姓去好事奉我」的呼聲——最終並非法老主動放人，而是法老被神懲罰催逼放人
