# 希伯倫族（Chevron）

## 定義

希伯倫族是利未人哥轄的三子希伯倫的後裔所組成的家族（民3:19, 27）。他們屬於哥轄諸族之一，同樣負責看守聖所至聖物（民3:31）。在普查中計入哥轄諸族總數八千六百名（民3:28），安營於帳幕南側（民3:29）。
