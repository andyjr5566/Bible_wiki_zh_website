# 承認罪

## 定義

指犯罪者在察覺自己犯了罪之後，在神面前具體承認所犯之罪的行動。 在利未記第5章的贖罪祭條例中，當人因不發言見證、摸了不潔之物或冒失發誓而有了罪， 一旦知道了自己的罪，就要承認所犯的罪，並牽引贖愆祭牲到耶和華面前， 祭司便為他贖罪，使他必蒙赦免。這表明認罪是得蒙赦免與恢復聖潔關係的必要前提。

- 本章重點：當人在各樣事上有了罪，知道後就要承認所犯的罪，並獻上贖愆祭以求贖罪與赦免。
- 與本章關聯：認罪是啟動贖罪程序的首要事件，銜接了人的罪愆與祭司的贖罪行動。
