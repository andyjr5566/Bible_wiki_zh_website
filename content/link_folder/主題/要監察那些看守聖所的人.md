# 要監察那些看守聖所的人

## 定義

民數記3:32 記載以利亞撒「要監察那些看守聖所的人」。此監督職能包括：確保各族按規定看守器具（民3:25, 31, 36）、執行遮蓋聖物程序（民4:5-15）、防止非利未人或外人擅入（民3:10, 38）、維持安營秩序與人員調度。監察體現聖潔傳遞的責任鏈：祭司以利亞撒代表神監察利未人，利未人再守護聖所不被褻瀆。
