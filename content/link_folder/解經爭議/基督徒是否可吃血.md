# 基督徒是否可吃血

## 定義

創九4禁止吃帶血的肉，新約耶路撒冷會議（徒十五20、29）仍禁戒吃血。但彼得見異象後食物潔淨條例被廢除。基督徒是否仍被禁止吃血，是解經上的爭議。

## 主題發展

### 舊約背景

- 洪水後神禁止吃血（創九4）
- 摩西律法重申：吃血者被「從民中剪除」（利十七10-11）
- 原因：血中有生命，血可在壇上贖罪（利十七11）

### 新約的處理

1. **彼得見異象**（徒十10-15）：食物潔淨的條例被廢除
2. **耶路撒冷會議**（徒十五19-20、28-29）：雅各裁定外邦信徒須禁戒四件事：
   - 偶像的污穢
   - 姦淫
   - 勒死的牲畜
   - **血**
3. 此禁令來自**聖靈的權威**，與姦淫禁令一同發出，被視為歷久常新
4. 保羅在林前十27-28的教導是針對**絆倒初信者的問題**，非取消血的神聖地位

### 兩種立場

- **仍不可吃**：耶路撒冷會議的禁令具有永久約束力
- **可以吃**：彼得見異象已廢除飲食條例，耶路撒冷會議的禁令是過渡性安排
