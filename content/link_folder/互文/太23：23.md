# 太23：23

## 定義

太23:23與出埃及記第二十一章的關聯：馬太福音23:23記載基督責備法利賽人「將薄荷、茴香、芹菜獻上十分之一，那律法上更重的事，就是公義、憐憫、信實，反倒不行了」，回應出21:31的牛觸死人律例。舊約以賠償處理疏忽致害，基督以公義、憐憫、信實為律法的核心，超越外在的律法遵行。

## 主題發展

出埃及記以賠償律例處理疏忽致害，基督以公義、憐憫、信實為律法核心。從外在遵行到內在精神，體現律法到恩典的發展——律法的精義在於公義與憐憫，而非外在的形式。
