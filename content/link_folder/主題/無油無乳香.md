# 無油無乳香

## 定義

民數記 5:15 規定疑恨素祭「不可澆上油，也不可加上乳香」。油與乳香常用於馨香的獻祭（如利未記 2:1-2），此處排除象徵此祭非為悅納、喜樂，而是為「思念罪孽」、引發神審判的儀式。
