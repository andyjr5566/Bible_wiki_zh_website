# 以色列向耶和華發願毀滅迦南人

## 定義

以色列人向耶和華發願說：「你若將這民交付我手，我就把他們的城邑盡行毀滅」（民21:2）。耶和華應允，將迦南人交付以色列，他們便將迦南人與其城邑盡行毀滅，那地方便叫何珥瑪（意即「毀滅」）（民21:3）。
