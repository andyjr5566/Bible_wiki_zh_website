# 無酵節條例

## 定義

利未記 23 章 5-8 節規定正月十四日黃昏為逾越節，十五日起守無酵節七日，首尾兩日為聖會，不可作勞碌工，並獻火祭七日。此條例確立無酵節的日期、聖會性質與獻祭要求。
