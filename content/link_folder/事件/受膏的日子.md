# 受膏的日子

## 定義

指亞倫及其子孫接續受膏為祭司的就職日子。在受膏之日，祭司要獻上細麵伊法十分之一為常獻的素祭（早晨一半，晚上一半）， 並在鐵鏊上用油調和做成烤好分塊的素祭。此祭物要全燒給耶和華，祭司不可吃，作為永遠的定例。
